紫微

<ul class = "ds-中州派分析段落">
<li>紫微在事业宫，主独当一面。但却并不意味其人可以自行经营，亦不主其人可为领袖。一般情况下仅主独立工作，或负责部门工作。</li>
<li>紫微见煞，古人认为可以经商。此乃经历竞争而后有成，由于成功以致其社会地位亦因而提高之意。现代商人地位已提高，故虽无煞，或且百官朝拱者，亦未尝不可经商，由经商而致富贵。</li>
<li>紫微化权在事业宫，仅主独当一面，或所从事工作独立性强。未必定主富贵。</li>
<li>紫微化科在事业宫，主名大于利，故以自由职业，或专业人士为宜。亦可从政，但恐贵而不富。</li>
<li>紫微得百官朝拱，又无煞曜，可以从政，必能发展至部门首脑。若命宫吉者，更主为方面大员，司部长官。</li>
<li>若紫微无煞曜同度，却有辅佐吉曜同会，又见禄马交驰，或会廉贞化禄、贪狼化禄、武曲化禄者，主职掌财权，从政亦为财经赋税之官。</li>
<li>紫微化科而禄存同度或拱照者，名利双收，以从政或主持大企业机构为宜，尤利上市、公众事业（如电话、快餐、航空、旅游业之类，现代社会类此行业甚多）。</li>
</ul>

子、午：紫微
<ul class = "ds-中州派分析段落">
<li>子午紫微守事业宫。以命宫或财帛宫得禄为贵。若命宫见「禄马交驰」者，无论经商或任职财经机构，皆可出人头地。若财帛宫廉贞化禄者，宜与人合作创业，但自己需负责业务管理。</li>
<li>子午紫微，喜辅弼一居事业宫、一居夫妻宫，（昌曲亦然），则主夫妻彼此有助力，虽非事业合作，亦主可互相匡助。</li>
<li>子午紫微化权，则利事业而不利婚姻，女命尤然。故绝不宜夫妻合作事业。凡女命权星在事业宫者，皆须注意婚姻，紫微尤然耳。</li>
<li>子午紫微得「火贪」、「铃贪」对拱，亦主突发。其突发与配偶无关，但发后恐因贪图享受淫乐而致暴败。</li>
<li>子午紫微化科，又见空劫，一生事业中树敌，故不宜从政，以自由职业为宜。</li>
<li>子午紫微化科，无论有煞无煞，皆主名大于利。若百官朝拱，则主在行业中有地位。不见刑耗空劫，可以从政，以行政管理为宜。</li>
<li>子午紫微与桃花诸曜同度，须视有无昌曲。有昌曲则职业儒雅，无则职业庸俗。</li>
</ul>

丑、未：紫微、破军
<ul class = "ds-中州派分析段落">
<li>丑未宫「紫微破军」在事业宫，以能发挥开创力者为佳。故无论从商从政，皆主开创而不主守成，因而便有劳累之感。</li>
<li>「紫破」与煞曜同宫，主劳身；与昌曲同宫，主劳心。若破军化禄，则一生事业多变，且常负担额外工作。破军化权，则事业多波折，且于开创局面之后不能坐享其成。</li>
<li>「紫破」会武曲化禄、廉贞化禄或贪狼化禄，皆主因能开创而致富。若不能开创局面，则仅主事业有大幅度波动。唯无论如何，均主手面阔绰，而实际收成不大。</li>
<li>「紫破」之开创，乃指凭空打开局面，或以崭新手法营运而言。故喜左辅右弼同度，则自多助力。</li>
<li>「紫破」为火铃夹，席不暇暖。与空劫同度而无刑煞者，则其开创乃由空中楼阁而来；唯若见煞刑耗曜，则一生事业往往功败垂成，且屡伤元气。</li>
</ul>

寅、申：紫微、天府
<ul class = "ds-中州派分析段落">
<li>寅申宫「紫微天府」同度，必须见百官朝拱然后施主有成，若无吉曜，则往往进退失据，或志大才疏。</li>
<li>「紫府」与禄马同度，或见禄马交驰，现代则宜经商，为政亦宜经营财政金融。</li>
<li>「紫府」而紫微化科，主白手与创事业；若天府化科，则主在现成局面下兴创。</li>
<li>若紫微化权，则往往理想过高，财力不足以支持其营运。从政从商皆然。若见刑曜，则以从事武职为佳，权力甚重。</li>
<li>现代「紫府」守事业宫者，亦可从事工业。唯紫微化权时，则不宜理想过高脱离实际。紫微化科，则一生多竞争，例如甫从事一种产品的制造，立即为人所模仿。</li>
</ul>


卯、酉：紫微、贪狼
<ul class = "ds-中州派分析段落">
<li>卯酉宫「紫微贪狼」以从事消费性行业为宜。见桃花昌曲，尤主所营事业带享受性质。</li>
<li>「紫贪」而贪狼化忌，常主与人合作不能长久。若贪狼化禄，则防大运行至天相宫垣，或「武破」宫垣生变化。</li>
<li>「紫贪」与火铃同度，事业主突破，唯若同时见空劫刑耗，则暴起暴败。</li>
<li>「紫贪」会廉贞化忌，宜从事见血的行业。会廉贞化禄，亦可从事政治。</li>
<li>「紫贪」与空劫同度，事业空头。唯波折甚大。年运三方见刑忌虚耗，即生破败。</li>
<li>「紫贪」不宜紫微化科，否则虽有号召力唯缺乏组织力。除非得百官朝拱，然后可以弥补缺点。</li>
<li>「紫贪」躔左辅或右弼，对宫为右弼或左辅，则构成良好格局，主下属众多。唯必须分润下属，然后事业始能持久。</li>
<li>「紫贪」为火铃夹，亦主突破。夹贪狼化忌者，一生暴起暴跌。</li>
</ul>


辰、戌：紫微、天相
<ul class = "ds-中州派分析段落">
<li>辰戌宫「紫微天相」，基本亦主起跌。唯与「紫破」之起跌不同。「紫破」者，其起跌乃出于主观改变；「紫相」者，往往受客观环境影响，常有英雄无用武之地的感受。</li>
<li>「紫相」守事业宫，常居「副贰」，虽实际工作独当一面，而名义则不如实际。</li>
<li>「紫相」守事业宫，见吉化，宜从事财经工作。若从商，以资金营运数额巨大者为宜。最喜巨门化禄来夹，主得道多助，故亦可从政。</li>
<li>「紫相」守事业宫，亦宜与人合作，唯必须实际参加领导核心工作。</li>
<li>「紫相」为刑忌夹，无煞，主事业受掣肘；有煞，则仅宜工艺以谋生。</li>
<li>「紫相」与科文诸曜同度，又见煞耗，则宜从事文艺、自由职业为宜。</li>
</ul>


巳、亥：紫微、七杀
<ul class = "ds-中州派分析段落">
<li>巳亥「紫微七杀」在事业宫，其特长在于管理，故宜以人力资源为主。唯若更见紫微化权，则独断独行，影响管理效果。</li> 
<li>「紫杀」不宜见廉贞化忌，否则易生破败，见常因感情用事而来。</li>
<li>「紫杀」守事业宫亦主身心劳累。见煞者且常受同行或同事挑战，应付为劳。</li>
<li>「紫杀」见禄马交驰，一生三度转业然后始安定。会破军化禄者，身兼三职或常负担额外任务。</li>
<li>「紫杀」最喜紫微化科，可加强管理领导之力。更百官朝拱，亦可从政。在商则为行业领袖。</li>
<li>「紫杀」见煞忌，一生必有破败，煞忌重者，且难东山复起。</li>
</ul>

天机

<ul class = "ds-中州派分析段落">
<li>天机守事业宫，主一生事业多变，而且不稳定。倘火铃同度，则更增加其波动性质。</li>
<li>天机化禄，不能改变其不稳定的性质。若吉星祥曜会合，则可能转变到从事有前途的工作。或仅多兼职，或同时从事两三种事业。</li>
<li>天机化权，性质变为稳定。但亦必先发生一次改变，然后才能从事较为稳定的事业。若左辅右弼同度，则身兼数职，或有多种事业发展，唯所兼职务均为较长期工作。</li>
<li>天机化科，则宜计划、管理、设计等工作，在行业中有声誉。若从商，以零售业为宜，唯必须注意，生意与知名度有密切关系。</li>
<li>若天机化忌，则主不守一业，甫安定便告更张。所更张皆源于计划错误。见空劫、煞耗诸曜者，更属动辄得咎。趋吉避凶之道，在于寻求保守，不可为似乎有利的假象所吸引，以致反为失机。</li>
<li>天机化忌与吉曜同会者，则以从事经纪业为宜。亦可成为代理商。若命宫有太阳化科，则宜代理名牌商品，或为著名字号经纪。唯一生必多竞争，且主受对手中伤。</li>
<li>凡事业宫天机坐守，从事行业以带不稳定性质者为宜。零售业无固定客户，为其一例。此外如运输、交通、旅游、传播等行业，亦带不稳定色彩。学者可由上例意会所谓不稳定之性质。</li>
<li>天机化权、化禄、化忌者，均有投机、冒险的倾向。其中以化忌者最为不宜见煞刑、虚耗、空劫等凶曜者亦不宜。</li>
<li>天机禄忌相冲者，一生事业必有倾覆之虞。见煞者倾覆，见吉曜则终可无恙。若吉煞交集；则视大运、流年吉凶以定其安危。</li>
<li>天机与昌曲、天才同度，宜计划、设计；与龙池凤阁同度，则主有技艺；与火铃同度，事业以不稳定者为宜。</li>
<li>天机化忌，则宜从事保险。若天机与天刑天月同度，则可从事外科手术，或用凶器的行业，或带危险性质的工作。</li>
<li>天机化科，事业亦主变动。由于命宫「天同巨门」之巨门必同时化忌，因此常劳心费神，兼顾额外工作，有能者多劳之叹。</li>
</ul>

子、午：天机
<ul class = "ds-中州派分析段落">
<li>子午宫天机独坐，对宫巨门，即使会吉星祥曜，其事业亦必费心机。倘有煞忌刑耗诸曜，则往往受客观环境影响，为事业奔波劳累。</li>
<li>唯天机在子午宫守事业，必成。「机月同梁」格，亦主创业之力薄弱。当局者亦必自有一套人生哲学来解释自己惰性。见煞者虽有惰性，但仍须不断为事业操心 ，然不易有建树。</li>
<li>若天机在子午宫化禄，则财帛宫太阴必同时化忌，增加其事业上投机的倾向。</li>
<li>子宫天机不如午宫。在午宫者，宜承继父业发展，亦宜入大机构，在现成局面下发展。倘经商，则宜注意商标声誉。在子宫，其辛苦劳累往往倍之。且易招是非怨忌。</li>
<li>子午天机化忌，必因事业而招人身攻击。在现代，子午天机守事业宫者，亦可从事传播、广告行业。此与商标声誉同义。</li>
</ul>

丑、未：天机
<ul class = "ds-中州派分析段落">
<li>丑未天机在事业宫，天机落陷，故宜在公司机构任职。若见辅佐吉曜，则服务机构局面较大。见煞忌刑耗者，则常转换事业，或转换服务机构，且局面不大。</li>
<li>丑未天机亦主事业带任劳任怨性质。</li>
<li>若天机化禄，则以从事零售业为宜。若天机化权，则于破军大限或「廉相」大限必有事业上的重要改变，可以乘时创业，亦以零售业为佳。</li>
<li>天机在丑未宫守事业，由于与天梁相对，有计划、监察、管理的意味。故亦可服务大机构任财经部门、会计部门、计划部门之职。如行业带流动、服务色彩者尤宜。</li>
<li>见煞忌，宜在工业界服务。</li>
<li>见巨门化权，宜口舌生财，故最宜从事销售部门之监管计划工作。有时亦可从事自由职业，如传播、广告、计划、推广等。</li>
</ul>

寅、申：天机、太阴
<ul class = "ds-中州派分析段落">
<li>寅申宫「天机太阴」守事业宫，若天机化禄则太阴必同时化忌，若从事投机，必然失败。且一生多是非刑忌。必须心胸广阔，不志在求财，随遇而安，反而可获良好际遇。</li>
<li>「天机太阴」守事业宫，亦主由现成事业发展。若太阴化禄、天机化科，又会天同化权者，主有祖上产业，或主有长辈荫庇，由是得克享富贵。克享之道，在于守成，而不在于开创。</li>
<li>若「天机太阴」守事业宫而无禄、权、科会者，则宜在政府或大机构发展。一般宜任职而不宜经商。</li>
<li>有煞忌、空劫，则绝不宜经商，否则必有一次重大破败，以大运、流年复见空劫刑忌，又有流煞冲会之年为克应。</li>
<li>若「天机太阴」为紫微化权、天府化科所夹，则「机阴」由动荡变为稳定，若任公职，必有权力，唯亦易招惹烦恼是非。</li>
</ul>

卯、酉：天机、巨门
<ul class = "ds-中州派分析段落">
<li>卯酉宫「天机巨门」守事业宫，必须有化禄或禄存同度，始有事业发展。唯巨门化禄则较优。盖天机化禄，则易因事业失机而生破耗。尤忌投机。</li>
<li>「天机巨门」守事业宫者，纵有祖业亦必破败，然后自兴。若见煞忌刑耗，则事业动荡。</li>
<li>「天机巨门」化忌者，若天机化忌，则仅主事业多变迁；巨门化忌，则事业宜注意口舌生财的性质。</li>
<li>「天机巨门」不喜火铃同度，纵有化禄或禄存，亦为破格。盖火铃二曜，均足以增加天机之浮荡，巨门之是非。</li>
<li>「天机巨门」守事业宫，亦不喜借会太阳化忌，主是非尤怨，故必须从事「口舌生财」的行业，如传播、法律之类。倘见吉曜，亦仅宜零售业。</li>
</ul>

辰、戌：天机、天梁
<ul class = "ds-中州派分析段落">
<li>辰戌宫「天机天梁」，最易兼行兼职，见吉化、吉曜，以从事顾问工作为适宜。</li>
<li>若「天机天梁」见禄、权、科会，及辅佐吉曜，则亦可从政，但较难独当一面，且职位调动频仍。由于借会「太阳巨门」，故宜外交工作。若服务于企业，则以公关之类为宜。</li>
<li>「天机天梁」见煞，以专业技能为佳。</li>
<li>「天机天梁」喜会太阳化禄，则须先建立地位而后寻求较高收入；亦喜会太阴化禄，则可从事财务专业，以带监督、管理性质者为宜。</li>
<li>「天机天梁」主变动，在现代，亦可从事电子、电脑等行业。</li>
<li>「天机天梁」与空劫同度，能突破传统概念，于行业中有所建树者，则亦能富贵。若不在乎事业，但求随遇而安，则事业变动频仍，终至一事无成。</li>
<li>「天机天梁」见吉化、吉曜，及天巫，亦主有祖上遗业承继，或由现成事业发展。但若吉煞交集，则必破尽祖业而后兴家。或不守现成事业，经挫折后另创新业。</li>
</ul>

巳、亥：天机
<ul class = "ds-中州派分析段落">
<li>巳亥宫天机，见吉化、吉曜，亦主有遗产、遗业承继。或主现成局面下发展。</li>
<li>巳亥天机守事业宫者，无论见吉曜或煞曜，均主有重大行业变动。唯无论如何改变，均不离服务性质。</li>
<li>巳亥天机与火铃、空劫同度者，事业变动频仍，或时时转换工作岗位。</li>
</ul>

太阳
<ul class = "ds-中州派分析段落">
<li>太阳守事业宫，吉则事业广大，声名鼎盛；凶则事业空虚，浮夸不实。</li>
<li>太阳主贵，故守事业宫颇为适合。吉则在行业中必有声名，纵凶亦主有号召力。</li>
<li>太阳守事业宫、入庙，又见魁钺、辅弼、昌曲者，不但事业广大，而亦主下属众多。但若不见禄存、化禄者，则仅得声名而无实利。倘稍见煞，更主虚名虚利。</li>
<li>「阳梁昌禄」守事业宫，主宜从事学术研究。但小心此格有许多变格，例如文昌化忌，即其「研究」性质便宜带有较低层次色彩。在古代，重士子，轻商人，故亦可视为商业研究的星曜组合；又如带有刑煞，则往往为工业的研究。</li>
<li>「阳梁昌禄」又主竞争，所以宜从事竞争性大的行业。在此各种不同的社会环境，有不同的行业，须加注意。</li>
<li>太阳常动，因此守事业宫时，亦主其一生必不守一业，除非所业是带极专门性质，或职业之涵盖面广，接触面广（如社会工作者，保险经纪等。）</li>
<li>太阳与火铃羊陀同度，其动更甚，故宜从事本质奔忙变动的行业。但此种动象又与天机不同，天机之动，缺少气派，所以可能为运输、航海之类，而太阳之动，则始终有一种气势，故如外交之类，或主持一机构之公关，或为传播事业，学者对其「动」的性质须多加体会。</li>
<li>太阳主散，故容易流为空虚不实，加空劫者尤然。但亦可利用此种性质，于空中楼阁创造事业，此即为发明创造，或仅凭一个特殊意念即可兴家的星系。</li>
<li>太阳与魁钺同会，古人认为乃「近贵」之兆，所谓「近贵」，即是能获人提拔扶持。在现代则为机会的提供。但亦可以看作是在政府部门工作，或半官方机构、大企业财团服务，较容易受上司赏识提拔。</li>
<li>昌曲入事业宫，为文学、艺术，但宜与太阴同度。若太阳昌曲同躔，其贵反而较轻，仅主其事业与文艺有关，或与传播、公关、演艺等有关。</li>
<li>太阳喜左辅、右弼同会事业宫，则可减少其发射力，亦即较少奔劳。</li>
<li>太阳化禄，事业以声名为主，若经商，尤须注意商标、商誉。若从事学术、传播、演艺，则须注意发挥一己之特长。</li>
<li>太阳化权，则巨门必同时化禄。在事业宫，此为受异族提拔、赏识之星系，或一生事业与异族有莫大关系，因此须提高外语知识，俾便适应。倘当局者的地理环境，无法接触外语及异族，则不妨将范围缩小为异乡，学习异乡方言，常亦有裨益于事业。</li>
<li>太阳化科，宜事业有荣誉感。例如名医、名律师，或有声誉的社会工作者之类，皆不出专业人士范围。有时亦表徵为一种必须有声誉，或必须受人信赖的行业，前者如经纪行，后者如公证行。</li>
<li>太阳化忌，所从事行业必须有排难解纷，或是非口舌的色彩。但所谓「口舌」是非却未必指纷争，如教学、演唱等，皆为「口舌以求财」。</li>
<li>太阳入庙，然后事业始主广大，兼且亦易声名显赫。若落陷，则纵有声名亦不出乡里，或仅在行业中有声誉，不为社会人士所知。</li>
<li>太阳亦喜三台八座，恩光天贵等杂曜朝拱，皆有助于事业。古人视为门生弟子众多之兆。</li>
<li>唯太阳与华盖同度，又同会空劫，则往往为哲学、宗教的表徵。与天德同度，则宜与人排难解纷。</li>
</ul>

子、午：太阳
<ul class = "ds-中州派分析段落">
<li>子午宫太阳，与天梁对拱，名重于利，最宜专业工作，或建立商标商誉。</li>
</ul>

丑、未：太阳、太阴
<ul class = "ds-中州派分析段落">
<li>丑未宫「太阴太阳」同度，吉则名利双收，凶则仅虚名虚利。</li>
<li>「太阳太阴」的星系，在事业宫，比较倾向于内务。故以行政管理、财务计划等为宜。</li>
<li>「太阳太阴」守事业宫，入庙则事业广大，落陷则事业空虚。但皆主变动，仅程度上有所差异耳。所谓变动，有时又主带动态的行业而言。</li>
<li>一般而言，太阴吉化主富，或所业与财政金融有关；太阳吉化主贵，所业以服务性者为佳。</li>
<li>「太阳太阴」与空劫同度，其由空中楼阁创造事业的能力，一般较太阳天梁星系为弱。因易改变主意，反成进退失据。</li>
<li>「太阳太阴」的组合，若太阳化忌，反宜从事参与灾难与困扰的行业；若太阴化忌，则仅宜内务工作，以不出锋头为宜。</li>
<li>「太阳太阴」守事业宫，若见煞刑诸曜，则事业上必有一段时期受到同业或同事的排挤。同时见忌星者，不宜同流合污，否则反受拖累牵连。</li>
</ul>

寅、申：太阳、巨门
<ul class = "ds-中州派分析段落">
<li>寅申宫「太阳巨门」同度，基本性质为口舌、为异族、为外务。这三种性质，均与太阳太阴的组合性质相反。</li>
<li>「太阳巨门」见吉化，均主传播、广告等业成名。巨门化禄者尤长于口才。若同时见辅弼、昌曲、三台八座、恩光天贵等曜，则宜从政，且宜外交。</li>
<li>「太阳巨门」而巨门化忌，格局高者宜公关、法律、传播、演艺；格局稍弱者则为教学、推销之类。倘更见煞刑诸曜，则古人认为所业与酒色财气有关。</li>
<li>若太阳化忌，所业非带排难解纷色彩不可，否则反易招压力。</li>
<li>太阳化禄、化权，均主因异族关系而得财。或凭口才、语言、歌唱等才艺以得财。此种性质，寅申宫重于巳亥宫。</li>
<li>「太阳巨门」的星曜组合，如太阳落陷，则不宜一味求名，故凡带出锋头性质的行业均有所不宜。此非指个人而言，例如广告行业，本身即带出锋头的性质。</li>
<li>但太阳落陷且化为忌星者，若有吉星同会，无四煞刑耗诸曜，则又以出锋头为宜，虽招指责，反而成名。如影视界的歹角、走偏锋的律师之类。格低者则为贩夫走卒。</li>
<li>「太阳巨门」见化忌不宜从政；巨门化忌固招是非，若太阳化忌则易招怨。</li>
</ul>

卯、酉：太阳、天梁
<ul class = "ds-中州派分析段落">
<li>卯酉宫「太阳天梁」同度，亦为名重于利，亦宜专业工作，但事业却较子午宫者不稳定，尤以酉宫者为然，必须日夜奔忙，或不居一地，或时时更换工作环境。</li>
<li>「太阳天梁」的星系，最宜从事医药、保险等服务性行业。但若见煞，又见阴煞、天姚、天刑、大耗、劫煞等杂曜，则易由偏门或半偏行业起家。</li>
<li>凡「太阳天梁」星系守事业宫，本身即带刑忌孤克色彩。医药用以治病，不病则不必求医；保险用以救济，无灾难则不必救济，则此意义，即可解释行业性质。</li>
<li>然而「太阳天梁」又可引伸为监察、管理，此是由刑法引申而来，成为纪律、条例的意义。即使学术研究，亦须遵守原则。</li>
<li>但无论如何，「太阳天梁」必带社会服务的色彩，故即使成「阳梁昌禄」格者，其研究性质亦必须直接服务于社会，不能单纯局限于理论。除非空劫同度，始作别论。</li>
<li>「太阳天梁」不见科文诸曜，又落陷，见煞忌，杂曜，又吉凶交集者，则多主半偏或偏门，但仍带服务色彩。极端而言，例如贩毒，亦可视之为药物的服务。</li>
<li>「太阳天梁」有时又为宗教，或神秘事物。此以见化盖、空劫为然。</li>
</ul>

辰、戌：太阳
辰戌宫太阳与太阴相对，落于罗网，因此与丑未宫比较，反而较踏实沉潜，专业色彩亦较浓厚。


巳、亥：太阳 
巳亥宫太阳守事业宫，对宫必为巨门，本质亦与寅申宫者相近，但外务性质则更重，巳宫者又重于亥宫。（以寅申二宫言，寅宫又重于申宫。）

---------------------------------------------------------------------
武曲

<ul class = "ds-中州派分析段落">
<li>武曲为财星，守事业宫时，利于从事财经工作，或为赋税官吏。此以得化禄、化权、化科者为宜。且宜见辅佐吉曜。</li>
<li>武曲又为金星，故主事业性质带有金属。倘与煞曜同躔，主工艺，古代订定为金匠、银匠、铜匠，即称「三匠」。现代工艺行业为多，凡工程技艺皆属此类。</li>
<li>倘武曲不与煞星同度，但却在三方会合，又见破军、七杀者，则金属性质转为利器，宜从事利器以求财的行业。如军旅、警察、外科医生及牙医等。</li>
<li>凡武曲居事业宫，均与金属有关（古代的钱银亦为金属），且主行动。例如从事借贷即为一种行动，与负责财务计划不同）。所以喜见左辅右弼，主行动有助力。昌曲未必皆喜，因为有时昌曲反而成为行动的障碍，令人犹豫不决。故武曲与昌曲同会，必须兼见辅弼，或兼见魁钺，然后始为将才或财赋官吏。</li>
<li>武曲居寅申、卯酉、巳亥六个宫垣，为武曲坐闲宫，古人说：「武曲闲宫多手艺」，此以与煞曜同躔为确。这种格局却喜见昌曲，主手艺或技艺不俗。唯产科、外科、牙科医生等亦属「手艺」，未必指工匠而言。</li>
<li>武曲与火铃同度，则不喜更落于巳午二宫，亦不喜多见属火的星曜，否则一生至少有一次卸职或失业。以重见武曲，又逢煞忌冲起的大限或流年为克应之期。</li>
<li>武曲化科，主声誉，故无论从事何种行业，均以建立声誉为要务。如首饰设计之款式、银行投资之行动，皆为建立声誉之目标。</li>
<li>武曲化忌，经商者每感资金不足，不能依理想发展。故宜从事技艺为宜。逢武曲化忌的大运流年，亦应小心财力不足以发展事业的现象。</li>
<li>武曲最不喜与陀罗同度，因陀罗亦为忌星。</li>
<li>但武曲化忌，或陀罗同度，却又宜从事带凶险、冒险性质的行业。如蛙人、殡仪之类。只须见吉曜同时会合，亦可发迹变泰。</li>
<li>大抵厘定武曲居事业宫的格局高低，以入庙与否为主。有无吉化及辅佐诸曜，则可作为参考。唯武曲最喜会廉贞化禄，即武曲亦同时化为科星，这种吉化则有助于订为高格。</li>
</ul>

子、午：武曲、天府
<ul class = "ds-中州派分析段落">
<li>子午宫「武曲天府」对宫为七杀，无煞忌刑而有禄，则主为财赋之官，倘更见辅佐诸曜，其人地位必相当。唯若三方见煞，则主为武职，同时见辅佐诸曜者，以军功起家。</li>
<li>「武曲天府」与煞曜同度，则宜工程技艺。亦以有禄为佳。</li>
<li>「武曲天府」而天府化科，虽无禄，亦以财经事业为宜。</li>
<li>「武曲天府」，子宫优于午宫，盖子宫者，对宫七杀在午，有火炼阴金之象，可影响「武府」之性质，经磨练而成大器，倘见辅佐诸曜同拱，更宜行伍出身。</li>
</ul>

丑、未：武曲、贪狼
<ul class = "ds-中州派分析段落">
<li>丑未「武曲贪狼」，见贪狼化忌者，终生不得闲暇，奔波劳碌，事业多端，甚或半官半商，为财是务。</li>
<li>「武曲贪狼」而贪狼化禄者，无煞曜而有辅佐吉曜，可成巨富。唯经商者难免暴利。故稍有煞刑，即易暴起暴败，不独「火贪」，「铃贪」格为然。</li>
<li>「武曲贪狼」亦可会财赋之职，或任财经企业，但「火贪」、「铃贪」皆主贪墨，唯武曲化为科星者，可免因贪致祸。</li>
<li>「武曲贪狼」而武曲化禄、贪狼化权者，其事业亦必与财经有关，但同时亦必经营潮流所趋之行业。</li>
<li>「武曲贪狼」守事业宫，喜与人合作，故所业亦不只一端。</li>
</ul>

寅、申：武曲、天相
<ul class = "ds-中州派分析段落">
<li>寅申宫「武曲天相」，主文职武做，或武职文做。在现代，文武区分不易，大致而言，内务属文，外务属武；财经等事业属文，工程等事业属武。</li>
<li>「武曲天相」与桃花诸曜同度，又与文曲同躔，则为伶人。唯以不见吉化吉曜者始确。倘煞曜重重，则主辛霜露酸，江湖流浪。</li>
<li>「武曲天相」为财荫夹者，甚宜任职财经部门；若为刑忌所夹，则经商亦有所不宜。</li>
<li>「武曲天相」守事业宫，有吉化，亦主与人合作，且须负实际责任。</li>
<li>「武曲天相」与煞曜同会者，宜由工程技艺发展。</li>
</ul>

卯、酉：武曲、七杀
<ul class = "ds-中州派分析段落">
<li>卯酉宫「武曲七杀」有煞曜在三方者，宜军旅出身。见辅佐吉曜，必可荣身。若煞在本宫，则仅主工程技艺。亦以带金属性质者为宜。</li>
<li>「武曲七杀」与四煞空劫同度者，主颠沛流离，无论从军从工，均主事业饱经挫折。</li>
<li>酉宫「武曲七杀」喜火星同度，卯宫则不喜。盖酉宫「武杀」得火星为锻炼，主经挫折而得成就。卯宫属木，得火星嫌火过旺，纳音属火者尤忌。主辛劳而无结果。</li>
<li>「武曲七杀」得禄者，亦宜财经工作，且擅长营运。以见吉化吉曜为佳，煞重则须防挫折。</li>
<li>「武杀」见煞刑诸曜，亦主外科手术，跌打疮科之类。</li>
</ul>

辰、戌：武曲
<ul class = "ds-中州派分析段落">
<li>辰戌宫武曲独坐，与贪狼相对，其性质与「武贪」大致相同，唯武曲落罗网，故贪得之性稍减。</li>
<li>若成「火贪」、「铃贪」格者，其事业亦带暴利或贪污色彩，故主暴起暴跌。</li>
<li>最不宜与空劫同度，主一生至少经一次挫折，或主不良不莠，难以成器。</li>


巳、亥：武曲、破军
<ul class = "ds-中州派分析段落">
<li>巳亥宫「武曲破军」，常主喜插手任何事业，不考虑自己的特长。故吉则事业众多，凶则事业多变。</li>
<li>与天刑同度者，宜军旅或纪律部队出身，或负情报治安之责。以见辅佐吉曜为宜，若煞忌重重，则须防凶危。</li>
<li>「武破」在事业宫，与铃星、陀罗、阴煞、华盖、天刑同度者，为偏门或半偏行业。</li>
<li>「武破」煞轻，亦宜工程技术起家。</li>
<li>若「武破」吉化，同时又见廉贞化禄或贪狼化禄者，亦可经商，则虽时生计较之心，但手面则异常阔绰。</li>
<li>「武破」见破军化禄，擅开创而不擅守成，主意多而不喜务实。故必须与魁钺同会，然后始主易得机会；与辅弼同会，然后始主多得助力。</li>
</ul>

------------------------------------------------------------------
天同

<ul class = "ds-中州派分析段落">
<li>天同守事业宫，主事业必由白手创兴。自另一个意义而言，则是纵有父业亦难承继，有须经历困难然后始事业有成。</li>
<li>天同化禄，或与禄存同度，白手创业的意味尤其明显。且有时主至中年以后始能立足脚跟，中年以前，事业反复多变。</li>
<li>天同化忌，往往早年即须工作，但事业之创兴却可能比化禄者为早，中年以后若能稳守，即可踌躇志满。存辰戌丑未四宫者尤确，又以在戌宫者最为上格，未宫次之。</li>
<li>天同在事业宫，往往不必见禄马，亦主宜离乡背井发展。</li>
<li>天同对四煞异常敏感。擎羊同度，主事业发生争夺、竞争，陀罗同度，主事业发展至一阶段，好发生无形变动；火铃同度，主多焦虑，须呕尽心血后始能有成。</li>
<li>当四煞同度之时，天同若化为忌星者，则一生事业必波折重重，与天同化忌不与煞星同度者，性质截然不同。</li>
<li>天同又为艺术之星，且倾向于音乐，居事业宫，有桃花诸曜同会者，可向艺术发展；唯「天同天梁」的星系，必须不见火铃始是，否则定主半途而废。</li>
<li>天同与空劫同度，亦加强其艺术气质，唯必须建立个人独特风格，且亦有此潜质。</li>
<li>天同在命宫或事业宫，火铃同度或对拱，声线带沙，若从事音乐、歌唱，必须注意风格之配合。</li>
<li>天同与昌曲、天才同度，亦宜文艺事业。龙池、凤阁同度，则为器乐，或为文化事业中的技术工作者。</li>
<li>天同与左辅右弼同会，须天同不化忌星始主得助力，否则纵有下属，却有帮不上忙之感，仍须亲力亲为。</li>
<li>天同又主享受，故居事业宫，每主适宜经营美观或带享受色彩之行业。</li>
<li>天同不宜入墓（即与长生十二神的墓星同度），主事业与个性不相投。或觉得有「不逢明主」的感觉。</li>
</ul>


子、午：天同、太阴
<ul class = "ds-中州派分析段落">
<li>子午宫「天同太阴」，若见禄存，或见化禄，则主有父业承继，且可守成。或主在现成局面下发展，不宜创新。</li>
<li>「天同太阴」守事业宫，亦主事业多变，且主有多方面的事业。即使在政府机关或企业机构任职，其责任亦多变动，或不只一端。</li>
<li>「天同太阴」而太阴化忌，则不守祖业，亦不能守成，必须经变动而后事业财源始告稳定。</li>
<li>「天同太阴」而天同化忌，主动荡不安，每须背井离乡而后始能白手创业。同时见火铃者尤确。</li>
<li>「天同太阴」在事业宫，每为内务外务兼顾人材，且有责任感。唯独当一面之机会则较小。</li>
</ul>


丑、未：天同、巨门
<ul class = "ds-中州派分析段落">
<li>丑未宫「天同巨门」，即使吉辅诸曜毕集，亦主事业每半途而废。</li>
<li>「天同巨门」可从事饮食业，见天厨、火星同度者尤然。</li>
<li>引申而言，「天同巨门」守事业宫，亦主口舌，故亦可成为广告、传播等业的表徵。唯仍须注意及消闲、享受的性质。故此有时又为潜水业及水管修理业。</li>
<li>「天同巨门」又主医药，此系因所会「太阳天梁」有医疗意味之故。唯多从事外科及危症之治疗护理。</li>
<li>「天同巨门」见煞忌，反主历艰危而成大业。天同忌又优于巨门化忌，艰辛较小，劳心费神较少。</li>
<li>若天同化禄，事业虽较优悠，但成就之期亦晚；巨门化禄，只宜「口舌生财」的行业。唯以整体成就而言，反而不及有忌星刺激者之佳。</li>
<li>唯「天同巨门」而煞忌刑耗重重者，则反可能一事无成，蹉跎至老。必须劳心费神，或从事本质即属劳心费神的事业，然后始主成就。后天努力可以扭转命运，此即为一例。</li>
</ul>


寅、申：天同、天梁
<ul class = "ds-中州派分析段落">
<li>寅申宫「天同天梁」同度，仅宜出任公职。若自己经营，亦仅宜组织有限公司，任监察、管理之责。或所经营行业，本身即有监察、管理之性质。</li>
<li>「天同天梁」所从事者必以内务为宜。如计划、管理之类。</li>
<li>「天同天梁」有吉化，又见昌曲同度，则可从事艺术工作，但仍须处理内务，即绝非表演行业，以见天梁化科者最为高格。</li>
<li>「天同天梁」守事业宫，亦主一生职务多变，或时时变换工作环境，或因工作须跋涉为劳，或时时须出差工作。见禄马交驰者更甚，火铃照会者亦甚。唯禄马交驰则财气较旺耳。</li>
<li>倘引申而言，「天同天梁」坐事业宫者，亦可服务于交通、运输等带变动性质的行业。</li>
</ul>


卯、酉：天同
<ul class = "ds-中州派分析段落">
<li>卯酉宫天同，与太阴相对，不主有父业承继。唯此星系却带艺术色彩。</li>
</ul>


辰、戌：天同

</ul>


巳、亥：天同

</ul>

廉贞

<ul class = "ds-中州派分析段落">
<li>廉贞守事业宫，古人认为属于武职，此乃由于古代社会仅重视仕途之故。在现代社会，廉贞其实亦带有浓重的商业色彩。</li>
<li>古人论廉贞守事业宫，入庙者，武职显赫，此乃以不与煞忌同度者而言。若有煞，不宜武职，可从事工程工艺。</li>
<li>廉贞亦为艺术之星，但倾向于土木工艺、竹木雕刻之类。在现代，则常为建筑业（不是地产业）。</li>
<li>廉贞会文昌、文曲，主文艺，但不倾向于工艺。主对色彩有特殊敏感，故倾向于绘画、图案设计之类。</li>
<li>廉贞守事业宫，无论何等行业，皆表现得有气派，亦可谓重外表多于实际。见煞者尤然。故最不喜空劫同会。</li>
<li>廉贞守事业宫，武曲必在财帛宫，若昌曲入此两宫度（尤其是文曲与武曲同躔），则主武职文做，或文职武做。在现代，此两种性质相当复杂，只能由行业分文武，再由职责任务分文武。</li>
<li>廉贞亦主精神享受。故亦可从事关于此方面的事业。</li>
<li>若廉贞化忌，则宜从事与血液有关的事业。此项性质须灵活运用。例如「血腥导演」亦属此类，由引一例，可知其性质之广泛也。</li>
<li>以从政而言，廉贞所会的紫微有吉曜，若百官朝拱之类，则宜文职，且能独当一面；若会「紫微七杀」、「武曲七杀」，或本身即为「廉贞七杀」，则宜武职荣身。亦以煞曜不居本宫为贵。</li>
<li>上述性质，可引申入工商两业。一般而言，工属武，商属文。此为现代社会之特徵，不可拘泥于古代社会的观点。</li>
<li>廉贞化禄于事业宫，论富裕的程度，不及廉贞化禄守命或财帛宫。因守事业宫时，有可能仅为从事外表奢华的行业。如夜总会、歌楼舞榭之类，故必须更详财帛宫的星曜以定其有财无财。</li>
<li>凡廉贞必带出锋头、自我表现、外表浮夸的性质，或所从事的行业带此类性质，故在斗数中，廉贞不必化科而「化科」在其中矣。善于体会此意，对界定事业性质甚有帮助。</li>
</ul>

子、午：廉贞、天相
<ul class = "ds-中州派分析段落">
<li>子午宫「廉贞天相」，常主与人合作事业，故须注意兄弟宫的星曜组是否吉利，藉此即可推断其事业之成败。</li>
<li>「廉贞天相」，亦为从政之星，或至少对政治有兴趣，则必须见天魁天钺，仕途始觉亨通。若见煞曜，则一波三折，起跌无常。</li>
<li>「廉相」会禄（武曲化禄为上，禄存次之），或会天府化科，宜握财权。若禄煞交躔，则最宜任军需之职。此即为武职文做一例。</li>
<li>若廉贞化忌，或为刑忌所夹，则以从事血光或凶险之行业为宜。如管理监房，即为凶险行业之一例。若不从事此等行业，则主终身受人压力。或遭排挤，或为人陷害。无论从商从政，均所不宜。</li>
<li>「廉相」为财荫所夹，其吉比同会魁钺为更大。主一生受人提携，且多助力。或主承继现成事业加以开展。</li>
</ul>


丑、未：廉贞、七杀
<ul class = "ds-中州派分析段落">
<li>丑未宫「廉贞七杀」，宜武职。军警界出身，或在刑法纪律组织工作，皆为适宜。三方有煞而本宫无煞者尤确。若本宫有煞，则不如从事工业。</li>
<li>「廉贞七杀」而廉贞化忌，更见煞曜同度（空劫不是），则主马革裹尸，故从事军警两途皆主殉职，或惹重大刑法官司。故不如从事带血光之行业，如饮食业、化验所等，反可趋吉避凶。</li>
<li>「廉贞七杀」坐事业宫，一生必至少有一次重大挫折，以见财星化忌、煞刑耗空劫凑会之大运及流年为克应之期。</li>
<li>「廉贞七杀」而廉贞化忌者，以三合宫见忌冲会之流年大运，为见血光克应之期。</li>
</ul>


寅、申：廉贞
<ul class = "ds-中州派分析段落">
<li>寅申宫廉贞守事业宫，与贪狼相对，以从商为宜。若从政，则易流为各党派集团的调和折衷人物，且易成为傀儡。</li>
<li>寅申宫廉贞，亦带投机性质，故若从商，最宜投身财经金融行业，负买卖之责。亦宜从事基金投资。廉贞化禄者尤为吉利，可以白手成富。</li>
<li>若廉贞有刑煞曜同会，则可为帮会人物，或事业带半偏色彩。</li>
</ul>


卯、酉：廉贞、破军
<ul class = "ds-中州派分析段落">
<li>卯酉宫「廉贞破军」必须见禄然后始主事业有成。 以廉贞化禄事业较为顺遂。若破军化禄，则不但多劳，而且事业必有挫折。</li>
<li>卯酉「廉破」，见煞轻者，亦带偏门或半偏门色彩，所从事行业，多外表浮华，或与酒色财气有关。倘吉煞交集，则不过中下级行政人员。</li>
<li>廉破守事业宫见吉，多因事业而应酬，又由应酬发展事业，故其所经营，不只一端。故须注意其兄弟之优劣，而定其事业之成败吉凶。</li>
<li>「廉破」而廉贞化忌者，不见天刑、四煞、空劫，则虽事业偏门，亦不致惹官非刑法，否则必有牢狱之灾。</li>
</ul>


辰、戌：廉贞、天府
<ul class = "ds-中州派分析段落">
<li>辰戌宫「廉贞天府」同度，事业倾向于财经。亦宜经营与金属有关的行业。</li>
<li>「廉府」守事业宫，最喜廉贞化禄，则可成富格。天府化科，则事业与声誉有密切关系。</li>
<li>古人以「廉府」守事业宫，见昌曲、辅弼、魁钺，为文武兼资，大富大贵。此意在现代可以参考，或可引申为工商兼业，或半政半商，唯必以与财经金融有关者为宜。</li>
<li>若廉贞化忌，则不宜从事金融更见煞，必因此而兴讼。</li>
</ul>


巳、亥：廉贞、贪狼
<ul class = "ds-中州派分析段落">
<li>巳亥宫「廉贞贪狼」同度，空劫又同度者，最宜艺术，尤以表演行业为宜。</li>
<li>若从商，场面必大。有禄则可，无禄者易流为外表豪华，内在空虚。</li>
<li>「廉贪」守事业宫，往往事业亦不只一端，亦主常易与人合作。</li>
<li>当「廉贪」与煞曜同度，则以从事工业设计、工艺美术为宜。 </li>
<li>「廉贪」而贪狼化忌，一生必有一次重大的转业。若廉贞化忌，则以半偏行业为佳。或从事饮食业亦可，则必以新潮作为号召，不能守旧有原则。</li>
<li>「廉贪」守事业宫者，亦多事业上的应酬，又由应酬而扩展事业，其性质与「廉破」略同，唯权力欲则稍逊。</li>
<li>「廉贪」必须得禄始主发越，倘无禄见煞，只宜工艺途中走。</li>
<li>若「廉贪」有文曲化忌对拱，又见煞忌刑耗诸曜者，主其人贪小，其事业亦如之。</li>
</ul>

天府

<ul class = "ds-中州派分析段落">
<li>天府守事业宫，以在现成局面下发展为宜。此并不是说一定有父祖之业可继，例如参加一已有规模的企业，成为领导阶层人物，即属此种性质。</li>
<li>天府入庙，可守现成事业。若落陷，则当局者可能过分谨慎，以致无法在现成基础下发展。若煞曜同躔者尤甚。</li>
<li>凡主星在午宫守事业宫者，皆主事业广大，天府亦然（太阴则宜在子）。以得禄为贵。此即所谓「安命寅宫，府午宫、相戌宫来朝，甲生人一品之荣」。此乃「武曲天府」守事业宫的上格。此格之特点，在于「府相朝垣」，且天府得命宫之双禄。若无禄者，则不入此格。</li>
<li>天府必须得禄，然后事业始主亨通。否则事业虽佳，亦不过外强中干而已。</li>
<li>天府无禄，亦无煞忌刑会照，而有吉星祥曜扶持，则守成亦可，不致殒越。若既无吉曜，却有煞曜凑会者，则事业易为小人所算，甚至受拖累而致失败。</li>
<li>最坏的结构是，天府落陷，天姚及文曲化忌同度，则主受骗。大运及流年见此亦然。唯原局见此星系结构时，有时又主此人以巧骗营生，否则亦斯文败类。</li>
<li>天府在事业宫，宜兼视财帛宫之天相，以得财荫夹者为上格。若刑忌夹，则事业亦受到限制，以任职为宜，不宜自行经商。</li>
<li>天府不喜火铃照射，主事业波动。以工业、工程发展者为宜。</li>
<li>天府有空劫同度者，事业多进退失据，既思改革，又持保守，以致事业发展受到障碍。故以从事实业为宜。切忌投机。</li>
<li>天府会羊陀，则主事业有波折，仍以从事工业或工程为佳。</li>
<li>仅于得会武曲，又见禄存或化科的场合，所会的天相亦无煞曜，然后始主可经营财经金融事业。仍以统筹大局为宜，不宜负责实际工作。</li>
<li>天府入庙，化科，则以从事信贷等事业为佳，若有煞，则仍宜工程或工业。</li>
<li>天府不主浮华，凡有经营，皆以脚踏实地为佳，否则徒自寻烦恼。</li>
</ul>

子、午：天府、武曲 


丑、未：天府

<ul class = "ds-中州派分析段落">
<li>丑未宫的天府，一般情形下宜任武职，或经营工业、工程。仅在「叠禄」的情形下，始主财经界可以得巨大发展。若从政，亦宜为财赋之官。</li>
<li>丑未天府守事业宫，巳亥天相无刑忌夹，命宫无正曜，禄存入度；或迁移宫贪狼化忌，且禄存在财帛宫者，皆为巨富之格。从商从政，皆主名利兼收。唯必以无火铃、空劫、刑耗同度者始合。</li>
<li>若丑未宫天相为财荫所夹，但天府却见文昌化忌来会，则其人必以带投机色彩之事业起家。因此须详推流年大运，以定其进退之机，倘时机失误，则成就受影响甚大。</li>
</ul>

寅、申：天府、紫微 


卯、酉：天府
<ul class = "ds-中州派分析段落">
<li>卯酉宫天府，若所会之天相为刑忌夹者，不宜投机，亦不宜从事金融行业。以任职公司机构为宜，更不宜从政。</li>
<li>卯酉宫天府，得武曲化禄来会，亦主任财赋之职。</li>

辰、戌：天府、廉贞 


巳、亥：天府
<ul class = "ds-中州派分析段落">
<li>巳亥宫天府守事业宫，对宫为「紫微七杀」，天府受此权力之星影响，常有独立自主之心，是否适宜，应配合命宫及财帛宫推断。</li>
<li>天府对拱紫微化权者，权力之星太重，有辅弼扶持尚可，否则恐反主孤立。</li>
<li>巳亥宫天府，仅慎之心较小，故事业往往带浮夸性质。有煞者更甚。若天桃、阴煞同度，文曲化忌来会，则易发展为空头，并因此事业失败。</li>
<li>凡天府在巳亥守事业宫，皆以脚踏实地，在现成局面下稳步发展为宜。</li>
<li>若吉星祥曜会合，命宫成火贪格者，则主事业突然发展，在流年大运良好时，亦可投机致富。</li>
</ul>

太阴

<ul class = "ds-中州派分析段落">
<li>太阴主富不主贵，故守事业宫时，必以见文昌文曲同会，然后始主其人有社会地位。</li>
<li>太阴会科文诸曜，宜文艺、传播事业，亦可从政，其政务亦必与文化有关。所谓科文诸曜，乃指昌曲、龙凤、天才、化科而言。</li>
<li>若见科文诸曜，又见桃花诸曜来会，则主艺术成名、更宜表演艺术。</li>
<li>太阴与魁钺或辅弼同会者，可从政，但必须有相当学历配合，然后始主得所擢升而任高位，否则徒有虚声，而升迁至一限度即便因循不进，以致有怀才不遇之感。</li>
<li>太阴入庙，或「机月同梁」格，得太阴化禄，天同化权，天机化科，而不见巨门化忌，又不见煞曜者，则反宜任职军警界，或刑法纪律部队，可得掌大权，此尤以夜生人为贵，日生人次之。</li>
<li>太阴落陷守事业宫，终生事业变动多端，财帛更难积聚。</li>
<li>若太阴落陷，文曲化忌同度，则为江湖落拓文士。</li>
<li>太阴与空劫同会，有科文诸曜合，可突破行业成规，有发明创造，或为工艺发明家；无科文诸者，则理想大于现实，无论从事何种行业，均易流为空虚不实。</li>
<li>成「机月同梁」格者，如无禄权科会，则必先祥太阴为入庙抑为落陷，入庙者尚可在大机构企业独当面，落陷者只宜因应时势，随遇而安。</li>
<li>太阴为财星，主计划而不主行动，故守事业时，即使所任职务与财政有关，亦仅宜计划策划，而不宜实际行动。</li>
<li>太阴入庙，得禄，无煞或煞少，与天机化忌同会合，反宜从事投机事业。但必须大运或流年配合。</li>
<li>太阴会巨门化忌者，主事业伤尽脑筋，或须费尽唇舌。</li>
<li>太阴与天机同度或相对，见吉化吉曜，宜从事工程工艺，反不宜投机，唯所从事工艺工程，仍带流动色彩，如航海工程、航空工程、运输工业等。</li>
<li>若天机化禄，太阴化忌，则绝不宜投机。盖太阴化忌守事业宫，有投资方向错误，或受人引诱误导而致投资损耗。</li>
<li>太阴会天梁化科，宜从事监督、管理工作，或如公证行之类。    </li>
<li>太阴最喜欢天同化禄，则亦主白手与家。唯若天同化忌者，有煞，主事业挫折；无煞，则由低层发展，终能成就。</li>
<li>卯酉、巳亥宫太阴见科文诸曜亦宜从事广告或传播，此以见太阳及天梁吉化者始成大器。</li>
</ul>

<ul class = "ds-中州派分析段落">
<li>贪狼守事业宫，主其事业倾向于消费、交际，或由社交而成事业。化禄、化权者，此项性质尤重，故绝对适宜于外务。</li>
<li>贪狼与昌曲同度，最宜公开事业，或可从事带娱乐性之传播事业。</li>
<li>若与桃花同度，则其事业以多与异性接触者为宜。否则宜从事消费，装饰或娱乐事业。</li>
<li>贪狼如与空劫同度，须视有无昌曲，有则为艺术，无则宜从事实业。</li>
<li>贪狼亦带艺术性质，但廉贞静态，贪狼却为动态。二者基本性质不同。</li>
<li>在现代，贪狼亦主运动，或带韵律性之运动（如芭蕾舞）。故若贪狼化权，见煞曜而不多者，往往主运动场上竞争。</li>
<li>贪狼与吉化、吉曜会合者，又见三台八座、台辅封诰、天官天福、恩光天贵等吉曜者，亦可从政，且主折衷调和，或为各派系势力之缓冲人物。</li>
<li>「火贪」、「铃贪」于辰戌丑未四宫，乃主持掌兵权，或在警察纪律刑法部门任职。</li>
<li>贪狼与羊陀同度，则以从商为宜。亦须从事消费行业或行业性质与装饰、娱乐、消费品有关。</li>
<li>贪狼与左辅右弼同度，则主下属众多，故所经营之事业亦必有相当气派。</li>
<li>贪狼亦为饮食之星（廉贞主血，贪狼主肉），不化忌，便主有饮食品味。见火星、天厨，可从事饮食业工作。</li>
<li>贪狼化忌在事业宫者，主凡插手事业必引起竞争。有煞曜同度者更甚，空劫同度则主因竞争而遭挫折。</li>
<li>若贪狼化忌有吉曜同会，则理想之事业虽不能达到目的，但却别有天地，可由此创建新猷。</li>
</ul>

巨门

<ul class = "ds-中州派分析段落">
<li>巨门守事业宫，须视太阳。太阳入庙会照则吉，若太阳落陷，则格局较下。</li>
<li>凡巨门在事业宫，基本性质，为「口舌生财」。故以口才为主，或须费唇舌以谋生。格局有高下，唯基本性质不变。</li>
<li>尤其当巨门化忌之时，若不从事依赖口才，或费劳唇舌的行业，则反招是非口舌。即使巨门为入庙的太阳会照，亦仅主是非较轻而已。若见四煞、空劫、刑耗，更多事业上的纷扰，或且牵涉讼事。</li>
<li>若从事口舌职业，虽巨门化忌，只要煞刑诸曜不重，亦仅主事业不稳定，且劳心费神，费尽唇舌而已，并不主官非词讼。</li>
<li>属于是非口舌的行业，为律师、外交家、政界、教师、帮会、星相、推销之类。以巨门化禄为贵，化权亦可，化忌带偏门，或一半牵涉政治。</li>
<li>口舌引伸为饮食，故亦可从事与饮食有关的行业。若从医，则与口腔，食道有关（天同巨门则为脑科、神经科。）</li>
<li>巨门喜化禄，或与禄存同度，则有时可将口舌的性质转化为声名。但以不主动出锋头为宜。所谓声名即口碑，与特意寻求声名者截然不同。</li>
<li>巨门与火铃同度，亦主横得横失，暴起暴跌。故以收敛锋芒为宜，且贵知机。</li>
</ul>

天相

<ul class = "ds-中州派分析段落">
<li>天相守事业宫，无论处何地位，均不宜在名义上占取最高位置，以「副贰」为宜，否则必受人攻击，排挤。</li>
<li>天相带服务性质，然而其所谓服务，绝不同于「太阳天梁」的社会服务，往往有政治色彩。或直接从政，或所业有政治背景，或虽非政界，但却对政治有浓厚兴趣。</li>
<li>天相本身无吉化，但却喜与吉化之星同度（由此可见其缺乏独立性）。或三方有吉化，亦为天相所喜。</li>
<li>会廉贞化禄、武曲化禄，古人认为皆宜参赞戎行。在现代，却可能为贩卖军需以至军火起家。亦主半商半政，长袖善舞。但若见煞者，则仍以不占手政治为宜。</li>
<li>会破军化禄，事业带开创性，但所开创的事业却不属于自己，往往一业开创即成，便须另调新职，负担新的开创业务。只有当所会的天府有禄存同度，或天相本身为财荫所夹之时，然后才能享受开创的利益。</li>
<li>破军化禄另有一重性质，即宜作为商品代表，此即「副贰」之意。所谓开创性，此时便变为打开代理商品的市场。如此则所得利益便应较大。</li>
<li>天相亦可借会贪狼化禄，则常主背井离乡发展。</li>
<li>天相与紫微化权相会，名大于利，亦权大于利。倘见煞曜，则不宜弄权，否则事业必多挫折，或主倾覆。</li>
<li>天相与紫微化科相会，亦名大于利，虽独当一面，但却往往易受架空，必须同时见辅佐吉曜然后始有主有名可实。</li>
<li>天相与羊陀火铃同会，或为羊陀夹、火铃夹者，事业多纠纷，且易受人掣肘。</li>
<li>天相守事业宫而为刑忌夹者，更见煞曜同度，半生牛马。有吉曜同会，亦时受压力。</li>
<li>天相与空劫同度，吉则可以凭意念打出天下，唯仍须与人合作。若无吉曜，或且见凶星恶曜者，则或为志大才疏，或不守一业，时生改变之心，以致反流离不安。</li>
<li>天相有禄马冲拱，亦主离乡。</li>
<li>天相守事业宫，是否能成富，须视天府所会合之星曜而定。「逢府看相」，逢相亦须看府。盖天府为库廪，天相为锁匙，有锁匙而库廪空虚，亦自徒然。</li>
<li>凡天相守事业宫，其行业性质，须由命宫与财帛宫决定，有时亦由夹宫决定。</li>
</ul>

天梁

<ul class = "ds-中州派分析段落">
<li>天梁为清贵之星，纵见禄、权、。科，及辅佐吉曜，亦不负行政之责，仅宜居监察、审核、司法、刑政之职。亦宜为政府代言人。</li>
<li>若在商，则为财务审核、人事管理之才。若为自由职业，则与医药、法律、广告、传播有关。</li>
<li>会太阳入庙，在现代亦为社会工作者，或为议员等清贵。属于半官方人士。</li>
<li>天梁为孤克刑忌之星，上述职务，细心体会，便知其带孤克刑色彩。盖此等职务，皆须遵循一定法则，或须面对大众，且向大众负责者也。</li>
<li>天梁亦为科学研究之星，因科学研究亦有一定法则，兼且面对大众。</li>
<li>上述职务看似纷纭，如何厘定，无非视所会之辅佐杂曜而定。如天刑为刑法；天月为医药；天巫利从政；禄马宜从商，昌曲宜广告及传播之类，皆有迹象可寻。然而定基本性质，却仍以正星组合为准。</li>
<li>大致而言，同拱太阳者，利传播，司法，医药；同拱天同者，宜内务，秘书，参赞；同拱天机者，宜计划，设计，审核，或半官方事业。</li>
<li>天梁落陷，以自由职业为宜。亦以有年资可积者为宜。</li>
<li>天梁化禄于事业宫，若从事财务稽核之职者，最为适宜。若所业与财务无关，则反因财而受困扰。从政者困扰尤大。</li>
<li>天梁化权，利于权威性工作，法律固有权威，监察及秘书亦未尝无权，于此可细作体会。</li>
<li>天梁化科，以清贵为宜，故须极力争取社会地位，然后财富始随之而来。换言之，名为主，利从于名，本末切不可倒置。</li>
<li>天梁化禄而成「阳梁昌禄」格者，必须以专利发明，专利事业为奋斗目标。若从商，亦宜代理名牌，或打开商誉，然后始能有所成就。盖此组星曜与事业关系甚大，本质虽佳，却必须努力奋斗然后有为。</li>
<li>天梁在巳、午、未三宫，孤克刑忌的性质特别大。尤以在巳宫者，从政则有特殊任务，从商亦别有目的手段，是则以同和大众为第一要务。在午宫者，见吉则事业广大，见凶则为众矢之的，亦必须以不针对别人为宜；在未宫者，一生事业变化多端，其变化为吉为凶，亦以所会辅佐诸曜为主，后天人事，亦宜从事行业本身亦具变动色彩者（如电子业），是则为趋吉避凶之道。</li>
<li>天梁与空劫同度，吉则为发明创造，凶则流为空想。此亦可藉后天人事补求。凡天梁在事业宫者，必须努力争取学历、资历，此盖属通则也。</li>
</ul>

七杀

<ul class = "ds-中州派分析段落">
<li>七杀居事业宫，主管理能力强，尤以紫微七杀为然。</li>
<li>古代无工业，最能聚众，且最能发挥管理能力者，厥为兵戎，故古人论七杀守事业宫，必许以武职荣身。现代则可为工厂管理，亦可为大企业人事部门。</li>
<li>凡管理，必须恩威并济，七杀守事业宫者，威有馀而恩不足，「紫微七杀」尤然。故以得辅弼、魁钺、三台八座朝拱同会者为贵，得此则自能则柔相济，虽千万人皆能号令。</li>
<li>七杀不宜与煞曜同度，火铃同会者更不宜，从事武职或人事管理，皆主生枝节，甚且有意外，或风波重重。</li>
<li>七杀有煞曜同度，最宜一技傍身，故必需学习专门技术，不宜管理。</li>
<li>故七杀守事业宫者，以本宫无煞始宜武职。若有煞而从事军警刑政、情报治安等业，则应居安思危。煞忌刑重者，且多意外，若会廉贞化忌，或武曲化忌，尤防有生命危险。</li>
<li>七杀守业宫者，亦主辛劳，凡事必须亲力亲为。即使下属众多，仍难安逸。</li>
<li>七杀与昌曲同度，以设计工作为宜。</li>
<li>七杀与空劫同度，吉则有巧思，出人意表，凶则一生多空想，以致不良不莠。</li>
<li>七杀与空劫同度，不宜投机，即使命宫见「火贪」、「铃贪」，亦主暴发暴败耳。</li>
<li>七杀守事业宫者（命宫亦如之），必须注意后天培养，尤以人和为第一，技能训练次之。缺此二者，则虽有作为，但同一星系，由于后天修养不同，影响实际命运甚大。</li>
<li>七杀守事业宫而有煞者，见擎羊、天刑，可习外科手术、针灸、妇产科、牙科等。廉贞化忌者尤宜。</li>
<li>若有煞而无化忌会合者，则仍以应用利器为主的行业为宜。如雕刻、理发及金属工程等。</li>
<li>七杀与火铃同度，则以见金火的行业为佳，如冶炼、烹饪、首饰之类。</li>
<li>子午宫七杀，与「武曲天府」相对，有煞者最宜工艺设计，主有独特概念，易为欣赏。</li>
<li>若辅佐诸曜同会者，则亦可主持企业或工业管理。或为军警武职。</li>
</ul>

破军

<ul class = "ds-中州派分析段落">
<li>破军守事业宫，大致略同七杀，其主要分别为七杀之开创力不如破军，然而风险亦较破军为小。七杀守事业宫，虽必有一段时期困境，只需不投机，纵有挫折，终能东山复起，而破军则可能一蹶不振，且于顺境之时，亦必见风波，仅能安然渡过而己。</li>
<li>以事业性质言，破军不与煞忌同躔，三方亦煞轻者，可以武职荣身。化禄、化权尤佳，更见辅佐吉曜同会，则富贵不小。</li>
<li>若破军与煞曜同度，则宜以工程技术起家。从商者必多挫折，纵能化险为夷，亦劳心伤神。</li>
<li>破军入庙而吉煞交集者，可以经商，是则成败无常，进退之机异常重要。往往成败仅系于一线。</li>
<li>以毅力言，破军优于七杀。故当破军守事业宫时，只须连续有三四年良好的流年，即能创出事业基础。</li>
<li>若破军化禄，则主兼行兼业，兼职兼差，且往往由不以为重点的行业或职务起家。</li>
<li>破军守事业宫，会「火贪」、「铃贪」往往于困苦艰难中突发。突发之后，若不亲力亲为，或贪心无厌，则又易归于失败。</li>
<li>破军宜于工业，其性质亦同七杀。但七杀之特长在于管理，而破军之特长却在创新。故宜注意新产品，新技术，或从事冒险性行业。</li>
<li>廉贞与七杀同度，宜从事工程，与破军同度，却反主处行政机构之下僚。此乃由于廉贞火可炼七杀之金，耐却与破军水火相克之故。不成格局，是故境况亦大不相同。</li>
<li>破军喜见辅弼，则能减少辛劳。见魁钺不过多机会而已，见昌曲亦不能扭转劣境。若昌曲化忌，则更易不安所业，不良不莠。</li>
<li>破军与桃花诸曜同度，倾向于半偏行业，若更见文曲化忌者，则带巧诈性质。</li>
<li>破军与羊陀同度，若无一技防身，则必终生不能有为，以致晚年困苦。</li>
<li>破军亦不喜与火铃同度，主风波叠起。</li>
<li>与空劫同度，见吉辅者，创新之能力极强，此时却又喜昌曲相济。</li>
</ul>
