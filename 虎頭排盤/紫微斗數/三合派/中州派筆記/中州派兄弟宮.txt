紫微
子、午：紫微

<ul class = "ds-中州派分析段落">
<li>子午宮紫微獨坐，躔桃花諸曜，兄弟行中以長女為先。或與長兄不和。</li>
<li>子午紫微見「百官朝拱」，則長兄長姐得力。唯必須甘心處於「佐貳」之副手地位，始可與人合作共事，否則於大限流年之「兄弟宮」，行至「武曲天相」同度宮垣，或破軍獨坐宮垣，見煞忌，合作關係會決裂。</li>
<li>子午紫微為「在野」或「無道」之格局，兄弟難靠。與人合作易受人剝削。</li>
</ul>


丑、未：紫微、破軍
<ul class = "ds-中州派分析段落">
<li>丑未「紫破」同度，見煞，兄弟各結成黨。或刑克不和，或有異胞兄弟。將此意義推廣，自己與共事的同事關係，亦易成黨派，且主政治、權力鬥爭。見煞曜尤確，見輔佐吉曜則較平和。</li>
<li>丑未「紫破」有煞忌刑曜，必於流年兄弟宮為「紫破」或天相時，發生不良克應。倘若「紫破」原有吉曜會照，則於流年兄弟宮行至「武曲七殺」或天府獨坐時，有良好克應，可以合作創業。</li>
</ul>


寅、申：紫微、天府
<ul class = "ds-中州派分析段落">
<li>寅申宮「紫府」同度，亦主長兄長姐得力；見煞忌刑曜則反受剝削、拖累。亦可將此性質推廣至合作夥伴或同事間關係。</li>
<li>寅申「紫府」同度，吉者，每於兄弟宮行至武曲、太陽、巨門、太陰等宮垣克應；凶者，每於兄弟宮行至破軍、七殺、廉相、巨門等宮垣克應。</li>
</ul>


卯、酉：紫微、貪狼
<ul class = "ds-中州派分析段落">
<li>卯酉宮「紫貪」守兄弟宮，見桃花諸曜主兄弟行中先見長女；見空曜則主兄弟行中先見長兄。</li>
<li>卯酉宮「紫貪」多吉曜，亦主兄長得力；倘見火貪則富裕，唯反與自身無益。若煞忌刑耗並見，主受累傾家；更見桃花諸曜，又見輔佐單星，主亂倫。</li>
<li>卯酉宮「紫貪」，與異性合作時每易生情愫，且易招不利。克應的年份，每為天同、巨門守兄弟宮之時。</li>
</ul>


辰、戌：紫微、天相
<ul class = "ds-中州派分析段落">
<li>辰戌宮「紫微天相」，亦主有異母兄弟；倘見魁鉞、輔弼、昌曲，則往往有雙胞。</li>
<li>辰戌「紫相」，見吉曜則主和好合作，見凶曜則各懷異見。皆每於流年兄弟宮行至「紫相」、破軍、「武府」的年份克應。若折產破裂，則於「機巨」、天梁的年份克應。</li>
</ul>


巳、亥：紫微、七殺
<ul class = "ds-中州派分析段落">
<li>巳亥宮「紫殺」，見吉則兄弟有權貴，亦主可依附權貴之友人。若見煞，則受欺負。</li>
<li>巳亥宮「紫殺」，會「武貪」與火鈴同度，更有吉曜，主兄弟或友人提攜自身發跡。</li>
<li>巳亥宮「紫殺」，吉則於天相、「巨陽」的宮垣克應；凶則於「武貪」、「廉破」的宮垣克應。</li>
</ul>



天府
<ul class = "ds-中州派分析段落">
<li>凡天府守兄弟宮，府庫不宜「空露」，否則主受兄弟或友人剝削連累。</li>
<li>凡天府守兄弟宮，會天姚、天虛、大耗，主兄弟或友人權術陰謀。</li>
<li>天府守兄弟宮，見輔佐諸曜中之一對，卻又另見「單星」，主兄弟各胞。</li>
<li>天府守兄弟宮，見煞忌刑曜主兄弟少或主多而刑克；見輔佐諸曜始主兄弟眾多。</li>
<li>天府守兄弟宮，最嫌會廉貞化忌，主兄弟或知交反目，恩反成仇。亦嫌會武曲化忌，主為兄弟或友人破財，煞重則為爭財之兆。</li>
<li>天府獨守兄弟宮在丑未者，吉凶多在天同、巨門，「武破」宮垣為兄弟宮時克應；在卯酉二宮者，吉凶克應之期，為「紫破」、「廉貪」、天相守兄弟宮之年限；在巳亥二宮者，吉凶克應之期，多在「天同太陰」、「太陽巨門」守兄弟宮之年限。須詳原局兄弟宮及克應星曜組合而視吉凶深淺。</li>
</ul>

廉貞
<ul class = "ds-中州派分析段落">
<li>凡廉貞守兄弟宮，兄弟宮及朋友之間，以「君子之交淡如水」為宜。除非吉化吉曜並集，則主得助力，此時即應謀善處。</li>
</ul>


子、午：廉貞、天相
<ul class = "ds-中州派分析段落">
<li>子午宮「廉貞天相」在兄弟宮，無煞則主感情融洽，且亦有友人助力。交遊面不太廣，但亦多朋友。</li>
<li>子午「廉相」見煞忌刑諸曜重重同會，則主兄弟中有殘疾者；與人結交，愈親密愈易反目成仇，煞輕則減等。反目克應，為太陽、天梁、巨門守兄弟宮之年。</li>
<li>子午「廉相」，結交不宜太濫，否則易結黨紛爭。則以破軍為克應之期。</li>
</ul>


丑、未：廉貞、七殺
<ul class = "ds-中州派分析段落">
<li>丑未「廉殺」在兄弟宮，見權祿及吉曜者，兄弟融洽，見煞曜則主表面和洽，助力不足；若更見忌星，則主因兄弟或友人而受損傷、破耗。以兄弟宮見煞忌刑耗之年為克應。</li>
<li>丑未「廉殺」見輔佐吉曜及吉化，可得友人合作之力，因而帶挈自身。</li>
</ul>


寅、申：廉貞
<ul class = "ds-中州派分析段落">
<li>寅申廉貞獨坐兄弟宮，亦主兄弟感情和美，但無實際助力。與人交往，交深則反易招怨。故以不與人合作事業為宜。</li>
<li>寅申廉貞，每易於破軍或「紫相」守兄弟宮的年份，與友合作事業，如見煞曜，則不出三年便告反目，或各懷心病。</li>
</ul>



卯、酉：廉貞、破軍
<ul class = "ds-中州派分析段落">
<li>卯酉宮「廉貞破軍」同度，見火鈴，兄弟或知交成仇。見吉曜及吉化，則可以融洽持久，且亦可合作營運事業，主得力。</li>
<li>卯酉「廉破」見輔佐單星，主兄弟各胞，亦主牽涉及友人之紛爭。如廉貞化忌，則受累甚大。以有忌星衝起廉貞化忌之年為克應。</li>
</ul>


辰、戌：廉貞、天府
<ul class = "ds-中州派分析段落">
<li>辰戌「廉府」守兄弟宮，兄弟多，交遊亦較廣。唯必須天府不「空露」，然後始吉，否則反主受剝削拖累。</li>
<li>辰戌「廉府」有煞忌，則兄弟間配偶易生不和，或友人間配偶不和。</li>
<li>辰戌「廉府」見咸池、大耗、天姚、沐浴，命宮見紅鸞天喜，主易與兄弟或事業夥伴之配偶暗戀。見煞則由是惹起大禍。</li>
</ul>


巳、亥：廉貞、貪狼
<ul class = "ds-中州派分析段落">
<li>巳亥「廉貪」守兄弟宮，兄弟或知交反易生尤怨。故以不與人合作事業為宜。</li>
<li>巳亥「廉貪」見貪狼化忌，主受兄弟或合作夥伴支排差遣，但仍受怨責。唯夥伴之工作能力則甚強。故須注意善處之道。</li>
<li>巳亥「廉貪」見廉貞化忌，又見煞，則尤怨甚深。</li>
</ul>


武曲
<ul class = "ds-中州派分析段落">
<li>凡武曲在兄弟宮，見擎羊或火星同度，皆主爭財。與煞曜空劫同度，兄弟朋友稀少，自身常感孤立。</li>
</ul>


子、午：武曲、天府
<ul class = "ds-中州派分析段落">
<li>子午「武府」守兄弟宮，吉曜多者始主和好，稍見煞曜，即主面和心不和。</li>
<li>子午「武府」見吉化亦不為助力，除非命宮見輔弼，然後始有兄弟友人相助。見煞忌刑曜，受兄弟友人拖累。不見祿，則受剝削爭奪。故凡「武府」守兄弟宮，一般不宜與人合作。以「機巨」、破軍、「紫相」守兄弟宮之年為克應。</li>
</ul>


丑、未：武曲、貪狼
<ul class = "ds-中州派分析段落">
<li>丑未「武貪」，見吉化、吉曜，則可受兄弟友人之益蔭，但不可爭權。見煞忌刑曜，則反主益蔭兄弟及友人。未宮優於丑宮。</li>
<li>丑未「武貪」與火鈴同度，無刑忌，自身命宮又會吉曜吉化者，主因兄弟或友人而橫發。</li>
<li>丑未「武貪」見煞忌刑，自身命宮又弱，則主一生受制。</li>
</ul>


寅、申：武曲、天相
<ul class = "ds-中州派分析段落">
<li>寅申宮「武相」同度，見吉則主助力，見煞（或「刑忌夾印」）則主動助友人反受連累。常以「同巨」宮垣為克應之年。</li>
<li>寅申「武相」又主兄弟或友人可以技藝相助。</li>
<li>寅申「武相」多桃花，又見煞忌，則配偶易受自身友人挑引。</li>
</ul>


卯、酉：武曲、七殺
<ul class = "ds-中州派分析段落">
<li>卯酉「武殺」守兄弟宮，見煞，主兄弟朋友間爭財。見輔佐吉曜，吉化，然後有短暫時期助力，但仍不持久。</li>
<li>卯酉「武殺」，若與人合作，須防弄權侵吞。故以不與人合作為宜。</li>
<li>卯酉「武殺」不利之克應，常為天機、巨門、「紫破」、「同梁」躔兄弟宮之年。</li>
</ul>


辰、戌：武曲
<ul class = "ds-中州派分析段落">
<li>辰戌宮武曲，見吉曜則主兄弟及友人和好，助力仍少。見煞忌刑者，則無助力，甚至生連累。羊陀夾武曲化忌，因兄弟友人而自身受傷，或破耗甚大。</li>
<li>辰戌宮武曲，見煞，主兄弟友人奪權。故不宜合作。不利之克應，為武曲、破軍、七殺躔兄弟宮之年。</li>
</ul>


巳、亥：武曲、破軍
<ul class = "ds-中州派分析段落">
<li>巳亥「武破」守兄弟宮，即使見吉曜，亦不主兄弟友人得力，合作亦不耐久。見煞忌刑曜，更主受掣肘。唯交異族、異鄉友人，情形則較佳。</li>
<li>巳亥「武破」會廉貞化忌，「武破」又與火鈴同躔，主因兄弟或友人而生血光之災。常以「廉殺」、「武破」躔兄弟宮之年為克應。（有過一個實例，是為友人輸血。）</li>
</ul>


天相
<ul class = "ds-中州派分析段落">
<li>天相臨兄弟宮，須注意「刑忌夾印」，及「財蔭夾印」兩個格局。不能單以天相會照的星曜而定。若「刑忌夾印」，主受兄弟或友人拖累甚深；若「財蔭夾印」，則主受兄弟或友人益蔭。以「夾印」之年為克應。</li>
<li>天相對武曲、廉貞、破軍三曜至為敏感。此三曜化吉則吉，化忌或加煞則凶。</li>
<li>天相見昌曲、左右相夾，或同會，無虛耗刑煞者，主兄弟雙胞。但必須家族有雙胞胎傳統者始是。</li>
<li>天相守兄弟宮，對煞曜至為敏感。四煞空劫多者，主孤獨，亦主有祀入之兄弟；或主自身祀出。</li>
<li>丑未宮天相，吉凶克應之年，以天相、巨門、「同梁」守兄弟宮之年為克應。卯酉宮天相，吉凶克應之年，以「機梁」、天相、「廉破」守兄弟宮之年為克應；巳亥宮天相，吉凶克應之年，以天相、巨門、「廉殺」守兄弟宮之年為克應。</li>
</ul>

巳、亥：天相



七殺
<ul class = "ds-中州派分析段落">
<li>七殺守兄弟宮，見祿、權、科及輔弼，兄弟眾多。命宮見輔弼，則兄弟宮之七殺雖不會輔佐諸吉，亦主兄弟眾多，但有刑克或不和。</li>
<li>七殺守兄弟宮，煞曜聚合者，則主孤單。更見天刑乃主刑克。否則兄弟自小分離。</li>
<li>若七殺在兄弟宮有煞刑諸曜，而命宮又見輔弼諸吉，則或為兩重父母，例如本身多兄弟，祀出后無兄弟。又或為本多兄弟而刑克至孤單。--凡言刑克，終身殘疾者亦為刑克之應。</li>
<li>凡七殺在兄弟宮者，命宮必有太陰相對，由太陰廟陷，可以幫助推斷兄弟及朋友間之人際關係。</li>
<li>子午七殺守兄弟宮，以「殺破狼」躔兄弟宮之年，為吉凶克應。寅申七殺守兄弟宮，除以「殺破狼」外，亦以「廉相」躔度為吉凶克應之年。辰戌七殺守兄弟宮，以「廉府」、「同巨」躔兄弟宮之年為吉凶克應。</li>
</ul>

破軍
<ul class = "ds-中州派分析段落">
<li>破軍守兄弟宮，多主有缺點，會煞曜主孤單，更會刑忌則兄弟刑克，知交反目。即使會輔佐諸吉，亦不利長兄或長姊。若與祿存同度，則且需獨立負起長子或長女之責任。與人合作，亦主開創而不主收成。</li>
<li>破軍守兄弟宮，除非孤單，否則兄弟年齡有差距，由八年至十二年以上不等。</li>
<li>破軍守兄弟宮，火鈴同度更見空劫，主兄弟有兩重父母（如祀出、過繼、童年受拋擲），否則主一生分離異地。故見此格局，不宜與人合作，否則易中道破裂。</li>
<li>破軍守兄弟宮而輔佐吉曜咸集，則可藉他人之力創業。（注意祿存同度時情形恰恰相反。）</li>
<li>子午破軍守兄弟宮，吉凶克應，為破軍、七殺、「廉相」躔度兄弟宮之年。</li>
<li>寅申破軍守兄弟宮，以「武相」、破軍、「廉府」躔度兄弟宮為克應之年。</li>
<li>辰戌破軍守兄弟宮，以「紫相」、破軍、七殺躔度兄弟宮為克應之年。</li>
</ul>

辰、戌：破軍
巳、亥：破軍、武曲



貪狼
<ul class = "ds-中州派分析段落">
<li>貪狼守兄弟宮，除巳亥二宮為貪狼陷地外，一般情形下均主兄弟和睦，亦主人交遊廣闊。唯不見吉化吉曜者，則多酒肉之交。</li>
<li>貪狼於巳亥陷地，「對星單見」，又見桃花者，多主有異胞兄弟；見煞，則兄弟知交多生尤怨，或各懷異心。若貪狼化忌，主受累。</li>
<li>貪狼守兄弟宮，若與火鈴同度，見吉，主受兄弟或合作夥伴益蔭，見煞忌，則反主突然受累，事前毫無徵兆。</li>
<li>貪狼在兄弟宮成「泛水桃花」格局者，生平多酒色之友，且因之受累。成「風流彩杖」格局者亦然。</li>
<li>子午貪狼，其吉凶克應，多在「同巨」、破軍守兄弟宮的流年；寅申宮貪狼，其吉凶克應，多在「紫相」、破軍、七殺守兄弟宮的流年；辰戌宮貪狼，其吉凶多在天梁、天機、「廉相」守兄弟宮的流年。</li>
<li>貪狼在兄弟宮，最嫌武曲化忌與貪狼化忌相照，若更見「鈴昌陀武」之格，則主受累傾家，或竟至意外。</li>
</ul>



子、午：貪狼

巨門
<ul class = "ds-中州派分析段落">
<li>巨門於兄弟宮中為惡曜，多主骨肉參商，雖有輔佐吉曜，或見吉化，亦僅主兄弟有成，並不主兄弟感情深厚。稍見煞曜，即多是非。若煞忌刑曜並見，則主受兄弟連累，或致涉詞訟、破耗。故巨門守兄弟宮者，絕不宜與人合作事業。唯「太陽巨門」守兄弟宮者，可與異族人合作。</li>
<li>「巨門交人，始善終惡」。故巨門在兄弟宮者，不宜酒肉深交，以君子之交為宜。</li>
<li>巨門守兄弟宮，場面上朋友多，一深交即易涉及陰私，或竟因此致禍。以見煞忌刑曜者為確。</li>
<li>子午巨門，以「同梁」、天機、巨門守兄弟宮為克應之年；辰戌巨門，以巨門、天同、「機陰」守兄弟宮為克應之年；巳亥巨門，以天機、天梁守兄弟宮為克應之年。</li>
<li>巨門有兄弟不和，骨肉參商的性質，但如與異族人交，則反而可互相利用，尤以與太陽同宮或相對時為確。但太陽必須入廟。</li>
</ul>

太陽
子、午：太陽
<ul class = "ds-中州派分析段落">
<li>子午太陽守兄弟宮，居子宮者無力，居午宮者則易貴而不實，各有缺陷。</li>
<li>子宮太陽易受友人挑剔；午宮者則得諍諫之友。見煞忌，長兄死亡。或主因兄弟友人之事牽涉風波。</li>
<li>子午太陽，兄弟知交皆易離散。見吉，則離散后仍主助力。</li>
<li>子午太陽，以天同宮垣為不利事端克應之期。</li>
</ul>

丑、未：太陽、太陰
<ul class = "ds-中州派分析段落">
<li>丑未「太陰太陽」同守兄弟宮，一般情形下主兄弟朋友眾多。入廟見輔佐諸曜則有助力；落陷見煞忌，則主生損害、刑克、剝削。吉凶交雜，多泛泛之交。</li>
<li>丑未「陰陽」行至「機巨」守兄弟宮的大限流年，絕不宜與人合作，主決裂、拖累。</li>
<li>丑宮者，若太陽化忌，主受兄弟或友人傷害；未宮者，若太陰化忌，主為兄弟或友人破財。</li>
</ul>


寅、申：太陽、巨門
<ul class = "ds-中州派分析段落">
<li>寅申「太陽巨門」同度，見吉曜則兄弟友人皆有助力，宜發展彼此有聯繫的業務。</li>
<li>寅申「巨陽」化權祿，主受異族友人提攜。化忌，則主兄弟朋友始善終惡，有極大糾紛。以「機梁」、「同陰」為克應之年。</li>
<li>寅宮者交遊較廣；申宮者，多泛泛之交。諸吉遍集，則主自身為兄弟友人服役。</li>
</ul>


卯、酉：太陽、天梁
<ul class = "ds-中州派分析段落">
<li>卯酉「陽梁」守兄弟宮，見煞主兄弟分離，無助力。見刑忌，主兄弟或友人爭產。見輔佐吉曜，則有可依靠之兄姊，或主得友人提攜。卯宮又優於酉宮。</li>
<li>卯酉「陽梁」化忌，亦主長兄死亡不幸。或有大助力之友人反生拖累。</li>
<li>卯酉「陽梁」，以兄弟宮經行「陽梁」本宮及對宮為克應之年。</li>
</ul>


辰、戌：太陽
<ul class = "ds-中州派分析段落">
<li>辰戌宮太陽，辰宮者，兄弟多和，戌宮多不和。</li>
<li>辰宮者多異性友人，見吉且主助力。</li>
<li>辰宮者主多異族朋友。</li>
<li>辰宮太陽化忌，亦主多異族友人；戌宮太陽化忌，主兄弟朋友多是非口舌。</li>
<li>辰戌太陽見吉曜，皆可與友人合作，唯戌宮者不耐久。以「同梁」、天機、巨門躔兄弟宮為克應之年。</li>
</ul>


巳、亥：太陽
<ul class = "ds-中州派分析段落">
<li>巳亥太陽皆主交遊廣闊。巳宮者主動；亥宮者被動。巳宮者且多異族友人。</li>
<li>巳宮太陽化忌，主知交離散；亥宮者主多是非爭端。</li>
<li>巳宮太陽吉化，見輔佐諸曜，得良好合作夥伴；亥宮者平常。</li>
<li>巳亥太陽，以兄弟宮經行太陽、巨門、天梁、天同諸宮垣，為克應之年。</li>
</ul>


天梁
<ul class = "ds-中州派分析段落">
<li>天梁守兄弟宮，入廟和美，落陷則不和美。會輔弼、魁鉞和美，會煞則不和美。</li>
<li>天梁無吉會，兄弟少，多則為各胞。見桃花諸曜，主姊妹多兄弟少。</li>
<li>天梁吉，則主兄弟或友人暗生助力。兇惡則主暗爭、排擠。吉凶皆在暗面而不在明面。</li>
<li>天梁有孤克之性，所以即使見吉曜，亦不主可與人合作。除非合作者完全退居幕後，作為自己的後台。</li>
<li>天梁對天機、太陰二曜異常敏感，吉化則和美得助，若化忌且見煞曜，則主兄弟或夥伴弄權，或有陰謀詭計。</li>
<li>天梁與天姚同度，主兄弟或夥伴深沉刁狡。更見刑煞，所會的太陽又落陷者，則主詞訟紛爭。見此星曜結構，以不與人合作為宜。</li>
<li>子午天梁守兄弟宮，子優於午。然皆以太陽、天梁躔度兄弟宮之流年，為吉凶克應。</li>
<li>丑未天梁、丑優於未，以「廉相」、破軍、太陽守兄弟宮之年為吉凶克應。</li>
<li>巳亥天梁，亥優於巳，以「紫相」、天梁、「機巨」躔度兄弟宮之年，為吉凶克應。</li>
</ul>

天同
<ul class = "ds-中州派分析段落">
<li>凡天同守兄弟宮，多主兄弟和好，有煞亦多表面和好。且主為人能結交中下層的朋友，唯無太多實際助力。</li>
</ul>


子、午：天同、太陰
<ul class = "ds-中州派分析段落">
<li>子午宮「天同太陰」同度守兄弟宮，兄弟眾多，但主兄弟行中以長姊為大。見煞忌，則兄弟行中長兄難養，以先見姊為宜。</li>
<li>子午「同陰」，見吉僅主兄弟溫馴，友人亦助力薄弱。見煞忌刑，再見空劫，主兄弟中有弱智者，又主受友情拖累。</li>
<li>子午「同陰」，與友人合作，常嫌友人不足以任事。見煞則感情破裂。</li>
</ul>


丑、未：天同、巨門
<ul class = "ds-中州派分析段落">
<li>丑未「天同巨門」同度，見祿必起紛爭；見煞者則易凶終隙末。</li>
<li>丑未「同巨」，見吉亦主一時助力。故不宜有長期合作計劃。</li>
<li>丑未「同巨」居兄弟宮的人，常有與人合資的衝動，則於太陰、天機、「同巨」守兄弟宮的年限，為口舌紛爭拆股之期。</li>
</ul>


寅、申：天同、天梁
<ul class = "ds-中州派分析段落">
<li>寅申宮「同梁」同度，稍見煞曜，即主兄弟或友人表面和好，內心各懷鬼胎。煞重，與兄弟或事業合作夥伴互相傾軋。見吉，則和好而易離散。</li>
<li>寅申「同梁」同度，煞刑並見，無兄弟但有姊妹。見昌曲，或見魁鉞，而不見輔弼，主有異母兄弟。</li>
<li>寅申「同梁」，傾軋離散克應之期，為天相、巨門躔守兄弟宮的年限。</li>
</ul>


卯、酉：天同
<ul class = "ds-中州派分析段落">
<li>卯酉天同獨坐守兄弟宮，卯宮兄弟五人，酉宮僅二三人。亦以先姊后弟為宜，否則長兄難養。</li>
<li>卯酉天同，亦主交遊廣闊，唯實際助力不大。略見煞曜，則多征歌逐色遊樂之友。</li>
<li>卯酉天同見昌曲，姊妹多兄弟少。再單見魁鉞，主各胞。</li>
</ul>


辰、戌：天同
<ul class = "ds-中州派分析段落">
<li>辰戌天同守兄弟宮，兄弟或友人間常多無謂爭執。見煞重，則主受拖累，或生大決裂。</li>
<li>辰戌天同，見輔佐吉曜，亦主兄弟多，或交遊廣，且多與友人合作的機會，但仍必須事事親力親為。</li>
</ul>


巳、亥：天同
<ul class = "ds-中州派分析段落">
<li>巳亥天同獨坐守兄弟宮，亦主兄弟眾多，唯一般情形下姊妹多於兄弟。同時亦主交遊面廣，尤擅結交地位不如己者。</li>
<li>巳亥天同亦主多與事業合作夥伴暗爭。見煞忌刑重尤確。有時又為自身配偶與兄弟或夥伴之配偶不和，引起爭端。其爭端克應之期，為「天機巨門」、天梁躔度兄弟宮之年限。</li>



太陰
<ul class = "ds-中州派分析段落">
<li>太陰守兄弟宮，入廟，夜生人最吉；落陷，夜生人值之，須見吉曜始主兄弟和美；若見煞，則多刑克或不和。</li>
<li>太陰守兄弟宮，日生人值之，雖入廟亦不和美；見煞主刑克、不和；見吉曜始和美，但亦有缺憾。如日生人值太陰落陷，則刑克甚重。</li>
<li>太陰守兄弟宮，有刑者，多主姊妹克應。（太陽守兄弟宮有刑，則主兄弟克應。）</li>
<li>太陰化忌守兄弟宮，無煞，雖主受兄弟或夥伴之累，但多屬無心之失。</li>
<li>太陰在亥宮化忌守兄弟宮，男命，反主得姊妹之力，或宜與女性合作事業。</li>
<li>卯酉宮太陰吉凶克應，以天機、天同、天梁守兄弟宮之年為重。辰戌宮太陰吉凶克應，以「紫破」、「同梁」守兄弟宮之年為重；巳亥宮太陰吉凶克應，以「同巨」、「陽梁」守兄弟宮之年為重。</li>
</ul>

天機

子、午：天機
<ul class = "ds-中州派分析段落">
<li>天機獨坐子午，見輔佐吉曜多者，亦主兄弟姐妹眾多。但一生交遊雖多，卻難得有知交，或與知交半路分張。若見煞忌刑曜，則易受兄弟或朋友拖累，凡事不可妄自出頭。</li>
<li>子午天機，不宜行至太陽化忌、太陰化忌的宮垣。主受兄弟、友人拖累。</li>
<li>子午天機，自身常喜與人合作，除非輔佐吉曜重重，否則必各懷機心，凶終隙末。</li>
</ul>


丑、未：天機
<ul class = "ds-中州派分析段落">
<li>丑未天機守兄弟宮，見煞主刑克，或兄弟有流產小產；見吉亦主早產。亦主與不甚相熟的友人合作，而致損破。</li>
<li>丑未天機見煞忌刑重重者，易與兄弟或友人爭訟。以太陽、巨門、天梁宮度為克應之年。若見吉，則主有兄弟友人為自己奔波。</li>
</ul>


寅、申：天機、太陰
<ul class = "ds-中州派分析段落">
<li>寅申宮「天機太陰」同度，見化忌，則兄弟朋友間皆易生權術陰謀。必須見吉，然後始主平和，但仍少助力。</li>
<li>寅申「機陰」見輔佐吉曜，則一生得兄弟及友人之力。與人合作雖不持久，但時時有使用夥伴。若更見祿馬交馳，則有遠方友人助力。</li>
<li>寅申「機陰」見煞，兄弟及知交流離，不處一方。</li>
</ul>


卯、酉：天機、巨門
<ul class = "ds-中州派分析段落">
<li>卯酉「天機巨門」守兄弟宮，見煞曜主始善終惡。見吉曜，則多短暫的助力，不宜謀求長期合作關係。</li>
<li>卯酉「機巨」，亦主多飲食之友。</li>
<li>卯酉「機巨」不宜兄弟宮行至「殺破狼」的宮度，見煞忌，主乖離、是非、詞訟。</li>
</ul>


辰、戌：天機、天梁
<ul class = "ds-中州派分析段落">
<li>辰戌「機梁」守兄弟宮，主兄弟姊妹有小產流產。亦主與知交或合作夥伴中道分張。見輔佐諸曜僅增加數目，不改變性質。</li>
<li>辰戌「機梁」主有能言善辯之友，見吉曜則諍諫之友。</li>
<li>辰戌「機梁」見吉者，於經行「巨陽」、「武貪」宮垣多主克應；見煞者，於兄弟宮經行「巨陽」、「廉破」、「武貪」宮垣多主克應。</li>
</ul>


巳、亥：天機
<ul class = "ds-中州派分析段落">
<li>巳亥宮天機獨守兄弟宮，即使無煞，亦主兄弟或友人相背，知交寥落。必須見輔佐諸曜，始主交遊廣闊，但知交仍稀。</li>
<li>巳亥宮天機，不宜與人合作，主有權術侵佔之類事端。見吉則多短暫助力。</li>
<li>巳亥宮天機，以兄弟宮行「同巨」、「武相」、「廉府」等宮垣，為吉凶克應之期。</li>
</ul>
