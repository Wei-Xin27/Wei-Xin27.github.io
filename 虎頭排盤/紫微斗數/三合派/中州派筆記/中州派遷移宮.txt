迁移宫所推断的运程，主要为较长时期的外移；现代旅行方便，所以亦可参考流月的迁移宫，推断旅行是否愉快，有无失窃意外。即是说，较长时期的迂移，视大运、流年的迁移宫；旅行则视流月。
若迁移至远地定居，头三年，推算时以迁移宫为重，原来流年命宫可作参考。唯轮排十二宫，视兄弟、父母、夫妻、子女等人际关系，以及财帛、事业运程，则仍依流年命宫轮排。
迁移宫亦用以推断当事人利在家乡发展，或利于离乡背井。在流年，往往又用以推算是否利于转变服务机构。
凡命宫不吉，事业及财帛两宫又不吉，疾厄宫亦不吉，唯迁移宫有吉化吉曜同垣，往往利于迁移以求趋避。反之，则宜静不宜动。本篇所论，即集中于研究迁移是否有利，以及迁移后的现象。所以篇中所论的情形，大致上可通用于星盘、大运、流年以及流月。唯仍应以星盘上的迁移宫作为本质，大运流年等迁移宫则作为本质的反应。故可参考「六十星系」所述，及命宫所述来作推断。
是否能迁移，须见动象始确。凡「叠马」（即原局天马为大运天马冲动，或大运天马为流年天马冲动等）、「禄马交驰」或流年命宫见火铃等，然后始主可以迁动。 
若动不由己，如随家人移居外地、随丈夫移居外地等，则本身命宫或迁移宫虽不动，但有关宫位(如田宅宫、夫妻宫)呈现动象，亦主成行。唯其迁移后的际遇，则仍用迁移宫来作推算。
迁移宫大凶，如见「廉贞七杀」化忌会煞；「铃昌陀武」；「巨火擎羊」之类，则虽有动象亦不宜迂动。亦往往由于有动象，然后才致人因动而陷于险地，不然，其人何致于迁移遇险耶？
见流年禄马交驰，甚至「叠禄」、「叠马」，史宜作为动象看，不可贸然断为在外发财。往往禄马交驰而暗冲煞忌者，实为出外求财反招凶祸之兆，此即《太微赋》所云：「禄逢冲破，吉处藏凶。」人生际遇，大抵每因贪财而罹险境。
故推断迁移宫，可由禄马、火铃，以及正曜动静(如天机主动、天府主静)，来推断有无动象；但迁移后的吉凶克应，则必须详视星系组合而论。不注意及此，往往会推断错误。试想，马交驰主动，倘禄马交驰又主发财，则岂非任何人每有动象，即能在外地发财耶？明白这道理，于推断时才能胸何成竹。

紫微
<ul class = "ds-中州派分析段落">
<li>紫微在迁移宫有百官朝拱，则宜迁移，主有人提携，或有助力；若为在野，或为孤君，则迁移后反主损失、孤立。</li>
<li>紫微与禄存同度，必为羊陀所夹，若羊陀叠冲，则出外主受人排挤；倘同时见吉，则主既有助力，亦有破坏力。</li>
<li>紫微化科在迁移宫，见辅佐吉曜，主在外有人敬重；化权，只宜凭一己力量打开局面，助力反少。</li>
<li>紫微躔擎羊主受打击；躔陀罗主惹麻烦；躔火铃主惹是非；躔空劫主受挫折。躔桃花诸曜主在外有感情困扰。</li>
<li>紫微在迁移宫，同会刑忌夹的天相，或再见天刑、大耗、天虚、空劫等曜，主在外受压力，或致发生纠纷、是非。</li>
<li>紫微与火、铃同度，又见天马，往往主迁移不只一度，即一迁再迁，与家人不聚一地。</li>
<li>紫微在迁移宫，喜辅弼夹；魁钺夹；昌曲夹。分别主得助力；得人提携，得人欣赏。不喜火铃、羊陀、空劫夹，其性质与同度相同。</li>
<li>紫微在迁移宫见桃花，又见昌曲，主在外有艳遇。若同时见煞忌，则在外易因色招祸，以致破财损失；再见刑劫，主因色惹官非；见虚耗天月病符，主因色致病。</li>
</ul>

天机
<ul class = "ds-中州派分析段落">
<li>天机居迁移宫，主利出门离乡发展。此乃由于星曜性质浮动之故。但却不一定离乡定居，有时仅为从事经常需要离家的行业。在古代，为驿夫、水客；在现代可为海员、航空、运输等业。其至有时仅主其人经常转工、转业。</li>
<li>迁移宫天机与火铃同度，主出外奔忙，风露辛酸，不安一地；与羊陀同度，主是非口舌；与空劫同度，主阻碍挫折。若诸煞交并，又见刑耗诸曜，则主破财或意外。尤其是当天机与天梁同度或对拱时，更主出外多困扰虚惊，但终能化解耳。</li>
<li>迁移宫天机见禄马交驰，则虽主经常迁动，但却多享受；若见天马而不见禄存，又躔煞曜，则在外必每多劳而无功，或奔波劳累，不守一地。</li>
<li>天机在迁移宫，与昌曲同度，不宜更见桃花，否则不但浮动不安，而且往往因情愫而羁留异地。见吉尚主在外成家，见煞则在外破耗。</li>
<li>天机在迁移宫与诸吉同会，则大利离开出生地发展。</li>

太阳
<ul class = "ds-中州派分析段落">
<li>太阳宜动不宜静，宜发散不宜收敛，居迁移宫，主不利祖业，离乡背井发福。</li>
<li>古代女命太阳居迁移宫，离家迁移，有时亦为再嫁之兆，这是因为古代女子无事业，故迁动不主开创。</li>
<li>太阳入庙守迁移宫，则出外发达，能有所表现；若落陷，则浮动不安，奔波劳碌。</li>
<li>太阳主贵，亦即对社会地位的争取相当重要，居迁移宫亦有同样意味，故往往表现为适宜在外求学、进修，增加资历，未必适宜在外经商。</li>
<li>命宫无正曜，对宫「太阳天梁」、「太阳太阴」、「太阳巨门」，皆宜离乡背井，移根换叶。若命宫有火铃者，更主他乡即是故乡。禄马同会命宫或迁移宫，亦主一生事业皆宜在异域，或受外邦人支持。</li>
<li>太阳落陷，又见煞忌刑耗诸曜，则不利迁移，往往徒劳无功，而且易招是非、破耗、挫折。</li>
<li>凡太阳居迁移宫，仅利于发贵求名，不主聚财。此乃基本性质。故除非有其他主财帛的星曜会合，即使太阳化禄，亦不主迁移可以致富。</li>

武曲
<ul class = "ds-中州派分析段落">
<li>武曲守迁移宫，主迁移异地后必多忙碌。能否有收获，则须视是否会吉曜而定，尤喜见禄，则主在外贸迁得利。若见煞忌刑耗诸曜，主在外多是非挫折，以致悲观，往往淹留异乡。</li>
<li>武曲与贪狼同度或相会，最宜在外经商，见吉化吉曜，或见「火贪」、「铃贪」，可主横发资财，同时见煞，则横发横破。</li>
<li>武曲在迁移宫化忌者，每主在异乡流落。见天月，则主染病异乡。羊陀夹者，主客死异乡。</li>
<li>若非「武贪」的格局，则不宜躔火、铃二曜，若更无吉曜同会，则主到处飘流，不居一地。倘同时又见禄马，则主因其本身职业须时变换环境。</li>
</ul>

<ul class = "ds-中州派分析段落">

<li>若与羊陀同度，则出外多是非，或惹灾厄。更天同化忌，见天刑者，是非必大，或且惹官非。若见天月则主染病他乡。</li>
<li>天同化忌而与空劫、大耗同度者，主出外破耗，或旅途失财，文曲化忌同度，则主被盗。</li>
<li>与羊陀、火铃、空劫同度，而天同吉化；或天同与天梁对拱者，则虽有灾厄而终能化解。如被盗而能寻回失物，有官非而终能和解，有是非词讼反因祸得福。</li>
<li>天同吉化，或与吉曜同会，则迁移大利，见魁钺者更主得贵人提拔扶持。</li>
<li>天同躔禄存，天梁躔天马，出门求财大利，但迁移必不只一度，或主迁移后仍得远方之财。古人谓主「客中作客得财。」</li>

廉贞
<ul class = "ds-中州派分析段落">
<li>廉贞守迁移宫，通常均主宜离开出生地发展。尤其星当命宫星曜较迁移宫之廉贞星系为弱时，更非离出生地发展不可。例如天相坐命，羊陀夹，对宫为「廉破」与禄存对拱，即为命宫不如迁移宫之一例。</li>
<li>廉贞坐迁移宫，古代认为利于仕宦，尤其是「廉贞天相」的组合，为离乡出仕，或任职僚幕之兆。在现代，可将大企业机构的行政管理职务，视为与僚幕有同一性质。</li>
<li>廉贞化禄，昌曲同度相会，古代认为宜离乡占藉参加典试，可中式，在原藉典试则不中式。现代亦可视为赴外邦求学的徵兆。</li>
<li>迁移宫廉贞化忌，有羊陀夹；或与羊陀借冲（一躔廉贞，另一经借星安宫后恰能与廉相会）；或羊陀与廉贞同度有流羊流陀冲起，更与忌星叠冲（尤忌冲武曲化忌），或会刑忌夹的天相，凡此格局，更见天刑、阴煞，均主出门死于外道，或惹凶祸灾非。</li>
<li>若仅廉贞化忌，见煞曜，则仅主出外因游乐应酬而生是非；见空劫，则主因争财而致祸。</li>
<li>廉贞守迁移，有天月同度，又见煞，逢化忌冲起，主染病他乡。</li>
<li>廉贞化禄，只主在外得意，并不主发财。亦主多人缘。</li>
</ul>

天府
<ul class = "ds-中州派分析段落">
<li>天府在迁移宫，一般安定。即迁移后可稳步发展。天府化科，尤主受异乡人尊敬。</li>
<li>府库空露，见阴煞、天姚，主出门受阴谋陷害以致破财、损伤。</li>
<li>若府廩充盈，又见吉星祥曜会合，则出门可以创业兴家。</li>
<li>天府与文曲化忌同度，出门小破财。主被扒窃或遇盗。更会武曲化忌，则破财程度增大，且主有损伤。</li>
<li>天府与擎羊、天刑同度，出门招忌，往往惹是非。</li>
<li>天府与擎羊、天月同度，主异乡染病。</li>
<li>天府与陀罗同度，出门往往迁延，致回乡受阻。天月同度则带病回乡。唯若煞忌刑重者，则主客死他乡。</li>
<li>天府在迁移宫，喜与紫微同度，主出门近贵。亦喜与武曲同度，主经商可以致富。唯必须见吉曜吉化始确。</li>
<li>逢府看相，故天相之吉凶，影响天府甚大。若府廩充盈，又会财荫夹、权禄夹、双禄夹的天相，则离乡决主发财。倘府库空露，会刑忌夹、火铃夹、羊陀夹的天相，则离乡必致倾败、挫折，或招灾病。</li>
<li>天府、昌曲同度；文曲化忌，或文昌化忌，又见禄存受两重羊陀交叠，须防在外经商为人倒账。</li>
<li>天府在巳亥宫，命宫必为「紫微七杀」，若紫微化权，最宜离乡背井创立事业，主远涉重洋兴创。若巳亥两宫更多辅佐吉曜，必为外邦侨领。</li>
<li>天府不喜会廉贞化忌，无论何种破败挫折，皆主感情创伤。若会武曲化忌，则为破财，或财转困难。</li>
<li>天府喜廉贞化禄，主谈笑用兵，求财顺利。唯若躔桃花、昌曲，则为在外多诗酒应酬，风月之乐。</li>
<li>天府与武曲化禄同会，经商大利，会辅佐吉曜者，必主远涉重洋致富。</li>
</ul>

太阴
<ul class = "ds-中州派分析段落">
<li>太阴居迁移宫时，主易为异邦人接受。若太阴入庙，更主人缘广结。唯若落陷，则亦招阴损。</li>
<li>太阴与昌曲同度，再躔桃花诸曜，在外主受异性垂青。见煞忌则主因色致祸。</li>
<li>太阴躔昌曲，化忌，因色破财。昌曲化忌则为被盗。</li>
<li>太阴与辅弼同度，不外得人助。与魁钺同度，则机会迭来。</li>
<li>太阴入庙躔火铃，仅主奔波；若落陷躔火铃，则主是非横逆。羊陀同度，奔忙无功，或在外有人阴谋陷害；空劫同度，无端破耗，太阴化为忌星时尤确。</li>
<li>太阴主静、主藏，故守迁移宫时，不宜与人竞争、争夺。</li>
<li>太阴在迁移宫，身宫亦在迁移宫，落陷，日生人，或下弦月生人，见火铃在命宫，父母宫又不吉者，主随娘改嫁。</li>
<li>太阴化忌守迁移宫，不宜在外乡异域投资。更不宜见猎心喜，主反招损失破败。</li>
<li>太阴化忌，受流年大运化忌冲起，双陀罗或双擎羊同躔，或两重羊陀夹，更见虚耗诸曜，决主在外横生是非，祸患起于突然。</li>
<li>太阴会天同化禄，白手成事业；会天机化禄，事业更张后即见鼎盛。</li>
<li>太阴化权守迁移宫，宜任职企业，职掌财经调度；化科者，主钱财上受人信任。</li>
<li>太阴与科文诸曜同度，不见桃花，古代认为主在外乡占藉发甲。现代则为留学有成之兆。</li>
</ul>

贪狼
<ul class = "ds-中州派分析段落">
<li>贪狼守迁移宫，在外多应酬。应酬性质，兼视命宫、福德宫而定。此三宫见科文诸曜，则多文酒之会；见桃花诸曜，则多酒色财气；见华盖、空曜，，则多佛道宗教的朋友。</li>
<li>贪狼化禄、化权，则应酬快乐，人缘广结。在子午二宫者，且多友人尊重。但若贪狼化忌，则每因无谓应酬而招惹麻烦、争夺。纵有吉曜同时会合，亦主多应酬而无收获。</li>
<li>贪狼与火星、铃星同度，在外热闹，唯仅多场面上的应酬，知交难得。有禄来会，则主因应酬而得财。</li>
<li>贪狼与火铃同度，若见煞或化忌，（煞忌齐会更劣），则主在外仪表面风光，一有实际利益即为人夺去。</li>
<li>贪狼与羊陀同度，主出门每多是非之挠，且往往由应酬、游宴而致。更见空劫，主遇窃盗；若加会桃花诸曜，则主因色致祸。见化忌，尤其是廉贞化忌者，情形更为严重。</li>
<li>贪狼独坐迁移宫，化禄、化权，或与廉贞化禄同会，即使不与火铃同度；亦主迁移在外精彩发达。见火铃尤主暴发。但若同时又见煞忌，则暴发之后必暴败。</li>
<li>流年贪狼守迁移宫，原局有煞，被流煞冲起，且见太岁、岁破、官符、大耗等曜，又逢忌星会合，主遭兵贼之灾，或受贪官污吏诬枉。</li>

巨门
<ul class = "ds-中州派分析段落">
<li>巨门守迁移宫，须视太阳，会太阳光辉者利迁移，主在外地多人缘，若太阳失辉，则主在外招是非口舌尤怨。唯太阳在巳午二宫，光辉太盛，亦有所不宜，出外不宜锋芒太露。</li>
<li>巨门在迁移宫，基本性质与在命宫相同，亦主「口舌以求财」。以化禄或禄存同度者，见辅佐吉曜者为佳。若吉煞混杂，则不宜从政，仅利从商。</li>
<li>巨门化权者，在外能以言词舌辩取信于人，凡太阳天梁昌曲吉会，则利讲学。</li>
<li>巨门化禄，则太是必同时化权，倘太阳庙旺，则主迁移后受外国人或异乡人提拔、赏识。唯若煞曜同时会照或同度者，则亦有是非困扰。或因事务需要，劳心费神。</li>
<li>巨门化忌，主进退不安，劳碌奔波。更见煞刑虚耗诸曜，则是非困扰，口舌纠纷相继而来。倘巨门化忌见空劫，则主在外处事，常多内心犹豫，进退失据。</li>
<li>巨门会巳午二宫太阳，倘太阳化禄、化权，则在外能以声势凌人。唯稍见煞曜，则当流年大运不吉时即易招非。若巳午二宫太阳化忌，更主是非口舌频仍，以不强出头及不露锋芒为宜，否则易招挫败。不见煞，则从事「口舌求财」的行业，可免是非。</li>
<li>流年巨门守迁移，见禄始可赴外乡异域，见煞则主枉费精神，若煞忌刑耗诸曜凑会，则主有无妄之灾。</li>
<li>凡巨门与禄存同度者，若更无吉化吉曜，则出门主受小人掣肘。流禄同度者亦是。</li>
</ul>

天相
<ul class = "ds-中州派分析段落">
<li>天相守迁移宫，一般情形下吉利，主近贵得利。见魁钺夹或同度相会者，更主受人提拔，唯仅利与人合作，或退居副贰，以不出锋头为宜，否则即受排挤。</li>
<li>天相见吉化，及禄存、魁钺、辅弼，皆主在外能创兴事业。亦宜从政，或带政治色彩的营谋。在现代，大企业机构的行政管理亦可视为带政治色彩。</li>
<li>天相在迁移宫，若为刑忌所夹，绝不利迁移，倘更见羊陀夹、火铃夹或空劫夹，或煞曜刑耗诸星同会，轻则主在外受人排挤，重则迁移异地惹牢狱之灾。</li>
<li>天相若为财荫夹，或为「六吉夹」，即魁钺夹、辅弼夹、昌曲夹、禄权科夹、紫府夹、日月夹，皆主在外得利。唯具体情形则不一。财荫夹主有现成机会；魁钺夹主有人提携；辅弼夹主有人扶佐；昌曲夹主事情顺利，禄权科夹主得财禄、权力、声望；紫府夹主受人赏识；日月夹主变动得利。以上仅一般原则，实际情况须视具体星曜分布而定。</li>
<li>大致而言，天相遇吉夹则多人缘，遇凶夹则主孤立。唯昌曲夹而天相又躔桃花，则主在外多交酒色徵逐之友；羊陀夹而禄存叠化禄，仅主小人妒忌，或小人包围。</li>
<li>流年天相守迁移宫，火铃照射，流羊、白虎、官符入度，主在外惹刑事官非。</li>
</ul>

天梁
<ul class = "ds-中州派分析段落">

<li>天梁与空劫同度，在外每经灾厄。见吉，则主因祸得福。</li>
<li>流年天梁遇吉，主在外经商可发，若会太阳化忌、天刑，流年官符、白虎入度，主招词讼。</li>

<li>七杀与空劫同度守迁移宫，主进退失据，理想常超过现实，以致不守一业，不守一地，时生改变之心。趋吉避凶道，唯宜从事工艺、工业，则当有新意念可助兴创。</li>
<li>七杀与禄存同度守迁移宫，若火铃、空劫同度，则主在外受压力，或主飘荡无依。</li>
<li>命宫天府会吉化（尤其是化禄），则一般不利迁移，除非七杀亦会权禄之星，然后始可移根换叶，若天府为府库空露，则以迁移为是。</li>
<li>流年七杀见吉化吉曜守迁移宫，一般主事业更新，或转换新职。廉贞化忌、武曲化忌来会，见煞，主出门有意外。</li>

寅、申：七杀
<ul class = "ds-中州派分析段落">
<li>紫微与七杀同拱，主出外为人敬畏，亦多人缘。若「紫杀」同度而百官朝拱者，更主振臂一呼，则群相响应，故在外亦能建立事功。</li>
</ul>


卯、酉：七杀、武曲
<ul class = "ds-中州派分析段落">
<li>武曲与七杀同拱，化禄，则迁移异地后，活动能力甚强，因此获利。武曲化忌，主灾病词讼，亦主资金周转不足。</li>
<li>卯酉二宫「武曲七杀」，结构最为不良（其中酉宫又优于卯宫），主出外受压力，若见煞忌刑耗，更主遭人陷害。</li>
</ul>


辰、戌：七杀
<ul class = "ds-中州派分析段落">
<li>廉贞与七杀同拱，主迁移异地能建立声誉。廉贞化禄者，顺利得财，且多助力；若廉贞化忌，则亦主挫败及意外。更天月同度，主在异乡染病；更天刑同度，则主祸生不测。</li>
</ul>


巳、亥：七杀、紫微

</ul>

破军
<ul class = "ds-中州派分析段落">
<li>破军在迁移宫，一般情形下不利迁移出外。除非命宫天相为刑忌夹、火铃夹、羊陀夹，然后始主不得不离乡发展。其中尤以羊陀夹为然。盖天相若为羊陀所夹，必有禄存同度，此禄存与破军对拱，大利迁移。</li>
<li>凡破军在迁移宫，必须靠专门技能起家。古人谓此为「巧艺走途中」。其中一个特例是：「加文昌、文曲武曲相会，优伶之人」。优伶自然可算具有专门技能。</li>
<li>若破军、文曲同度，化忌，则或为落拓书生流落江湖，或为江湖浪子，擅诸般巧术，包括戏法、卖药等在内。</li>
<li>破军见魁钺最吉，主自己的技能有发挥的机会，且受赏识。</li>
<li>破军在迁移宫亦得禄为佳，则可藉专门技能在异乡立足。</li>
<li>破军化权，宜在异乡凭技艺为人服务。见煞者，主工艺或手艺。</li>
<li>破军与四煞同度守迁移宫，火铃主流离，羊陀主破败；煞重则主人缘不足，灾祸频仍。</li>
<li>破军守迁移，亦喜龙池凤阁、天才、昌曲同会，可增加其专门技艺的优秀。当空劫同躔相会之时，亦可视为技艺的巧思。</li>
<li>破军见煞轻不忌，最不喜煞忌齐会。尤不喜子午宫见廉贞化忌，则主伤心落泪，或自身灾病。</li>
<li>紫微化权，必同时见武曲化忌，则主在外不宜经商。亦主权威虽重，而挫折频仍。</li>
<li>凡破军守迁移宫，若不见化禄，则不利迁居异地生根，但却利流动各地以生财。</li>
</ul>

子、午：破军
<ul class = "ds-中州派分析段落">
<li>破军在子午二宫，见吉化、吉曜，则亦可经商，但所经营者仍须带有专门的性质。如在异邦开家乡菜馆，即是一例。</li>
</ul>


丑、未：破军、紫微
<ul class = "ds-中州派分析段落">
<li>丑未宫「紫微破军」守迁移宫，出外得人敬畏，尤以百官朝拱的情况下为然。若只见朝拱而少主要辅佐，但又见煞，则在异邦当为一行一业的领袖人才。</li>
</ul>


寅、申：破军
<ul class = "ds-中州派分析段落">
<li>破军在寅、申、酉三宫落陷，不利迁移，若离乡，则主飘流无成。以羊陀交会者为然。</li>
<li>寅申宫破军，与武曲化忌冲拱，又与火铃、空劫、天刑、天虚、大耗、阴煞等曜同会，主迁居异地至少必有一次重大灾祸，或甚至死于异乡。</li>
</ul>


卯、酉：破军、廉贞
<ul class = "ds-中州派分析段落">
<li>破军在寅、申、酉三宫落陷，不利迁移，若离乡，则主飘流无成。以羊陀交会者为然。</li>
<li>「廉贞破军」不宜经商，主客缘不广。若廉贞化忌者，煞重，亦主客中发生意外。</li>
</ul>


辰、戌：破军

巳、亥：破军、武曲

<ul class = "ds-中州派分析段落">
<li>「武曲破军」见煞尚无妨，若武曲化忌则主手头困乏；会廉贞化忌，亦主灾病。</li>
</ul>
