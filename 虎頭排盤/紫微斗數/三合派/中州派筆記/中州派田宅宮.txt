田宅宫在推算斗数时，有下述数点作用：家宅的兴衰，产业的兴败；老亲丧服；所服务机构的盛衰，以及有无变动更张；住宅或公司的迁动。
如上所述，足见田宅宫的代表意义非常复杂。在从前，不喜分炊分家，则关于产业、家宅的命运际遇，往往只能在家长的星盘中看出细节，此点不可不知，因为时代虽不同，但其意义则仍存在。
封建时代，富贵穷通往往牵连六亲，所谓「六亲同运」。所以有时关于家宅田产的命运，须参看其他宫度。例如有子成龙，自己受到封荫，得赐宅第，则只能在子女宫看到吉兆，不在田宅宫；又如受兄弟拖累，以致抄家，亦只能在兄弟宫看到凶兆，未必在田宅宫。
现代人卖买田宅，有时带有投机性质，关于这种性质的吉凶休咎。一般情形下不能在田宅宫显示出来。
由田宅宫看丧服，以「长子嫡孙」最为准确。
任职的公司机构兴衰，固可由田宅宫看，自己经营而自己亦参加工作的公司，亦作服务机构论。
田宅宫见蜚廉，主生白蚁，或多虫类；见阴煞、天姚，主有阴人作崇。

<ul class = "ds-中州派分析段落">

武曲
<ul class = "ds-中州派分析段落">
<li>武曲为财星，独居庙地，古人谓主得祖、父大业。然此不专主产业，亦主事业上的承继。故田宅宫一般喜武曲坐守。</li>
<li>武曲化禄或与禄存同拱，不见煞曜，祖业丰厚易守成，见辅佐吉曜更主陆续添置。但「田宅宫」吉曜集中，难免「命宫」、「事业宫」、「财帛宫」星曜相对消弱，故须注意是否能将祖业发扬，或仅能保守，或甚至因不思进取而致破败。古今社会背景不同，应特别注意。</li>
<li>武曲守田宅宫，忌与破军同拱。故于寅申巳亥四宫为弱地。若武曲化忌，更见空劫，大耗，天刑，必主倾家。即武曲不化忌，亦主祖业动摇。</li>
<li>武曲守田宅宫，吉者，常主能在业务兴旺的机构服务。凶者，则主时生变动。古人雇庸，最重视其人之田宅宫，凡武曲坐田宅而吉，认为旺主；武曲坐田宅而凶，则视为不利主人。此意可以参考。又，武曲守宅，宜服务财经机构。</li>
<li>武曲守田宅，火星同度，又见虚耗刑忌，更有流煞冲会，是年主火灾，「武贪」见火星虽不主火灾，但武曲化忌或贪狼化忌之时仍主火灾。</li>

天相

<ul class = "ds-中州派分析段落">
<li>天相为财荫夹者，主有祖荫；为刑忌夹者则无，或难有但却破败。</li>
<li>天相必须见禄，然后始主可以自置。更见辅佐诸曜者，则产业丰盈。</li>
<li>天相为火铃夹者，主家宅不安，住宅喧闹，或有缺陷。若更空劫耗同度，则主火灾。</li>
<li>天相喜昌曲夹、龙凤夹、辅弼夹，则住宅华丽。若有煞同度，则仅主外表美观。</li>
<li>天相为刑忌、羊陀夹，受四邻骚扰或威胁。</li>
<li>天相与煞忌刑耗同会，主因产业惹是非纠纷。煞重则主破荡倾败。</li>
<li>天相为禄权科夹，魁钺夹，皆主住宅近贵。亦主田宅发科名。</li>
<li>天相守田宅必须视天府，库存充盈者吉，府库空露者凶。</li>
<li>天相喜会武曲化禄，主产业丰厚。若武曲化忌则主倾败。廉贞化禄者，必因喜事而置业，或家宅有喜庆。倘化忌，主伤心流泪，或有不虞之灾。</li>
<li>天相为凶曜夹，或与凶煞同会，亦主受主管机构掣肘；或服务机构不宁，有压力。</li>
<li>天相对风水相当敏感，见煞，武曲破军对拱者尤甚。亦主终身住旧宅。</li>
</ul>

<li>天梁见四煞，家宅不安。在机构内亦多是非，必须见辅佐诸曜，化科、化权，始主安宁，及受人尊重。</li>
<li>天梁一般不受风水影响。唯若阴煞同度，又见火铃，则受影响。</li>

七杀

<ul class = "ds-中州派分析段落">
<li>七杀最喜得禄，虽无祖产父业，却能自置。倘七杀同躔禄存，又叠化禄，唯同时见煞曜空劫者，则祖业先破，至大运佳美时却能恢复旧观。</li>
<li>「武曲七杀」的宫度，因七杀落陷，一般不利田宅，稍见煞，或对宫天府不吉。即易产业破败。必须武曲化禄，然后始主自置产业丰盈。</li>
<li>七杀守田宅最不宜忌星同度，武曲化忌主倾败；亦主被逼离开服务机构。若廉贞化忌者，则防家宅有伤心流泪之事。唯若见空劫刑耗，则亦主各种事端而致折卖产业。</li>
<li>七杀与火羊同度者，主中晚年始能置产；若与陀罗铃星度，则家宅不安，在服务机构权内亦多阴小暗算。</li>
<li>七杀与空劫刑耗同度，主家业破败。</li>
<li>七杀与武曲化忌同度，主降黩；见煞忌，主受掣肘或处分；与武曲化禄同度，见天巫，主升迁；见辅佐吉曜则主有权。寅申宫七杀，更主受人提拔而致任主管之职。</li>
<li>七杀与陀罗火星同度者，流煞冲起，主火烛虚惊。</li>
<li>七杀对风水不敏感。唯「廉杀」、「武杀」亦宜注意风水。</li>

破军
<ul class = "ds-中州派分析段落">
<li>若破军落陷，则主住宅破旧，亦主在无创建意味之机构服务。</li>
<li>破军化禄权者，更见吉曜，主旧宅拆建翻新。亦主转换良好之工作环境。</li>
<li>破军与羊陀同会，见大耗，主田宅有水灾。倘见羊陀，更见文曲化忌会照或同度，则主受拖累或欺骗而致变卖产业。</li>
<li>破军与火铃，空劫同度会照，主产业损失。更会武曲化忌及阴煞、大耗、天刑者，则墙倾屋圮。</li>
<li>「紫微破军」同度，见禄存或化禄，而无煞忌刑耗诸曜，则主有意外资产可得。尤喜见天魁天钺同度。稍见煞者，暴得暴失。</li>
<li>唯「紫微天相」，则不主得意外资业。仅于财荫夹，又见破军化禄的情形下，始主置业。亦主转换工作护环境。</li>
<li>破军与廉贞同度及会照，喜廉贞化禄，则产业增加，若化忌，则主耗散。</li>
<li>流年田宅宫见破军入庙，又有吉曜，则主迁新宅。若落陷见煞，则家宅破败。会武曲化忌者尤主降黩，或服务机构停业。</li>
<li>破军在田宅，不受风水影响。</li>
<li>子、午：破军</li>
</ul>
