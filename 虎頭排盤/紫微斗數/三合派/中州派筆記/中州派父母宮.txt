推断父母存亡，先看太阳太阴；太阳落陷主父先亡；太阴落陷主母先亡。若二星俱落陷或俱入庙，则视太阳太阴有无化忌。太阳落陷化忌不利父；太阴落陷化忌不利母。太阳入庙化忌不利母；太阴入庙化忌不利父。
日月反背者，日生人不利母，夜生人不利父。
上述推断，仍须参考父母宫星曜组合而定。若星曜祥和，则不主刑克，可不必理会上述太阳太阴之克应。唯仍可藉此推断父母之性格。
若擎羊或陀罗独守父母宫，会照太阴太阳者，主刑克，仍须参考上述克应而定。
火星或铃星独父母宫，又见太阳太阴落陷或化忌者，主二姓延生。或主幼年与父或母不相聚。具体性质详三方星曜而定。
父母宫不喜见辅佐单星，尤不喜见对星复见单星（如见天魁天钺，又见文昌而不见文曲）。主有两重父母，或父母再婚，或父有外室。
父母宫除推断父母存亡际遇外，尚可推断自己跟上司、主管及主管机构的关系。则迳以宫垣中星曜推断即可，不必理会上述原则。因上述原则仅适用于本身父母。
父母宫见禄权科，不可以为必属父母显贵，但却必主不克父母。
父母宫见禄、权、科，许多时候仅主上司的特质。一般情形下，可视为宜在大机构企业服务。或主上司任要职，有权势名誉。
父母宫见四煞，亦不一定主刑克，有时仅主人际关系不佳（包括与本身父母及上司主管的关系）。刑克与否，主要视星曜组合而定。
父母宫见禄存、天马，往往主父母迁动。但有时亦主自己时时更换上司。
父母宫吉曜吉化重重，而命宫星弱，不为美论，必须见荫庇之星，然后始主对自己有力，否则仅主自身为人附庸。
父母宫喜见天梁、天寿、天福、解神，则主父母长寿。
<li>父母宫星曜性质，主父亲或主母亲之克应，仍须视太阴太阳及日夜生人而定。如太阴在巳宫化忌，父母见紫微在子，火星同度，则主父亲离家，自己依随母氏。盖父亲时时离家，母亲自然化忌矣。</li>
流年父母宫不喜见流昌流曲。如见，星曜又不吉者，则须检视流年命宫及小限，如见丧门、吊客、白虎等曜，又有煞忌，即须防孝服。长子嫡孙尤确。
有时流昌流曲不出现于流年父母宫，而出现于流年命宫。盖昌曲为礼乐之星，性质不佳时即主丧服之礼。
流年父母宫见煞忌，亦主父母灾病。唯跟上司的不善关系，往往亦由是克应。
流年父母宫见天巫亦不主吉，盖天巫为遗产之曜，有禄冲会，同时见煞忌者，亦须防父母孝服。

紫微

<ul class = "ds-中州派分析段落">
<li>紫微守父母宫，主父母有权威。然如无百官朝拱，则仅主父母主观，在家庭有暴君倾向，其权威性质在此。故若见煞忌刑曜，则主自己童年不为父母爱护，或父母/上司常主观命令，左右自己的意志。</li>
<li>见辅佐单星，或桃花诸曜，主有继母或父亲有外宠。</li>
<li>若紫微化科，则父母主观。在野孤君者，主观程度更甚。</li>
<li>若贪狼吉化，见科文诸曜，亦主感情融洽；见桃花诸曜，父有偏房外室。</li>
<li>唯若廉贞化忌，煞重主刑，煞轻则主不和；贪狼化忌，亦主父有外室；武曲化忌，或为父母事业倾败，或为刑克分离，或为灾病。与上司的关系，可依上述性质同断。</li>
</ul>

<li>「紫微贪狼」与火、铃同度，亦主少小即离开父亲或母亲。见吉，其分离性质温和，如出外寄宿，或父亲出外谋生之类。</li>

<ul class = "ds-中州派分析段落">

<li>若破军化禄，则离别克应之期在第二个大限。火星同度则提前。</li>

巳、亥：紫微、七杀
<ul class = "ds-中州派分析段落">
<li>巳亥宫「紫微七杀」，对四化影响极为敏感。基本性质则为父母有责任心，无论任何种情形下，对自己均能呵护。</li>
<li>故最喜见廉贞化禄、破军化权、武曲化科同时相会，则两代感情融洽，且主受父母提携。</li>
</ul>

天机

<ul class = "ds-中州派分析段落">
<li>天机守父母宫，基本性质不利父子间的人际关系。见禄权科会，然后始主稳定，否则主两代远离。入庙无煞，则远离的性质温和。</li>
<li>天机与桃花、天马同会，主少小过继、出祀，或年长入赘，或供养岳家而不供养父母，或亲近岳家。视煞忌轻重而定。</li>

<li>天机与火铃同度，主离不主克。</li>
<li>天机化忌，基本性质为两代人际关系不佳。</li>
<li>「不利父母」，乃指出生后父母境遇欠佳。持此以验上司，亦主上司在工作上境况不利。此项徵验可以推断流年。</li>
<li>凡天机在父母宫，稍见刑忌即有事端，应详各流年父母宫而厘定其克应。克应之期，多在太阴太阳的宫度。</li>
</ul>


子、午：天机
<ul class = "ds-中州派分析段落">

<li>天机巨门的星系，又主父母与当局者的配偶不和，见煞尤确。</li>

丑、未：天机

<li>「天机太阴」而天机化禄者，太阴必同时化忌，夜生人父母宫在寅宫者，早年不利父亲。</li>

太阳
<ul class = "ds-中州派分析段落">
<li>太阳在父母宫，入庙者，无克；更见吉，童年受父母爱护。日生人更吉。</li>
<li>若太阳落陷，视煞忌会合程度，而定是否刑克，或不为父母所喜。夜生人更凶。</li>
<li>入庙见煞，而太阴又与煞星同度，主父母不和。</li>
<li>落陷见煞，而太阴又有煞星同度，主先克母，或父母因事分离。倘太阴无煞同躔，则童年父运不利。</li>
<li>日生人，父母宫太阳入庙，又会吉星、吉化，则不但童年受父母溺爱，且主父亲境遇良好。夜生人减等。</li>
<li>推断与上司之人际关系，以「太阳太阴」最难推断。若见吉，则关系良好，唯见吉曜吉化太多，却又主自身常因对上司怀有感情，以致不思独立进取。见吉煞交集，则主与自己关系良好的上司易转换；见忌，则防初时关系良好，但却易因误会而转为恶劣；见忌又同时见辅佐吉曜，则此种性质尤确。</li>
<li>流年太阳于父母宫化忌，不可即作刑克论，煞刑交并，始主灾病，若煞轻，有时只为与父母不和，产生歧见。又须详流曜，见天煞、丧门、白虎，然后始应于父母，否则可能主与上司不和。</li>
<li>流年太阳化禄，又见辅佐诸曜重重，可能其克应在于上司主管。</li>
</ul>

子、午：太阳

丑、未：太阳、太阴
<ul class = "ds-中州派分析段落">
<li>「太阳太阴」同度者，一般情形下主父母和美，与自身关系亦良好。若太阴化禄，则主幼年受父母溺爱；太阳化禄，则受父母助力，或得父母之戚友提携；在未宫者较不利母，唯在丑宫较不利父。</li>
<li>「太阳太阴」同度于父母宫，若太阴化科见煞者，古代以为主婆媳不和；太阳化科，见煞者，古代以为福荫不及三代（即自身之子女易破败）。此种情形，今日只能参考。</li>
</ul>


寅、申：太阳、巨门
<ul class = "ds-中州派分析段落">
<li>太阳巨门在父母宫，主父亲曾离乡安家。若见化忌、四煞，则父有灾病；见吉化吉曜，则父有事业。</li>
<li>「太阳巨门」见吉化者，以服务异族为佳，主得异族上司提拔。</li>
<li>「太阳巨门」化忌，则主与上司不和，因此以自由职业为佳。</li>
</ul>


卯、酉：太阳、天梁
<ul class = "ds-中州派分析段落">
<li>「太阳天梁」守父母宫，若太阳吉化，则得父母荫庇之力甚大。唯见辅佐吉曜过多，则反主自身欠缺父母以外之助力。</li>
<li>「太阳天梁」在父母宫，如无吉曜，则虽不见煞忌，亦主父母主观。同时亦主与上司易有成见。</li>
<li>「太阳天梁」以太阳化权为最吉，不但可受父母荫庇，且多助力。</li>
</ul>


辰、戌：太阳

巳、亥：太阳

武曲
<ul class = "ds-中州派分析段落">
<li>武曲守父母宫，若不入庙，又无辅佐吉曜扶持，则主刑克父母。亦主与上司关系不良。</li>
<li>若武曲入庙，有辅佐吉曜及吉化，则不主刑克。与上司、老板的关系则仍易疏离。</li>
<li>武曲守父母宫，见化忌，同时见煞，则须参考命宫及大运、流年命宫、父母宫，而推断其是否刑克，或仅主父母事业破败。</li>
<li>凡武曲守父母宫者，古人皆主张出祀或离乡，认为因此可免刑克。关于此点，今日似不必认为可以克应。但父母对自身缺乏助力，或易形成代沟，则宜加以注意。</li>
<li>武曲化禄于父母宫，唯于第一个大运却化为忌星者，往往主父母产业凋零。若武曲化忌，于第一大运化为禄星者，则主父母灾病。</li>
<li>武曲化忌，最忌羊陀夹、火铃夹，主父母灾病，或有极大破败。</li>
<li>武曲在父母宫，更与羊陀、火铃、空劫、刑耗诸曜同会者，少年时父母必不利。流年大运父母宫见此种星系，亦防父母突生灾病或事业破败。甚至父母宫有武曲化忌对冲，亦有此种情况。</li>
<li>唯武曲化忌，天厨、火星同度者，往往又主父母所业者带煞气，如饮食业之类。不见天厨，亦为操利器行业。如外科医生、理发之类。其职业贵贱，视有无辅佐吉曜同会而定。</li>
</ul>


子、午：武曲、天府
<ul class = "ds-中州派分析段落">
<li>「武曲天府」的组合，最为有利。无煞忌刑耗，则不主刑克。若煞轻，有天寿同宫，亦不主刑克。</li>
<li>「武曲天府」会廉贞化忌，则主父子参商。会刑忌夹的「紫微天相」，亦然；或主受上司的压力。</li>
</ul>


丑、未：武曲、贪狼
<ul class = "ds-中州派分析段落">
<li>「武曲贪狼」与火铃同度，主克；亦主与上司关系有突然变化，权利纠纷，凶终隙末。若武曲化忌则亦主父母横破。</li>
<li>「武曲贪狼」而贪狼化禄，则主上司提携，唯有煞，则仍主始终关系变劣。</li>
</ul>


寅、申：武曲、天相
<ul class = "ds-中州派分析段落">
<li>「武曲天相」见财荫夹，主得父荫，亦主得上司提携助力；刑忌夹者，则主刑克，亦主受上司忌才。</li>
<li>「武曲天相」亦不喜火铃夹、羊陀夹，主刑克；亦主受上司排挤，或多争吵、误会。</li>
</ul>

卯、酉：武曲、七杀
<ul class = "ds-中州派分析段落">
<li>「武曲七杀」基本亦主刑克，见煞忌者尤甚。若无煞，反见吉化、吉曜，则仅主关系疏离。亦不喜见火星同度，再会桃花，则主自身为外室或庶室生子。</li>
</ul>

辰、戌：武曲

巳、亥：武曲、破军
<ul class = "ds-中州派分析段落">
<li>「武曲破军」刑克亦重，与「武杀」大致相若。凡「武破」、「武杀」在父母宫者，亦主时时转换工作。</li>
<li>「武曲破军」而武曲化忌，则极可能为父母产业暴破暴败。</li>
</ul>

天同
<ul class = "ds-中州派分析段落">
<li>天同为带感情的星曜。入庙，则主两代和洽，亦不见刑克。唯落陷时，则仍多意见不和。</li>
<li>天同化忌，若煞重，仍主伤心流泪。须详大运及流年父母宫而定其不利性质。</li>
<li>天同与擎羊同度，则主受上司压力，以致不和。陀罗同度，见刑忌者，主受上司牵累；火铃同度，时时变换上司。</li>
<li>天同化禄，唯与天梁同度或对拱者，始主受父母荫庇，否则反主父母于自己童年时出外谋生。</li>
<li>天同火星，亦主幼年离开父母。</li>
<li>天同与煞忌刑曜同会，又有吉曜或吉化者，尤其是天同化禄，往往主父母产业虽丰，但得不到手。</li>
<li>天同躔昌曲、桃花，主父有外遇，或主自身为偏房继室所生。</li>
<li>天同与左辅右弼，或天魁天钺同会，以见齐对星始吉，见单星者，主父母两次婚姻，不必见煞已然。</li>
<li>天同守父母宫，流年不宜廉化忌，若父母宫见昌曲及煞曜（或命宫见昌曲、父母宫见煞），当主孝服。</li>
</ul>

子、午：天同、太阴
<ul class = "ds-中州派分析段落">
<li>「天同太阴」，基本星系性质主感情丰富，唯若与刑忌煞耗诸曜同会者，则亦主刑克分离。最不喜太阴化忌，亦父母不全。</li>
<li>「天同太阴」吉化，则无刑克。吉化而煞重者，仍主不和。亦主与上司参商。</li>
</ul>

<li>「天同天梁」与天马同度，主父母须离乡背井。更见火铃，则主飘流。此种星曜性质，有时又主父亲所业者带流动色彩。</li>
<li>「天同天梁」而天梁化科者，一般情形下主利父不利母。更见煞，主刑克。</li>

廉贞

<ul class = "ds-中州派分析段落">
<li>廉贞在父母宫，由于其为带感情及血缘的星曜，所以性质相当复杂。尤其是当廉贞化禄或化忌之时，性质异常敏感，所以须小心推断。</li>
<li>廉贞化禄，一般主不刑克，或感情融洽。唯此时太阳必同时化忌，基本性质不利父母，故此时必须详视太阳、太阴所同会的星曜（不论在何宫垣），及父母宫廉贞所同会的星曜，然后始能作出推断。</li>
<li>廉贞化忌，一般主不利父母，或刑克、或生离、或感情不融洽。须见太阳、太阴同会星曜而定。</li>
<li>若廉贞化忌，与火铃、羊陀、空劫、天刑、天伤等星曜同会者，一般主刑克。</li>
<li>若廉贞化忌，与火星同度，或与天马、天虚同度，主远离父母。</li>
<li>若廉贞化忌，与铃星同度，或与杀、破、狼同度而无吉，且见虚、耗、刑、伤者，则宜出祀、过继，两重父母。</li>
<li>廉贞与昌曲、桃花诸曜同宫或会照，则主为偏房继室所生，或上代阴盛阳衰，或父有外遇。见辅佐对星带单星者，则决主父有外遇；见昌曲化忌，决主偏房继室所生。</li>
<li>廉贞化忌，桃花诸曜毕集，主父母婚姻情况复杂。</li>
<li>凡廉贞带桃花者，宜从事以异性为顾客对象的行业，或带精神享受色彩的行业。</li>
<li>廉贞入流年父母宫，与杀、破、狼同会，更见煞忌刑耗交侵，则主妨父母。尤不喜流年白虎与流羊相会于流年父母宫。</li>
<li>若廉贞化忌会武曲化忌于父母宫，主父母是年有病症或死亡。若武曲躔父母宫而会廉贞化忌，则有可能为父母破败。断上司关系，则可能为遭上司解雇。</li>
<li>廉贞与流昌流曲相会，昌曲化忌，则亦为不利父母的星系结构。有时此克应亦在流年命宫或小限出现。</li>
</ul>

<li>若刑忌夹印而廉贞本身化忌，则于该流年或大运，主父母刑伤。</li>

天府

<li>天府守父母宫，一般情形主父母双全。仅于府库空露的情形下，始有刑克，或主不克享父荫。</li>
<li>天府与擎羊同度，主两代不和。与陀罗同会，主有刑克。见天刑者更重。</li>
<li>天府与火、铃同度，主生离。见天刑者，少年时父母不全。</li>
<li>天府与空劫同度，主重拜父母。</li>
<li>天府禄存同度，见煞，主父子参商。又主父母喜独掌财权。</li>
<li>天府与化禄同度或对拱，以武曲化禄最宜，则父母财荫颇厚。见天巫同度，或天巫在命宫、财帛宫者，主父母遗产丰厚。</li>
<li>唯流年父母宫天府，忽见重重化禄或禄存冲会者，乃须防丧服之应。</li>
<li>天府化科，主无刑克。若太阳同时在庙旺宫垣化禄，则且主得父亲荫庇之力。</li>
<li>天府得「百官朝拱」，必须见对星，不喜对星带单星，主两重父母，或自身为偏房继室所生。</li>
<li>天府与桃花同躔，见昌曲单星，主父有外遇。</li>
</ul>

<li>太阴落陷则不利母。倘仅躔火、铃，则只主分离。若煞重，则防刑克。</li>
<li>太阴独坐更不宜落陷。轻则主幼年狐独，不为母亲疼爱。重则远离父母，或刑伤生母。</li>
<li>太阴擎羊同度，主与母亲缘份浅，落陷更甚。</li>
<li>太阴陀罗同度，主与母亲不和，各怀成见，亦落陷者更甚。</li>
<li>太阴与天马、天虚同度，不见火铃亦主远离父母。见火铃，少幼时父母及自身星散。倘更见单星，往往主过继或祀出。</li>
<li>太阴与天梁、天寿相会，则虽落陷化忌，亦仅主灾病，不主刑克。</li>
<li>太阴化权、化科于父母宫，盘中太阳无力（如不会化吉曜），则幼年母代父职，或母亲持家。若太阴在巳，往往为父亲离家或死亡之应。</li>
<li>流年父母宫太阴会煞刑白虎，母氏有伤。</li>
<li>太阴化忌见羊陀，更见文曲，主有人进谗，以致与父母或上司不和。</li>

天相

<li>倘见煞忌，又见辅佐单星，则须详太阴、太阳之庙旺而定其父母情况。可能为一亲早丧，一亲另行婚嫁。</li>

天梁
<ul class = "ds-中州派分析段落">
<li>天梁为寿星，又为荫星，在父母宫入庙，无煞，或煞不重者，主父母双全。</li>
<li>倘入庙无煞，而有吉曜吉化会合，则主必得父母福荫。见天巫及禄，主父母有遗产，或继承父母事业。</li>
<li>天梁落陷，主有刑克。见煞忌者尤确。古人认为，过继或出祀者可免。在现代，往往主远离即可避免。</li>
<li>天梁与天马、擎羊同度，主离家。或主入赘，或主供养岳家而与父母疏离。</li>
<li>父母宫天梁，与命宫一见火星、一见铃星者，则为重拜父母之兆，或入赘，或瞻养岳家而不瞻养父母。</li>
<li>流年太阳化忌照天梁，又有昌曲及流煞冲会者，若见白虎丧门入限，主孝服。</li>
</ul>

七杀
<ul class = "ds-中州派分析段落">
<li>七杀于父母宫并非善曜，即使入庙，亦与父母关系疏远。尤不喜与武曲或廉贞同度。</li>
<li>七杀守父母宫，煞轻者，主有代沟。亦主与上司不和。</li>
<li>七杀不喜会孤辰、寡宿，否则主孤。亦主与上司关系疏远。</li>
<li>七杀会紫微化权，父母成见极深。</li>
<li>七杀会紫微化科，又见吉曜，父母有声誉。但两代关系未必和谐。</li>
<li>七杀与火铃同度，或会天马，主远离父母，且远离之性质不良。</li>
<li>七杀不喜见武曲化忌、廉贞化忌，若再见凶煞恶曜，主父母刑伤，或事业倾败，或恶疾缠身，详所会杂曜而定。</li>
<li>七杀不与恶曜同度，但却会煞三合宫，且见武曲化禄者，主父母任武职，或牙医、外科医生之类（持利器以求财）。</li>
<li>七杀得禄、权、科会，主得父母荫庇之力。见天巫主得遗产，或继承父母事业。</li>
<li>流年七杀守父母宫，见流煞冲会，又见化忌者，须防孝服。</li>
</ul>

破军
<ul class = "ds-中州派分析段落">
<li>破军守父母宫，基本性质不良，稍见煞即主远离，煞重则主刑克。</li>
<li>破军与火星、铃星同度，决主远离。火铃二星一居父母宫，一居命宫，主与父母关系疏离。见孤辰、寡宿，尤主幼年即远离父母。</li>
<li>破军独坐于父母宫，基本性质孤独。亦主跟上司关系疏远，故有时亦主时时变换工作。唯见辅佐对星者，则可以调和关系。</li>
<li>破军与擎羊、陀罗、空劫、天刑、天伤、天虚等同度，主刑克。古人认为宜重拜父母，二姓延生。在现代，可能为离家后再难与父母相聚，唯却与岳家相处。</li>
<li>破军与忌星相会，多主刑克，若更见羊陀交并者尤确。以论上司关系，则主终身难得长上提携，且亦为上司夺功。</li>
<li>破军化禄，则父母和美，更见辅佐吉曜者尤佳。</li>
<li>破军化权，绝不宜羊陀同宫，否则主父子关系不和，或与上司不和。</li>
<li>破军与辅佐单星，见桃花诸曜，主两母。见化忌，则生母刑伤。</li>
<li>破军守流年父母宫，有辅弼单星同度，流煞飞入，主父母病重。亦主与上司结怨。</li>
<li>破军守流年父母宫，见武曲化忌或廉贞化忌，有流昌流曲，亦主孝服。若无流昌流曲者，则以见火铃羊陀为然。</li>
</ul>
