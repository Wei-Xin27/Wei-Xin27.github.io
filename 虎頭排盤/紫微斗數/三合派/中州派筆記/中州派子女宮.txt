子女宫 
子女宫除用来推断自己跟子女的关系、子女数目、子女性格之外，并用来推断自己跟后辈的人际关系。所谓后辈，系指门生弟子，以及自己所提拔扶助，或亲切关系的下属，如助手助理等而言。
举例来说，假如你是一家商店的老板，跟一般伙计的人际关系，仅视「交友宫」（古代称为「奴仆宫」）；但假如有一位伙计，成为你的左右手，而你又有心提拔其人，使可以代替你独立处理业务，则跟这位伙计的人际关系，便得视「子女宫」而定。
在现代，推断子女的数目已失去意义，但古代传下的推断法则，却仍可参考，因为藉此可推算有无后辈的助力，以及其数目的多寡。
古代推算子女宫的法则如下：由子女宫所躔星曜组合，厘订子女数目，与及其品格地位，然后参照本宫之辅佐煞化，以视其有无刑克，数目增减，品格地位之优劣变化等等。于推断时，当然应以本宫为生，然后参考「三方四正」。一般来说，若本宫星曜为善星，见辅佐诸曜不但主数目增加，且亦主增高子女的地位评价；若会煞忌刑耗空劫，则主子女有刑克。唯若本宫所躔为恶曜，则辅佐诸曜只视子女数目的增加，除非构成格局，否则不作为子女品德地位的改善；若煞忌刑耗空劫同会，不视为刑克，只作为子女品德地位的评价。
古人推断子女宫，三方四正会南斗星多，主先生男，或多子；会北斗星多者，主先生女，或多女，又以日生人见太阳落在阳宫，主多生男，夜生人见太阴落在阴宫，主多生女。此外，又以日生人不喜子女宫躔太阴；夜生人不喜太阳躔子女宫，主刑克分离，凡此皆可参考。
以古人推断之法，用来推算自己跟晚辈的人际关系，准确程度亦相当高。
「中州派」所传，凡见辅佐诸曜「单星」，往往主两重子女。在古代，则多指妻妾生子，或养子祀子，唯现代人事关系复杂，应包括女性两次婚姻的子女在内。
凡视孪生子，必看天相。天相见左右昌曲夹，或在对宫加会，又多见杂曜「对星」，如三台、八座之类，怕主有孪生子。唯若同时火铃夹、空劫夹、羊陀夹者则不是。
禄存在子女宫，并不表征子女富有（会化禄始是），若正曜不甚吉，主庶室生子。在现代，往往又为未婚生子。见火铃同度者尤确。
天相为羊陀夹、火铃夹、或空劫夹，再会煞忌刑耗者，主未婚生子。会昌曲对星，而其一化忌者尤确。
子女宫见火铃同度，或见天马禄存，主子女远离膝下。

紫微

<ul class = "ds-中州派分析段落">
<li>凡紫微在子女宫，均主迟得子，门生弟子以及追随自己的晚辈亦主迟得。亦以迟得为宜。</li>
<li>紫微在子女宫，须见辅弼，然后才可以成为自己的助力，若见昌曲、禄马、魁钺，仅对子女或晚辈有利。</li>
<li>紫微在子女宫，四煞空劫并照者，宜收养或祀继，然后始能得子。同时见桃花，则主偏房或外室生子。夫妻宫不佳，有生离死别之兆者，则主继室生子。</li>
</ul>


子、午：紫微
<ul class = "ds-中州派分析段落">
<li>紫微在子午宫，主子女性格倔强，爱恶随心。因为贪狼对拱，故多主头胎生女；在古代，定子女数目为三男二女。若为「在野」，则子女孤独，且性格与自己不合；若为「百官朝拱」，则子女地位崇高，且子女数目增加。</li>
<li>紫微在子午，与禄存同度，主子女自私。又躔火铃，则主未婚生子。</li>
<li>紫微在子午，只需不为「在野孤君」，则主下属晚辈得力，且有领导才能。若「在野」，则常易因主观而误事。若更见煞忌刑曜，则须提防难于驾御。</li>
<li>紫微在子午宫会昌曲，主子女聪明秀发，但若更见桃花诸曜，则主女多子少。又同时见煞刑诸曜，则子多克而女能留，终至变成女多子少。见煞而不见桃花，则主生二子。</li>
<li>紫微在子午宫带煞忌，主反受下属晚辈箝制。以贪狼宫垣为克应之期。又或以会合武曲化忌的宫垣为克应之期。</li>
</ul>


丑、未：紫微、破军
<ul class = "ds-中州派分析段落">
<li>「紫微破军」在丑未宫，以先生女为宜。倘煞刑诸曜会合，头胎生子者，防破相，重则刑克。或不足月生，或有产难。</li>
<li>「紫破」守子女宫主子女三人。见辅佐诸曜对星者，数目增加，但数目仍为单数，同时见煞，则由双数克至单数。</li>
<li>「紫微破军」在丑未，见昌曲桃花，主女多子少，亦主偏房或外宠生子。</li>
<li>「紫破」为昌曲左右同夹，亦主有孪生子。以不见煞同度为确。只见昌曲夹，主子女聪明；只见左右夹，主子女得力，且富创造力。</li>
<li>「紫破」见煞，一般主下属具反叛性。行至巨门、太阳、「武杀」见煞忌，为克应之期。又或自己命宫行至「刑忌夹印」，子女宫太阴化忌，则主受子女及晚辈拖累。</li>
</ul>


寅、申：紫微、天府
<ul class = "ds-中州派分析段落">
<li>寅申宫「紫微天府」主子女五人。辅佐对星同会，数目增加。且子女有相当数目。</li>
<li>寅申「紫府」见辅佐「单星」，必主有两重子女；故宜详视夫妻宫，推断其两重子女的意义为何。</li>
<li>「紫府」与禄存同度，见火铃，主偏房生子。亦主无晚辈追随。</li>
<li>「紫府」会武曲化禄，或廉贞化禄，或财荫夹的天相，则主子女富有，唯与儿子的关系易疏远，与女儿关系则佳。</li>
<li>寅申「紫府」会廉贞化忌，主因子女伤心流泪；亦主心腹反叛。会武曲化忌，主子女刑伤，或主子女倾家，亦主受晚辈箝制拖累。须详实际星曜组合而定。克应之期，为廉、武、巨、阳见煞忌刑曜守垣之大限或流年。</li>
<li>「紫府」守子女宫，而府库空露，主子女或晚辈常觊觎自己的钱财。</li>
</ul>


卯、酉：紫微、贪狼
<ul class = "ds-中州派分析段落">
<li>卯酉宫「紫贪」见辅佐诸曜对星，主子女七人；「孤君」则主四人；在卯者多行生女；在酉者主先生子。</li>
<li>「紫贪」与桃花昌曲同躔，子女极聪明，主先生女，同时见煞则主偏房生子。</li>
<li>「紫贪」两邻，一躔昌曲单星；一躔左右单星，则子女婚姻有事端，多为其配偶与自己不和。</li>
<li>「紫贪」喜见武曲、廉贞化禄；不喜化忌，其推断与「紫府」同。</li>
<li>「紫贪」与火铃同度，防子女或晚辈骄奢，并不主子女发达。若贪狼化忌者则更甚。</li>
</ul>


辰、戌：紫微、天相
<ul class = "ds-中州派分析段落">
<li>辰戌宫「紫微天相」，主子女四人；见辅佐对星则子女数目增加；见煞，则由单数克至双数。最不喜为刑忌夹，主子女懦弱，晚辈不得力。</li>
<li>「紫相」躔火铃，主子女宜远方发展。会禄存、化禄，则可勷助自己事业。</li>
<li>「紫相」与桃花、昌曲诸曜同躔，子女聪明，再见煞忌，则恐流连酒色风月。若空劫同度，则主偏房生子。</li>
<li>「紫相」守子女宫，一般主有晚辈扶助自己的事业，若煞忌刑曜重重，则反为所累。其吉凶克应，以「武曲天府」、廉贞二垣为枢纽。</li>
</ul>


巳、亥：紫微、七杀
<ul class = "ds-中州派分析段落">
<li>巳亥宫「紫微七杀」，主子女三人。见辅佐对星，数目增加，但不超过六人。同时见煞忌刑曜，子女有刑克。</li>
<li>巳亥「紫杀」，子女极强顽。须见禄以调和，否则有代沟，晚辈性质亦如是。</li>
<li>巳亥「紫杀」见「武曲贪狼」躔火铃，主子女远离，更见忌星，则子女因贪致败；论与晚辈的关系，则主受其侵吞剥削。</li>
<li>巳亥「紫杀」最喜对宫天府与禄存同度，则为创业人才。见武曲化忌，则长子少年体弱，但亦为独当一面的人才。唯亦主受晚辈拖累以致自身受损失。</li>
<li>巳亥「紫杀」躔禄马，或与禄马对拱，主子女能于远方创业。又主自己桃李满天下。</li>
</ul>

天机
<ul class = "ds-中州派分析段落">
<li>天机在子女宫，一般均主子少，唯偏房外室生者则多。在社会人际关系而言，亦主随自己的下属及晚辈稀少，或时时更换，关系不常。且均主迟得。</li>
<li>天机同会辅佐对星，则子女数目增加，但仍有女多子少的情况。尤其是与桃花诸曜同度时，决主女多，在申宫与太阴同躔时尤确。下属晚辈亦有同样情况。</li>
<li>天机同会四煞空劫，则主无子。躔桃花诸曜则可生女。若只会两三点煞曜，同时又见辅佐吉曜，则可偏房外室生子。</li>
</ul>


子、午：天机
<ul class = "ds-中州派分析段落">
<li>子午宫天机，与巨门相对，主只一子。又主时时更换直接追随自己的下属；如有门生弟子，关系亦不密切。</li>
<li>天机在子午宫，若与火铃同会，主有小产或流产。亦主子女出生后不久，即因客观环境远游外出，以致时时分离。</li>
<li>天机在子午宫见吉化，又得吉曜扶持，而无火铃同躔者，宜培养子女为专业人才，不宜从商。尤以计划、管理、财务等业为宜。</li>
<li>天机在子午宫，巨门化忌，或天机化忌者，宜注意子女性格，须改善其过高理想，不肯脚踏实地的缺点，否则易自误。</li>
<li>子午天机，以「武曲七杀」、「廉贞贪狼」、及天机各垣为吉凶克应的枢纽。</li>
</ul>


丑、未：天机
<ul class = "ds-中州派分析段落">
<li>丑未天机与天梁相对，主子女一二人。但另有小产流产。若与四煞空劫同会，则主无子，或有亦流产。亦主下属晚辈极少，且时时更换。</li>
<li>丑未天机见吉化，则可增加子女数目，且子女地位亦主提高。</li>
<li>丑未天机在子女宫，子女可能发展为喜挑剔的性格，宜加以改善，发展为监察、管理、计划等，即将挑剔化为心思缜密。</li>
<li>丑未天机，以「廉贞天相」、武曲、天梁各垣，为流年大运的吉凶枢纽。</li>
</ul>


寅、申：天机、太阴
<ul class = "ds-中州派分析段落">
<li>寅申宫「天机太阴」同度，在寅宫者子女三人，即二子一女；在申宫者则主女多子少，只主一子，若与桃花诸曜同会，女更多，唯亦主偏房外宠生子。</li>
<li>寅申「天机太阴」同度，见煞，又见桃花诸曜，复见左辅右弼，主「姑子归宗」。即自身无子，以姊妹之子承祀。唯现代或已少这种情形。</li>
<li>寅申「天机太阴」会天同化忌，又见煞曜重重者，主有小产流产。或子女小时多灾病。见辅佐诸曜对星或吉化，主子女数目稍增，亦可改善子女之品格及天赋。一般主心思浮荡不定，化忌则内向，且精神紧张。吉则主聪敏。</li>
<li>寅申「机月」，以天同、天梁、巨门三垣为克应之吉凶枢纽。</li>
</ul>


卯、酉：天机、巨门
<ul class = "ds-中州派分析段落">
<li>卯酉「天机巨门」，主只生一子。见桃花诸曜则多女。</li>
<li>卯酉「天机巨门」纵使吉化，亦主两代参商。见煞忌则更甚。用来推算跟晚辈的关系，则主隔膜，煞重又见阴煞、天虚等曜，则主为亲信出卖。</li>
<li>卯酉「天机巨门」为始善终恶之曜。推算子女宫时宜注意及此。尤不喜与火铃同度，则参商更甚。</li>
<li>若天机化忌，则多分离隔膜；若巨门化忌，则口舌纷争。必须见禄然后能改良性质，但只主彼此表面融和，内心仍多顾忌。</li>
<li>「天机巨门」以本身宫垣，巳宫的天梁，亥宫的天同为吉凶克应的枢纽。</li>
</ul>


辰、戌：天机、天梁
<ul class = "ds-中州派分析段落">
<li>辰戌「天机天梁」，稍见煞即主小产、流产。若煞忌刑耗诸曜重重，则主子女幼年多灾病，且有意外虚惊，及年长则远离父母。故以此推论与晚辈下属的关系，亦主不久长。</li>
<li>辰戌「机梁」见吉化，或会吉曜，则子女聪明，但有主见。不宜在幼年时对其诸多限制，否则阻碍其心智发展。若论晚下，则主可得独立工作能力之助手。</li>
<li>辰戌「天机天梁」，与火铃同度，主子女远离。与羊陀同度，则宜培养其理工方面的兴趣。</li>
<li>辰戌「天机天梁」，借会「太阳巨门」，以借会寅宫者为佳。「太阳巨门」主口才，故亦宜培养子女这方面的发展。</li>
<li>辰戌「机梁」，以「太阳巨门」、「天同太阴」两垣，为吉凶克应的枢纽。</li>
</ul>


巳、亥：天机
<ul class = "ds-中州派分析段落">
<li>巳亥宫天机独坐，与太阴相对，以巳宫为佳。亥宫天机对巳宫太阴落陷，日生人及下半月生人，对长女有妨。</li>
<li>巳宫天机，主二女一子；亥宫者，子极迟得。桃花诸曜同度则女儿数目增加。辅佐吉曜同度，则子女数目增加。</li>
<li>巳亥天机与四煞空劫同度，正室无子，主偏房外宠生子。</li>
<li>躔桃花，煞重，又见左辅、右弼「单星」，主「姑子归宗」。</li>
<li>巳亥天机若无吉曜，但与空劫、天姚同度，主晚辈及亲信有权术阴谋。又见昌曲、辅弼对星，则主女性者有助力。</li>
<li>巳亥天机以「天同巨门」、「太阳天梁」为吉凶克应的枢纽。</li>
</ul>

太阳
<ul class = "ds-中州派分析段落">
<li>凡太阳落陷在子女宫化忌，又见煞曜同会，夜生人，头胎以生女为宜，长子难养。日生人则情况略轻。</li>
<li>凡太阳在子女宫遇上述情形，主提拔后进反招怨忌。</li>
<li>凡太阳与太阴化忌会合，则视火铃，若火铃与太阳同度，则主子女与父母分离。此时即须视夫妻宫，推断有无夫妻分离的征兆。</li>
<li>凡太阳在子女宫，入庙吉，日生人尤吉；落陷凶，夜生人尤凶。若日生人入庙，化忌亦未必凶，仅主父子有代沟。倘辅佐诸曜毕集，太阳化忌而与巨门对拱或同度者，有时反主子女为法律、外交、推销等「口舌生财」的专业人士。</li>
<li>凡太阳在子女宫，必兼视太阴、天梁、巨门三曜，然后始能综合推断。</li>
</ul>


子、午：太阳
<ul class = "ds-中州派分析段落">
<li>子午太阳与天梁相对，以午宫为佳。子宫太阳，夜生人，头胎以生女为宜。若生子，主不成器。且子女易远离。午宫者，日生人尤吉，主男三女二，头胎以生子为宜。太阳化忌，主头胎有损。</li>
<li>子午太阳亦喜辅佐诸曜同会，子女数目可增加，且主子女有社会地位。午宫尤佳。</li>
<li>子午太阳与煞曜空劫同度，则子女有刑克分离，仅留一子送终。</li>
<li>子宫太阳，化忌，主父子有代沟；午宫太阳，化禄，则须防子女虚浮。与晚辈亲信关系亦可同断。</li>
<li>子午太阳以天同、「天机太阴」为吉凶克应的枢纽。</li>
</ul>


丑、未：太阳、太阴
<ul class = "ds-中州派分析段落">
<li>丑未宫「太阴太阳」同度，未宫主子女众多，无煞，可以八胎以上；更躔辅佐吉曜，数目尤多。丑宫无煞，四胎。见辅佐吉曜，数目增加。</li>
<li>丑未「阴阳」躔桃花诸曜，则主女多子少。夜生人尤甚。</li>
<li>丑未「阴阳」，有一化忌，主子女主观横蛮，人生亦较多挫折。晚辈亲信性质可以同断。</li>
<li>丑未「阴阳」与昌曲、龙池凤阁、天才等曜同度，主子女聪明。见魁钺则主受贵人提拔。化禄仪科则主贵。</li>
<li>丑未「阴阳」见巨门化忌借会者，主子女多思多虑，易悲观。子女宫行至借「机巨」安星的宫垣，多生事端。详实际星曜组合而定。</li>
<li>丑未「太阴太阳」以「机巨」、天同、天梁各垣为流年大运吉凶克应之枢纽。</li>
</ul>


寅、申：太阳、巨门
<ul class = "ds-中州派分析段落">
<li>寅申「太阳巨门」，子女三人以上。主有辩才，及有学习外语、方言的能力。见昌曲，主聪敏机智；见魁钺，主有说服别人的能力；见辅弼，主有领导力。若太阳或巨门化禄、权、科，则主子女富贵。</li>
<li>寅申「巨阳」，躔天刑，见吉曜吉化，子女为法律人才；若叠禄，则宜异族生财；若见昌曲禄马，则宜外国谋生。</li>
<li>寅申「太阳巨门」若巨门化忌，则必会吉化，故主子女「口舌生财」，并不主凶。</li>
<li>若煞忌刑曜齐集，则主子女有刑克，并不使其本质变坏。</li>
<li>可用以上星曜性质来推断晚辈、亲信的本质，则能知人善用。</li>
</ul>


卯、酉：太阳、天梁
<ul class = "ds-中州派分析段落">
<li>卯酉宫「太阳天梁」，擎羊同度，主子女出生时需动手术。若太阳化忌，则长子幼年有灾难虚惊。</li>

<li>「太阳天梁」只需见禄，与文曲同躔亦吉，在现代为理工科学研究人才。</li>
<li>卯酉「太阳天梁」，以「天同巨门」、破军两垣，为吉凶克应的枢纽。</li>

辰、戌：太阳
<ul class = "ds-中州派分析段落">
<li>辰戌宫阴阳对拱，主子女四人。辅佐及桃花皆主子女数目增加。辰宫太阳主聪敏及具魄力，戌宫者平常。</li>

巳、亥：太阳
<ul class = "ds-中州派分析段落">
<li>巳宫太阳独坐，与巨门对拱亦主子女聪敏，唯在巳宫则更具毅力。</li>
<li>巳亥宫太阳见吉化、吉曜同会，子女亦为创业或专业人才，可参考「太阳巨门」，唯巳亥宫者，商业性质更重。</li>

<li>巳亥宫太阳，以天同、天机、巨门为运限年限吉凶克应的枢纽。</li>

武曲
<ul class = "ds-中州派分析段落">
<li>武曲临子女宫，一般情形下主子女少，又较利女不利子，故多先生女。亦主追随自己的后辈较少（此则同时须视左辅右弼）。</li>
<li>武曲在子女宫，亦主发妻无子，但可有女。而偏房、继室则可生子。若火铃同度，则亦主外室生子，或祀养他人之子。</li>

<li>武曲在子女宫，羊陀不甚畏，只畏火铃同度。唯若武曲化忌，为羊陀所夹（这情形只有亥宫的「武曲破军」），则主子女刑克甚重，甚至可能自杀。亦主无子送终。</li>
<li>武曲为火铃夹亦不利。主子迟得。「武曲贪狼」与火铃同度，子虽在四十岁以前可得，但必主生离。辅佐吉曜同会，则为良性的分离，如儿子寄宿学校之类。</li>
</ul>


子、午：武曲、天府
<ul class = "ds-中州派分析段落">
<li>子午宫「武曲天府」，主得三子。由于天府为南斗主星，故若辅佐对星会合，数目可增加至五胎。若同时见煞忌刑空劫，则仅得一子送终。「府库空露」，则子女及晚辈不得力。</li>

丑、未：武曲、贪狼
<ul class = "ds-中州派分析段落">
<li>丑未宫「武曲贪狼」，主四十后得子。这是古人的说法。古代婚姻年龄较早，四十岁得子已称迟得，在现代，得子的年龄应更晚。</li>
<li>丑未「武贪」见煞在外，又见桃花诸曜，则主偏房、外室生子。发妻则生女。</li>
<li>丑未「武贪」若与火铃同度，则主得子尤迟。若早年生子必须离家、寄养，否则恐有刑克。父子感情至晚年有隔膜。推断跟晚辈亲信的关系，亦主始吉终凶。</li>
<li>丑未「武贪」以「机梁」、「巨阳」两垣为吉凶克应之期。</li>
</ul>


寅、申：武曲、天相
<ul class = "ds-中州派分析段落">
<li>寅申宫「武曲天相」，主先祀一子，然后亲生一子。会桃花，则偏房先生子，然后发妻生子。在现代，亦主婚外生子。与晚辈亲信人际关系不佳。</li>
<li>寅申宫「武曲天相」，见煞忌刑曜，主子女有刑克。见辅佐吉曜对星，始主子女数日增加；见单星，主婚外生子。</li>
<li>寅申「武曲天相」以贪狼、七杀两垣，为占凶应验的枢纽。</li>
</ul>


卯、酉：武曲、七杀
<ul class = "ds-中州派分析段落">
<li>卯酉「武曲七杀」，仅得一子。若见煞，如多于一子，则主长子有破相、伤残，且往往主继室、外室生子。见辅佐吉曜，主得子性格刚烈。若煞曜空劫与吉曜吉化齐会，则主因数性刚而破家。</li>
<li>卯酉「武曲七杀」，武曲化忌为火铃夹者，刑克伤残极重。必须同时见辅佐吉曜，然后始主迟得子。若躔桃花昌曲，则可能有女无子。与晚辈人际关系同此推断。</li>
<li>卯酉「武曲七杀」见煞，不宜招祀外子，恐恩反招怨。故亦主追随自己的晚辈反叛。</li>
<li>卯酉「武曲七杀」，以巨门、天机、「廉贞贪狼」三垣为吉凶克应之期。</li>
</ul>


辰、戌：武曲
<ul class = "ds-中州派分析段落">
<li>辰戌宫武曲独坐，子可较早得，主生一子。见辅佐诸曜则可增加至三子。唯仍以先得女为佳，否则长子幼年恐多病。</li>
<li>辰戌宫武曲与煞同度，则仅得一子，四十以后得者易养。</li>
<li>辰戌宫武曲会贪狼化忌，又有煞曜同度，则主防子女骄奢倾家。躔空劫、火铃尤甚。若躔羊陀，则仅主极迟得子。</li>
<li>辰戌宫武曲会贪狼化禄，又见桃花，则主外室或继室生子。倘又躔火铃，仍防子女骄奢以致倾败。</li>
<li>辰戌宫武曲，以「廉贞天相」、贪狼、七杀三宫垣为流年大运吉凶克应的枢纽。</li>
</ul>


巳、亥：武曲、破军
<ul class = "ds-中州派分析段落">
<li>巳亥宫「武曲破军」，主生一子。见煞，多于一子则有刑克，刑至一子为止，须过继与人然后可免。或主长子破相。</li>
<li>「武曲破军」守子女宫，基本意义为父子不和。或因各种客观情形以致父子分离。持此推算跟晚辈的运程，可以同断。</li>
<li>巳亥「武破」，仅于无煞曜同度，但三方四正有辅佐诸曜「对星」会合或同度的情形下，可得三子。唯仍易与子女分离。在此情形下，儿子（特别是长子）往往不足月生，或须动手术生产。</li>
<li>巳亥「武破」不宜见廉贞化忌，常主晚年丧子。唯仍须详晚运子女宫之实际星曜组合情形而定。</li>
<li>巳亥「武破」以本宫及「廉贞七杀」、「紫微贪狼」三宫垣为吉凶克应的枢纽。</li>
</ul>

天同
<ul class = "ds-中州派分析段落">
<li>天同在子女宫，一般情形下主子女众多，虽头胎以生女为宜，则虽见煞曜，亦不主小产、流产、刑克。</li>
<li>天同在子女宫，主与子女感情易沟通。亦可持此论断跟晚辈的关系，唯嫌其怕事。</li>
<li>天同在子女宫化忌，若为天同、太阴；或天同、天梁的组合，更见煞曜，须防子女中有弱智、自闭症者。</li>
<li>天同化禄，主子女白手可以兴家。但亦主追随自己的晚辈易脱离自己，另起炉灶。</li>
<li>天同见辅佐诸曜单星，主有两重子女。若夫妻宫不见再婚或有外宠的情形，则为收养祀子。</li>
<li>天同与桃花诸曜同度，主多生女。尤以天同、太阴的组合为然。</li>

子、午：天同、太阴
<ul class = "ds-中州派分析段落">
<li>子午「天同太阴」，一般情形下主女多子少，且主先生女。在子宫者，子女五人，午宫者三人。见辅佐吉曜可增加数目；见煞忌空劫则有刑克、伤残。</li>
<li>子午「天同太阴」见昌曲、龙凤、天才，主子女极聪明。单见龙池，则主有音乐天赋。若躔火铃，则防聪明太露。</li>
<li>子午「天同太阴」见阴煞、天姚，又无辅佐对星同会，则子女恐过份内向。若化忌，则可能发展为权术之士。</li>
<li>子午「天同太阴」见「太阳巨门」吉化，又有吉曜同会，则主子女受异族提拔、赏识。又主有学习方言、语言能力。唯不如子女宫「太阳巨门」躔度者之佳。主要分别，在「天同太阴」者际遇较少。</li>
<li>子午「天同太阴」以「太阳巨门」、「天机天梁」及本宫为吉凶克应的枢纽。</li>
</ul>


丑、未：天同、巨门
<ul class = "ds-中州派分析段落">
<li>丑未宫「天同巨门」主子女三人，亦以先生女为宜。见辅佐诸曜增加，桃花重重，则头三胎先生女，然后始得一子。</li>
<li>丑未「天同巨门」喜煞曜激发，否则主子女意志薄弱，唯有煞则又主刑克。</li>
<li>丑未「天同巨门」躔昌曲，又躔桃花，则子女聪敏而毅力不足。女儿尤易失足，或有畸恋苦恋。</li>
<li>丑未「天同巨门」若巨门化忌，则主骨肉参商，亦主与自己提拔扶持的人反目；倘天同化忌，则子女多灾病。</li>
<li>丑未「天同巨门」，以化禄为宜，但却又主子女刚毅不足。</li>
<li>丑未「天同巨门」，以紫微、贪狼、「太阳天梁」为流年大运吉凶克应的枢纽。</li>
</ul>


寅、申：天同、天梁
<ul class = "ds-中州派分析段落">
<li>寅申「天同天梁」宜先生女后生子，主有子二人。唯在申宫者，稍见煞忌，即易仅留一子。辅佐诸曜同会，则子女数目增加。</li>
<li>寅申「天同天梁」见煞忌重重，亦主子女伤残，或弱智。火铃、空劫尤甚。</li>
<li>寅申「天同天梁」见天马，主子女离家发展。</li>
<li>天梁化禄，更见辅佐吉曜，宜培养子女为专业人才；见煞，则以工程科技为佳。</li>
<li>寅申「同梁」，见天机化忌、天同化忌，与煞曜同度，主有流产、小产。</li>
<li>寅申「同梁」见太阴化忌，又见禄马，子女破尽祖业而后离乡兴家；天同化禄，亦主子女自兴，唯只主不能承继祖业。</li>
<li>寅申「同梁」以天机、巨门、太阴三垣及本宫为流年大运吉凶克应的枢纽。</li>
</ul>


卯、酉：天同
<ul class = "ds-中州派分析段落">
<li>卯酉天同独坐主女多子少。卯宫者子女五人，酉宫则四人或二人，或由单数刑至双数；不刑则主少小离家。</li>
<li>卯酉天同，与子午宫「天同太阴」有类似性质，可以同参。唯不会「太阳巨门」，故性格较为好高鹜远，且理想高而少实际。</li>
<li>若巨门化忌，主子女性格重理想而脱离现实，但却固执。</li>
<li>卯酉天同以巨门、太阳、太阴三垣，为流年大运吉凶克应的枢纽。</li>
</ul>


辰、戌：天同
<ul class = "ds-中州派分析段落">
<li>辰戌宫天同独坐，子女二至三人。见煞则只一人。见辅佐吉曜则增至三人。</li>
<li>辰戌宫天同，主子女感情重于理智。见昌曲桃花者，感情尤重。</li>
<li>辰戌宫天同化禄，主子女不享祖业而自兴。化忌，则宜业破败。</li>
<li>辰戌宫天同与陀罗同度又躔火铃，主中年子女有伤残、意外。</li>
<li>辰戌宫天同，以天梁、巨门二垣，为吉凶克应的枢纽。</li>

<ul class = "ds-中州派分析段落">

<li>巳亥天同，最不宜见巨门化忌，主刑克分离。见煞者尤甚。</li>
<li>巳亥天同化禄，虽不见马，亦主子女离乡背井。</li>
<li>巳亥天同，视借会之「太阴太阳」定子女吉凶休咎。如太阳化忌，有煞，夜生人决主克子。</li>
<li>巳亥天同以「太阴太阳」、「紫微天相」二垣为吉凶克应的枢纽。</li>

廉贞
<ul class = "ds-中州派分析段落">
<li>廉贞喜入子女宫，一般情形下，主两代感情浓厚，有隔膜亦能互相谅解。所以廉贞化忌，往往便代表感情上的创伤，亦即子女的夭折。这跟巨门化忌之表征为口舌不和，恰成对比。</li>
<li>廉贞化禄在子女宫，代表感情上的喜悦，故可表征为得佳儿佳女、星曜吉化未必表征子女数目增加、唯廉贞化禄则可有此表征。</li>
<li>廉贞化忌、化禄，对此星曜的影响非常敏感，上述仅为一般原则，具体星曜组合如何，时能改变一些细节。</li>
<li>例如同一廉贞化忌，有时仅代表子女以怨报德，并不代表刑伤。主要视煞星与廉贞是否同度而定。若有两颗以上煞曜进入子女宫，则可视为刑伤夭折。若煞曜仅在三方，则为以怨报德。</li>
<li>廉贞与天府同度或相对，为最稳定组合；与破军同度或相对，最不稳定。此为推断原则。</li>
</ul>


子、午：廉贞、天相
<ul class = "ds-中州派分析段落">
<li>子午宫「廉贞天相」，主子女二人；见桃花诸曜或辅佐诸曜皆主增加。若见辅佐单星，又见桃花重重，则主两重子女。</li>
<li>子午「廉相」为财荫夹，子女虽可秀发，但得子迟，三十以后子女常易因小产流产而致刑克。若为刑忌夹，则反主子女妨害父母。须详夫妻宫同断。</li>
<li>子午「廉相」有煞忌空劫刑耗诸曜同会，主亲子刑伤，以立祀子为宜；若同时见吉曜，则立祀后可得一子。同时见桃花，主外室庶室生子。</li>
<li>子午「廉相」以贪狼、破军、及本宫为流年大限吉凶克应之期。</li>
</ul>


丑、未：廉贞、七杀
<ul class = "ds-中州派分析段落">
<li>丑未「廉贞七杀」，主仅得一子。见辅佐诸曜可增加数目，且子女品德性格亦有改善。若四煞空劫天刑同会，则主有刑克，可能克至仅得一子。</li>
<li>丑未「廉贞七杀」，有火铃同度，子女主生离，或宜寄养。廉贞化禄，则感情仍深厚；化忌，则主父子参商。又主晚辈以怨报德。</li>
<li>丑未「廉贞七杀」见吉化，及见辅佐吉曜则子女成才。</li>
<li>丑未「廉杀」以天相、「紫贪」、「武破」及本宫为吉凶克应的枢纽。</li>

寅、申：廉贞
<ul class = "ds-中州派分析段落">

<li>寅申廉贞以杀、破、狼三垣为流年大运吉凶克应的枢纽。</li>

卯、酉：廉贞、破军
<ul class = "ds-中州派分析段落">
<li>卯酉宫「廉贞破军」，主得一子。见辅佐诸曜对星，则子女数目增至双数。见煞曜空劫刑耗，则子女克至仅留一人；若吉煞混杂，亦主克至单数。见辅佐「单星」，亦主子女单数。</li>
<li>卯酉「廉破」一般情形下主长子难养。加吉、吉化，则或主长子破相。若见辅佐诸曜「单星」，则主收养别人子女；或自己将长子长女出祀承继。然亦有特别情形，主有「假子」（如向政府或宗族虚报之类）。</li>
<li>卯酉宫「廉破」，最喜化禄。廉贞化禄则子女善于运筹帷幄；破军化禄则甚具组织力，宜依此善为培养。对追随自己的晚辈，同此推断。若廉贞化忌，则子女性格反叛，而且往往命运有重大挫折。故再加煞刑空劫，则主夭折或伤残。但化忌而见吉曜，则反主为人温文有礼，而具突破现状之才能。亦可依此推断晚辈，故主有青出于蓝的佳弟子。</li>
<li>卯酉「廉破」，以本宫、「天机天梁」、「太阳巨门」为吉凶克应的枢纽。</li>
</ul>

<li>天府与禄存同度，头胎常过产期生产。所会天相为财荫夹者尤然。唯若火铃与天府同度，则反主不足月生。</li>
<li>天府与煞曜刑耗同会，主子女性情倔强，又主破相。</li>
<li>天府若得「百官朝拱」，倘辅弼、昌曲、魁钺毕集，可至十胎。更见桃花，则子女在十人以上。</li>
<li>天府坐子女宫，一般以对宫，及天相、破军所躔宫垣，为流年大运吉凶克应的枢纽。</li>
</ul>

太阴
<ul class = "ds-中州派分析段落">

<li>太阴在庙旺宫垣，夜生人、上半月生人，子女数目均多，主五至八人。若日生人、下半月生人，则为三至五人。</li>
<li>太阴在陷宫，夜生人、上半月生人，子女数目可有三人，若日生人又为下半月生人，则以先招祀子或外室先生子为宜，然后始主得子。</li>
<li>太阴守子女宫，见煞忌空劫刑耗诸曜，则子女有刑克。以日生人、下半月生人为最重；夜生人、上半月生人则轻。</li>
<li>太阴庙旺，见禄，主子女富有；落陷，则主子女软弱。庙旺见昌曲等科文诸曜，主子女聪明；见魁钺及三台八座、恩光天贵等科名诸曜，则主子女贵显。见辅弼，则主子女有助力。</li>
<li>太阴与火铃同度，更见天机，子女决主离乡背井。亦主多病。</li>
<li>太阴落陷而见吉化吉曜，则子女虽佳，但终嫌欠领导力。宜后天予以抚育栽培。</li>
<li>太阴落陷，禄存同度，则仅得一子。煞重，则宜庶室或外室生子。</li>
<li>太阴与桃花诸曜同度，多女少子，落陷且见火铃，则或主有女无子。</li>
<li>太阴独坐子女宫，以太阳、巨门、天同及本宫为流年大运吉凶克应的枢纽。</li>
</ul>
