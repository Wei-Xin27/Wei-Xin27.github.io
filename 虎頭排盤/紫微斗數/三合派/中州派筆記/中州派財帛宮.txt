紫微 

<ul class = "ds-中州派分析段落">
<li>紫微守财帛宫，在最好的情形下，亦仅主财源充裕，并不主能积聚。盖紫微偏重于名誉与权力，不是财星。</li>
<li>紫微化权守财帛宫，见煞，因同时必见武曲化忌，故绝不主富厚，除非从事「持利器以求财」的行业（如外科、牙科、饮食业、理发业之类），否则反主场面似乎丰裕，实际却支绌周张的情形。</li>
<li>紫微化权不见煞，反见「百官朝拱」，则金钱方面易得他人支助，命宫为「廉贞天府」者，亦主可掌财权。</li>
<li>紫微化科，则宜凭名誉、商誉以求财。得「百官朝拱」者尤然。见煞，不过声誉减等，或范围缩小。</li>
<li>紫微喜与禄存或化禄同度，或对拱，则可改变其外表充裕，成为内在充实，化科化权的紫微尤宜。</li>
</ul>


子、午：紫微
<ul class = "ds-中州派分析段落">
<li>紫微与火星同度，居子午宫，因与贪狼相对，故亦主一时横发。或对宫贪狼成「火贪」、「铃贪」格者亦然。除非命宫或事业宫有吉化、吉曜，否则难以久长。</li>
<li>子午宫紫微在财帛宫，有左辅右弼同会，主财源来自多方面。余宫紫微减等。亦主时时可得别人在财帛上加以支持、供应。</li>
<li>若对宫贪狼化忌，本宫紫微吉煞交集，则受人供应金钱，必带有相当交换条件，甚至可流为贪墨。</li>
<li>子午紫微为「孤君」者，主进财艰辛，若「在野」，即为重重煞刑耗曜在三方四正包围，则主挥霍，纵一时横发亦不能久存。</li>
</ul>


丑、未：紫微、破军 
<ul class = "ds-中州派分析段落">
<li>丑未宫「紫微破军」在财帛宫，主一生必可得一次意外财。以程度言，未宫优于丑宫；见吉者性质优良，见煞者则带凶险性质。唯得意外财后，必于流年经行至丑或未宫时，即生极大破耗。若有空劫同度，破耗后较得财前更贫困。</li>
<li>丑未宫「紫微破军」而破军化禄者，若掌财权，则易贪墨。或主贪图侥幸，犯险冒难以得偏财。</li>
<li>若破军化权，对主运用金钱的能力强，经波折而成富。故在财帛宫，「紫破」之破军化禄不如化权。唯二者皆主挥霍。</li>
<li>凡「紫破」、「紫贪」、「紫杀」皆主骤发骤败，「紫破」之败，必因发后又妄图开创所致。</li>
</ul>


寅、申：紫微、天府 
<ul class = "ds-中州派分析段落">
<li>寅申宫「紫微天府」，需视有禄无禄以厘定其吉凶。若有禄，则可积聚而成富，无禄，则不外充足而已；倘无禄有煞，或且见空劫，则仅主表面风光，且常易因求财反生破败。</li>
<li>「紫微天府」仅主积蓄致富，故与「紫微破军」不同，在保守社会，「紫府」较佳，在进取社会，「紫破」则较佳。唯「紫破」于发后须采取保守态度，防再见破耗。</li>
</ul>


卯、酉：紫微、贪狼 
<ul class = "ds-中州派分析段落">
<li>卯酉宫「紫微贪狼」亦主骤发，若紫微化科，则财来较悠久，若贪狼化禄，则财来较急骤，然而亦隐藏破耗之机。</li>
<li>「紫贪」有「百官朝拱」，则主可发财而非暴发；若「在野」或「孤君」，则仅主贪财，可能不择手段。</li>
<li>「紫贪」与禄存同度，见空劫者，主暴败。若火铃同躔，暴发暴败尤速。然有火铃而无空劫，则尚可图补救，于发后持盈保泰即可避免。</li>
<li>「紫贪」为火铃夹者，亦主骤发。夹贪狼化禄，则骤发程度更大。尤喜夹紫微化科，则骤发且有声誉，或由是建立商誉。</li>
<li>「紫贪」在财帛宫，亦主挥霍，时时作无谓交际应酬。但贪狼化禄者，却亦能因应酬而得财。</li>
<li>「紫贪」在财帛宫，财源来自多方面。</li>
<li>凡「紫破」、「紫贪」、「紫杀」皆主骤发骤败，「紫贪」之败，必因发后即浪费金钱所致。</li>
</ul>


辰、戌：紫微、天相 
<ul class = "ds-中州派分析段落">
<li>辰戌宫「紫微天相」虽吉化吉曜汇集，一生亦必有一次波折，或呈胶着状态，难得寸进。必须渡过此段时期，然后始能开运。</li>
<li>倘有刑煞忌耗诸曜，则不但时生波折，而且容易因多意外开支，而致入不敷支。</li>
<li>「紫相」为财荫夹者，可克享父荫，或掌家族财权，克承父业。若为刑忌夹，则主祖业凋零，而身亦常因财源而生压逼感。</li>
<li>「紫相」有左辅右弼同会，则宜协助他人理财，不宜积极开展自身事业。纵有自身事业。亦以作为他人之附庸者为宜。如代理业务、经销业务、经纪行业等。亦可从事服务性专业。</li>
<li>唯「紫微天相」同度守财帛宫者，一生亦必有意外之财，可主突发。无煞忌者则能储存。</li>
</ul>


巳、亥：紫微、七杀 
<ul class = "ds-中州派分析段落">
<li>「紫微七杀」在巳亥宫，理财胜于积财，故常常事业广大，而自身积蓄却不相称。这种财帛宫的格局，只宜以企业公司的资产，作为自己的资产。</li>
<li>「紫杀」与禄存同度，守财能力较大，但易因财失义。</li>
<li>「紫杀」在财帛宫亦主能骤发，唯较难持久。见煞者尤甚。</li>
<li>「紫微七杀」亦主财帛变动，故如自己经营事业，则必须详视流年吉凶，以为进退参考。于逆境时不宜妄动，否则必多破败。见禄者则较佳。</li>
<li>「紫杀」与空劫同度，破败甚速。</li>
<li>唯「紫杀」见「禄马交驰」之局者，无煞，在外埠外国能发，在出生地反多波折。亦即仅适合离开出生地的环境。</li>
<li>「紫微七杀」而紫微化权，尤宜背井离乡。</li>
<li>凡「紫破」、「紫贪」、「紫杀」皆主骤发骤败，「紫杀」之败，则在于环境忽生变动。故「紫杀」较难由当事人主动趋避。</li>
</ul>

天机

<ul class = "ds-中州派分析段落">
<li>天机在财帛宫，主财帛有变动或波动。通常表现为财来去，难以积存。入庙者较佳。若落陷，则财源稍顺即生变化，或必须劳心费神，腾挪变化然后始能保住财源。</li>
<li>天机化权，则性质比较稳定，然而仍必须转换财源，表征为转行转业。故以自由职业为佳；亦宜从事专业工作，则可不改行业，但改变来财之客户即可（例如律师，不必固定的客户）。</li>
<li>天机为计划的星曜，在财帛宫时，亦宜倾向于劳心以求财。</li>
<li>天机化禄，叠会禄存，且有吉曜，则亦可骤发。唯其骤发，必与聪明机巧有关。若化忌，则机关费尽反而财源中断，所谓「机深祸更深」是也。</li>
<li>天机化禄，财气佳，但仍主财源不只一端，会吉曜，主有多方面的财路。若吉煞交集，则多反复进退。</li>
<li>天机化忌，主财运稍顺即遭破坏，打击或排挤。必须于逆境时处之泰然，让时间澄清一切，表现出泱泱大度，然后始可保泰持盈，安渡逆境。</li>
<li>天机化忌者，亦以从事专业为宜，且以劳心费神的行业为佳。</li>
<li>天机擎羊，主劳心的程度更大。</li>
</ul>


子、午：天机
<ul class = "ds-中州派分析段落">
<li>子午宫天机入庙，必与巨门相对，故主劳心费神，破唇费舌以求财。巨门化禄者佳，化忌者劳神费舌的程度更大。</li>
<li>午宫天机，命宫太阴入庙；子宫天机，命宫太阴落陷，故子宫不如午宫。在午宫者焦虑较小亦较少。在子宫者，须心情开朗以应付横逆，且必须不用机关心计，然后始能扭转命运。</li>
<li>子午宫天机若与煞星同度，则主易失机缘。又或因求全求备，追求理想而反致误失良机。且一生多竞争，以致尽抛心血。</li>
<li>子午天机与空劫同度，主机谋失误。</li>
<li>子午天机化禄，或禄存同度，表征为钱财流通，故可从事经纪行业，费唇舌而能得钱财流通之利，此亦以与巨门相对之故也。</li>
</ul>


丑、未：天机
<ul class = "ds-中州派分析段落">
<li>丑未宫天机落陷，与天梁相对，主机关巧计以求财。因此最宜从事费心机计谋人行业，如电脑程序、市场策划之类。倘有天刑、擎羊、奏书等曜，则宜从事法律工作。</li>
<li>由于与天梁相对的关系，若天梁化科，则主依靠知名度而决定财帛的多寡。天机化禄、权、科，会昌曲化科者亦然。但若天机化忌，或会昌曲化忌，则往往计谋失误，失时便应坦然承受一些失败，作为经验，以助日后的开运。</li>
<li>丑未天机与天梁化禄相对，则必主机关巧计以求财。倘见阴煞、天姚、天虚同度，则计谋的性质更重，遇吉曜，可以得财，倘有煞曜同度，则主枉抛心力，不如随缘，尤以空劫同度者更甚，亦不宜独立经商。</li>
<li>丑未天机见煞忌刑耗诸曜，主因官司破败。故不宜争讼。</li>
</ul>


寅、申：天机、太阴
<ul class = "ds-中州派分析段落">
<li>寅申「天机太阴」同度，见禄存或化禄，宜在财经机构服务。古人认为乃「财赋之官」因为「天机太阴」仅主财经计划，不主求财的行动，是故不宜经商。</li>
<li>若太阴化忌，则主关于投资的计划失误。此时申宫又较优于寅宫，即失误的程度较小。然均不利于经营，只宜服务公职或财经机构，以上市公司最为适宜。但仍不宜主持实际投资项目。</li>
<li>若太阴化禄则主财气丰盈，虽任职亦主收入丰厚。天机化禄者，过手的金钱数目相当大。</li>
<li>「天机太阴」亦不喜见命宫天梁化禄，故不宜经商，即使任职亦主人缘不佳，以其事业宫必同时见陀罗与天同同度之故，主事业上多暗争，影响财帛。</li>
<li>「天机太阴」最喜与「禄马交驰」同度，则主发财于远方。故以离开出生地为宜。</li>
</ul>

卯、酉：天机、巨门
<ul class = "ds-中州派分析段落">
<li>卯酉宫「天机巨门」，主竞争甚重，经手之事每惹人垂涎。见煞忌者尤甚。</li>
<li>若巨门化禄，则主得异族之财。叠见禄存，尤足以成巨富，唯必费尽心神。</li>
<li>若巨门化忌，则往往功败垂成，枉抛心力。此时不如随缘，或从事竞争较小的行业。</li>
</ul>

辰、戌：天机、天梁
<ul class = "ds-中州派分析段落">
<li>辰戌宫「天机天梁」同度，机关巧计以求财的意味，若会吉曜者，则主易得荫庇，故有祖业，故宜从事专业。天机化科或天梁化科，均主名利双收。</li>
<li>「机梁」不宜化禄，天梁化禄尤主多暗争，每易受人排挤，故必须劳心费神以求财。</li>
<li>「机梁」多变动，故见辅弼、魁钺者，指财源不只一端，往往兼职兼业。</li>
<li>「机梁」会化忌，无论天同、太阴、太阳、巨门，皆主是非暗争极重，尤不喜太阴化忌，主时时枉费精神，求财手段错误。</li>
</ul>

巳、亥：天机
<ul class = "ds-中州派分析段落">
<li>巳亥宫天机与太阴相对，不宜自行经营，但却可从事服务性行业，或居间代理。</li>
<li>巳亥天机见禄马交驰，以巳宫天机较吉，更宜作外国公司经纪或代理人。</li>
<li>巳亥天机不主聚财，化吉亦只主增加金钱流通程度，故亦可从事财经、税务、核数工作。</li>
<li>巳亥天机与火星同度者，财帛更难积聚，则源亦时生变动，故不可因一时顺遂即理想过高。</li>
<li>巳亥天机得禄，并主为人觊觎，以致收入不稳，财源不久。</li>
</ul>
-------------------------
太阳

<ul class = "ds-中州派分析段落">
<li>太阳主贵不主富，故太阳守财帛宫者，其财皆凭社会地位而来。因此必须争取地位，如利于学术者，应追求较高学位；利于经商者，应追求代理名牌商品，或自行创造商标商誉；利于任职者，应积极进修，争取声誉，或凭人际关系创造升职条件。</li>
<li>太阳主散，太阴主聚，故太阳守财帛宫，除非有太阴同度，否则必主散财。散财性质可能格于环境，可能为人际关系所逼，亦可能基于自身性格。详各星系组合而定。</li>
<li>一般而言，太阳化禄，乃自身喜散财；太阳化忌，主受六亲剥削；太阳化科，乃因追求荣誉而散财。唯太阳化权较为稳定，可以藉人事控制，作后天补救。</li>
<li>一般情况下，煞星对太阳的影响并不如对其他正曜之重要。换句话来说，太阳与四煞空劫同度。不过增加困扰与教财的趋向而已。倘如命宫星曜组合吉，则仍不乏进财之力，并不因煞曜而致贫乏。</li>
<li>太阳入庙，情形远较落陷者为佳。盖入庙的太阳影响命宫，社会地位较受落陷太阳影响者为高。既然太阳居财帛宫，得财多寡决定于社会地位，故自以财帛宫太阳入庙为优胜。</li>
<li>太阳主贵，故喜会财星，则富贵绵远。然而太阳化禄亦主散财，因此只有当见禄存、太阴化禄、巨门化禄时，始主富贵。太阴化禄最吉，亦最宜经商。单见禄存则远不如见禄马交驰。</li>
<li>太阳守财帛宫，若夫妻宫有聚财之星、如天同化禄，或「天同太阴」任一化禄化权之类，则财富宜交配偶处理。若田宅宫佳，如「紫微贪狼」而贪狼化禄，则可籍置业以聚财。此为利用星曜性质，作后天补救。</li>
<li>太阳守财帛宫，吉煞交集者，每主事业空虚不实，表面风光，而内在周绌。此可藉后天人事匡正，必须脚踏实地，不滥用声誉，然后始得改变。</li>
</ul>


子、午：太阳
<ul class = "ds-中州派分析段落">
<li>子午宫太阳，必与天梁相对，午宫固优于子宫，唯因天梁的影响，主贵的程度甚大，因此必须凭声誉地位以求财。子宫太阳落陷，不过花耗精神而已，故尤宜进行艰苦学术研究，或以解决难题（如医治危症、重症；或济困扶危的社会服务），作为一己的财源。</li>
<li>子午太阳守财帛宫，能成「阳梁昌禄」之格，必有专业地位。唯午宫声名较大于子宫。</li>
<li>子宫太阳见煞忌空劫，则花耗精神甚大；午宫者较小。</li>
<li>子宫太阳见吉化、吉曜，虽主富贵，然不如午宫者远甚。午宫太阳居财帛宫，若日生人，又有吉化吉曜，且见禄，则主既富且贵，权势及财权皆倾动一时一地。</li>
<li>子午太阳守财帛宫，唯不喜天梁化禄，盖益主贵而不主富，稍多进益，即有是非尤怨谣忌。</li>
<li>现代社会，子午太阳亦喜经营名牌，或藉广告创造的商誉，见科文诸曜者尤然。但另一方面，又可转为自身经营传播、广告。</li>
</ul>


丑、未：太阳、太阴
<ul class = "ds-中州派分析段落">
<li>丑未宫「太阳太阴」同度，基本性质主先散后聚。尤以丑宫者，「后聚」的力量大于未宫。夜生人又优于日生人。</li>
<li>太阳太阴同守财帛宫，见辅弼、魁钺、昌曲者，不见煞忌，主富而且贵。唯必须白手兴家，虽有祖荫亦不克享。太阴化禄者富多于贵，太阳化禄者贵多于富。前者在丑宫较利，后者在未宫者较优。</li>
<li>「太阳太阴」而太阴化忌，主因事业浮夸，妄事投资而引致损失；若太阳化忌，则主因虚名而受拖累。见煞刑诸曜，尤主由是引起是非，甚至因此涉讼。</li>
</ul>


寅、申：太阳、巨门
<ul class = "ds-中州派分析段落">
<li>寅申宫「太阳巨门」同度。太阳化权，巨门化禄者，主受异族提拔得财。亦主凭口舌以得财。无吉化，有禄马交驰亦可；太阳化禄亦有此种征兆；唯财易因出手阔绰而散去。</li>
<li>寅申宫太阳化忌者，必主受人拖累；巨门化忌者，则主引致官非词讼。更见煞耗空劫，则主因此负债、破产或倒闭。</li>
<li>若见巨门化忌而煞不重者（主要为不见空劫及铃星），则主以口舌以求财。例如经纪人、教师等。</li>
<li>「太阳巨门」纵使见吉化、吉曜，亦主财由自身创立，不主得现成产业。亦主必须竞争而后得。见煞忌者，竞争尤剧。煞忌交集，必须任劳任怨以求财。</li>
</ul>


卯、酉：太阳、天梁
<ul class = "ds-中州派分析段落">
<li>卯酉宫「太阳天梁」同度，成「阳梁昌禄」格者，由社会地位决定其财富之多寡。倘见文曲化科者，太阳必同时化权，又得借会巨门化禄，尤主既贵且富，权力与财富皆令人艳羡。</li>
<li>卯宫「太阳天梁」即使不见煞忌，亦主争产争财。唯见吉化及禄存，财帛却优于酉宫。</li>
<li>卯酉宫「阳梁」纵有吉化，如大运财帛宫见忌星，又与空劫同会者，此大运即为财帛困乏之期。若无此大运，则于流年见同样情形时亦克应。</li>
<li>「太阳天梁」最不喜空劫，主虚空不实，表面浮华而内在困乏。太阳化禄、化科者浮华更甚，天梁化禄者亦主有隐忧。唯太阳化权者较稳定，天梁化科者清贵，精神享受多于物质享受，而其物质享受亦由名誉而来。</li>
<li>然而成「阳梁昌禄」格者，见空劫同度，却往往为发明创造的因素，可藉此进财。</li>
<li>「阳梁」而太阳化忌，主财源不稳，亦主因受人拖累而破败。会巳宫太阴化忌，亦为破败之兆，亥宫太阴化忌则反主能突破环境以得财，如商誉突然建立，学术上有所突破之类。</li>
</ul>

辰、戌：太阳
<ul class = "ds-中州派分析段落">
<li>辰戌宫太阳与太阴相对，一般情况下，辰宫优于戌宫，唯太阴化禄者，却以戌宫为佳。</li>
<li>戌宫太阳，日月反背，唯见吉化吉曜者，反成奇格，主经艰苦而确立事业，财帛亦由是而来。少年时期艰苦者，中年后名利尤大。</li>
<li>辰戌太阳喜魁钺夹，主有人提拔。若火铃夹者，则主财易散难聚。</li>
</ul>

巳、亥：太阳
<ul class = "ds-中州派分析段落">
<li>巳亥宫太阳独坐对巨门，亦宜是非口舌以求财。可参考寅申宫「太阳巨门」同度的情形推断。</li>
<li>唯亥宫太阳落陷，极不宜见煞忌，否则仅能艰苦渡日。日生人尤甚。</li>
<li>巳亥宫太阳见禄马交驰，主远离出生地以求财。在巳宫者，见吉曜必能富裕；在亥宫者，见吉曜主受异性提携，唯仍难成富局。</li>
<li>巳亥太阳守财帛宫，必主多理想而欠财力实现，以致理想挫折。巳宫者较吉。</li>
<li>巳亥太阳，吉煞交集，则必须凭口舌以生财。若见科文诸曜，亦能富贵。</li>
<li>巳亥太阳喜巨门化禄拱照，则受异邦人赏识，由是提高社会地位，财源亦因之畅顺。</li>
</ul>

-----------------------------------------------------
武曲

<ul class = "ds-中州派分析段落">
<li>武曲为财星，主求财的行动，故必先有事业然后有钱财。一般而言，最利经商，但有时亦利于从事专业。唯必缺少父荫，须藉一己之力开创。</li>
<li>因武曲擅长于行动，所以亦长于以钱赚钱，一般很少会安于任职。唯「武曲天相」者有时例外。</li>
<li>武曲化禄，或与禄存同度，不见煞忌刑曜，必可成巨富。克应之期，多在财星叠禄的大运。唯原局若同时见文曲化忌者，则多损失。</li>
<li>武曲为金属之星，故亦宜从事与金属何关的职业，如手术、妇产科、牙科、机械工程等。古代的通货亦为金属，如银两、铜钱之类，故当武曲化禄、权、科时，亦为从事财政经济、金融企业的表征。若又见煞，则倾向于工程技术等专业。</li>
<li>武曲化忌，主周转困难、失业、破产之类，视同会煞曜凶星多寡而定。最不宜见地劫、地空、天刑、大耗、天虚等曜。</li>
<li>武曲躔寡宿，男命，主财权落于妻手。</li>
<li>武曲躔孤辰，主白手兴家。</li>
<li>武曲与龙池凤阁同度，主以金属工艺兴家；更见天月，则为利用金属器械的医疗人才。若见天刑，亦可为军警保安人员，唯必须无煞曜同宫始合，若有，则为持利器之下等行业，如屠宰、肉贩、理发之类。</li>
<li>武曲躔擎羊，为刑法，然亦主竞争；躔陀罗为阻滞。最不宜兼躔火星。古人有「武曲火铃同宫，因财被劫」，「武曲羊陀兼火宿，丧命因财」的说法。</li>
<li>「铃昌陀武，限至投河」，在财帛宫主重大倾败。所谓「铃昌陀武」，即武曲与文昌、铃星、陀罗同度或相对。若原局有「铃昌陀武」的结构，但非财帛宫，则当大运或流年财帛宫值此星系时，有流煞冲起，亦有破败的倾向。</li>
<li>武曲见贪狼则不畏火铃，否则必因财而生事端。</li>
<li>「武贪」主挥霍。「武杀」亦主挥霍，唯「武贪」之挥霍乃志在应酬；「武杀」之挥霍，常常一文不落虚空地。 二者又与「武相」之疏财仗义截然不同。</li>
</ul>


子、午：武曲、天府
<ul class = "ds-中州派分析段落">
<li>子午宫「武曲天府」，为财星喜得财库，故必须见禄然后始成格局。武曲化禄最佳，因同时必叠禄存，可成巨富。武曲化权亦佳，因事业宫亦必见禄存，富亦不小。武曲化科，则事业宫廉贞叠禄，亦成豪富。</li>
<li>若武曲化忌，则必同时府库空露，则主一生困乏，甚或寅吃卯粮。此时天府同时化科，仅主在财帛上有信誉。</li>
<li>武曲化禄、权、科而见空劫，则一生必有一段时期感到困乏。以大运财帛宫化忌之年，或见煞耗空劫之年为克应。</li>
</ul>


丑、未：武曲、贪狼
<ul class = "ds-中州派分析段落">
<li>丑未宫「武曲贪狼」同度，有火星或铃星同躔，主暴富。若同时再见煞忌，则暴起暴跌。</li>
<li>丑未「武贪」，一般情况下主得偏财，有煞而不重者，更倾向于偏门行业。例如火星陀罗同度；擎羊铃星同度之类。唯亦同时必主争夺，而且财帛聚散如浮云。</li>
<li>「武贪」为火铃夹，亦主横发。若煞忌同躔，则横发横破。</li>
<li>「武曲贪狼」见吉曜，尤其是文曲文昌，为社交活跃之征兆，其财亦由交际应酬而来。若见煞忌空劫刑耗，则虽终日应酬，亦劳而无功。即所谓「煲无米粥」。</li>
<li>「武曲贪狼」化禄化权，则不宜同时有文曲化忌同躔，如有，则进财虽多，但散财亦不少。唯当事人却常抱「小财不出，大财不入」的态度，不以为不利。</li>
<li>「武贪」、「武杀」主挥霍，「武贪」之挥霍乃志在应酬。</li>
</ul>


寅、申：武曲、天相
<ul class = "ds-中州派分析段落">
<li>寅申宫「武曲天相」同度，不擅理财，或仗義疏財。</li>
<li>「武曲天相」以武曲化科为最佳，盖必同时会廉贞化禄与破军化权，若无煞忌空劫，可成富局，唯难成巨富。</li>
<li>若有巨门化禄、太阳化权相夹，则主发财于远方，或主受外国人提拔。</li>
<li>若刑忌夹、羊陀夹、火铃夹者，一生财帛损耗甚大，主财帛为兄弟或友人所夺。若与人合作，切须慎防。</li>
<li>「武曲天相」亦主凭特殊技能以求财。见科文诸曜，则倾向于文学艺术。</li>
<li>「武相」喜禄马交驰，主离乡背井以建立事业，因此成富。</li>
<li>「武相」与辅佐诸曜同度，仗义疏财。但若昌曲化忌，则每因此反受拖累，或致受骗。</li>
<li>「武相」见煞，进财艰辛。躔擎羊，主争财；躔陀罗，进财拖延，二者均须改变行业。躔火铃，财难积聚；躔空劫，内在空耗。</li>
</ul>


卯、酉：武曲、七杀
<ul class = "ds-中州派分析段落">
<li>卯酉宫「武曲七杀」，主白手兴家。见吉化者，必白手以成大业。</li>
<li>「武杀」亦为横发之星系。唯必须见化禄或禄存始是，且须无煞曜同躔。若有煞，则主争财，不主横发。或一发即蹶，财帛慢慢消磨。</li>
<li>「武杀」见吉曜，投资额巨大，其资财往往在企业资产上。不主积蓄。若同时见煞，则事业虽大，但财帛反而有损。</li>
<li>「武杀」若见煞忌刑耗并集，则主倾败、倒闭或破产。以武曲化忌冲会廉贞化忌最为恶劣。</li>
<li>「武贪」、「武杀」主挥霍，「武杀」之挥霍，常常一文不落虚空地。 </li>
</ul>


辰、戌：武曲
<ul class = "ds-中州派分析段落">
<li>辰戌宫武曲独坐，与贪狼相对。必须经历艰辛而后成富。</li>
<li>武曲对「火贪」、「铃贪」，亦主突发。若有吉曜吉化而无煞忌，则不主横破。</li>
<li>辰戌武曲得禄，可成富局。以禄存在命宫，天马在命宫或迁移宫者最为上格。唯天马不宜与空劫火铃同度，否则破格。此乃因辰戌二宫为罗网，必须藉命宫禄马之力以脱罗网羁缠。</li>
<li>辰戌武曲化忌，必生破败。见煞曜刑耗，亦主困乏。</li>
</ul>


巳、亥：武曲、破军
<ul class = "ds-中州派分析段落">
<li>巳亥宫「武曲破军」，性质不良，因一生起伏波澜甚大。</li>
<li>「武破」得禄权科会者，一生人亦至少有一次困境，但克服困难之后，仍能致富。</li>
<li>若武曲化忌，则不宜经商，以任职为宜。更不宜投机，否则终生陷入困境，极难恢复元气。</li>
<li>「武破」见贪狼化忌，主无心插柳柳成荫，于事业外别开生路，由是富足。盖此时必同时见破军化禄，主财源有意外改变。</li>
<li>「武破」而破军化禄、化权者，一生事业必屡有变化，大致上愈变愈奇。</li>
<li>「武破」守财帛宫，勇于投资，而俭与应酬。所以平常并不大手笔，只于创业时往往孤注一掷。</li>
<li>「武破」见禄马，必须离乡背井以成富局。命宫有火铃，则立业异乡，祖业不守。</li>
</ul>

-----------------------------------------------------
天同

<ul class = "ds-中州派分析段落">
<li>天同守财帛宫，主白手兴家。</li>
<li>火铃同度者，有祖业亦必散尽。然后始主自身创兴。若空劫同度，则自身开创亦有破败，仅能东山复起，结局富裕。</li>
<li>天同在财帛宫不喜独坐，若独坐，必须中年辛苦求财，至晚年始得顺遂。亦即白手兴家的程度更大，情形亦较艰辛。</li>
<li>天同化禄，决主晚年可成富局。但若同时见煞，则发迹之前，必须财帛破败净尽，至身无一文，然后始有意外收获。若无煞者，则破败程度较轻。</li>
<li>天同化权，一般情形较为稳定，虽亦主白手兴家，但「白手」的程度不致身无分文。</li>
<li>天同与陀罗同度，往往五十岁后始环境变佳。能否成富，须视晚运优劣而定。</li>
<li>天同化忌，主一生财运反复。更见煞刑虚耗诸曜，则财源稍顺又告更迁，必须时时另起炉灶。</li>
<li>天同喜见禄马交驰，则为离乡背井，白手以成富局。更叠化禄，尤主发财万镒。</li>
<li>凡天同在财帛宫，所业宜带享受性质。或为消遣享受时所应用的物品。倘见科文诸曜，尤主优雅享受；见桃花，则宜以异性为顾客对象。</li>
</ul>


子、午：天同、太阴
<ul class = "ds-中州派分析段落">
<li>子午宫「天同太阴」同度，为天同坐财帛宫众星系中，最利财源的星系。天同化禄，则破尽祖业后突然兴起；太阴化禄，则一生至少有一次意外财源。</li>
<li>「天同太阴」在子宫者，女命，往往主与异族通婚而成富。男命则主得外族财。唯太阴化禄者不是。</li>
<li>「天同太阴」与禄存同度，事业宫见天马，亦主发财于远域。</li>
<li>「天同太阴」若太阴化权，则少年必主飘流，在异域先有地位，再成富裕。</li>
<li>「天同太阴」若天同化忌，或太阴化忌，财帛皆难积聚。又见煞刑诸曜者，且见虚耗，主财源波动。</li>
<li>「天同太阴」与擎羊同度者，因财起纠纷。或主财源起竞争。</li>
</ul>


丑、未：天同、巨门
<ul class = "ds-中州派分析段落">
<li>丑未宫「天同巨门」，基本性质主因财而生是非口舌，故最宜费唇舌以求财。巨门化忌者，尤非费唇舌以求财不可。若见羊陀，亦主劳心。</li>
<li>「天同巨门」与阴煞、天虚、大耗同度，则主机关巧计。见天刑，宜从事法律，或与纠纷有关的事业（如执达吏、公正行）。</li>
<li>「天同巨门」与昌曲化忌同躔，更见桃花及刑耗，主因色破败。其中隐痛且不足为外人道。</li>
<li>「天同巨门」喜左右夹、昌曲夹，则积财能力稍强。若火铃夹者，则主财帛不积。倘火铃夹巨门化忌，又见刑耗，则为因官非词讼而破财。</li>
<li>「天同巨门」见吉化、吉曜，亦宜传播工作，或专业工作以求财。</li>
</ul>


寅、申：天同、天梁
<ul class = "ds-中州派分析段落">
<li>「天同天梁」在寅申宫，见吉曜，主财源丰茂，而当局者却并不斤斤计较金钱。</li>
<li>若天同化禄，则主有意外财，但非横财。但天梁化禄，则财源往往受人指责。</li>
<li>天梁化禄与火铃同度，更见天姚、阴煞，则主贪墨。</li>
<li>「天同天梁」躔禄马，或对面福德富有禄马，亦可因远方财而成富，但不主商业贸迁。见天巫者，主远方有现成事业，因承继或接受而成富局。</li>
<li>「天同天梁」与天才同度，或与龙池凤阁同度，主凭技艺生财。</li>
<li>「天同天梁」与昌曲、桃花同度，见天马而不见禄，主财由异性来。</li>
</ul>


卯、酉：天同
<ul class = "ds-中州派分析段落">
<li>卯酉宫天同独坐，与太阴对拱，决主白手兴家。天同化禄者，凭薄资以成富局。</li>
<li>卯酉天同与太阴化禄对拱，因事业宫巨门必同时化忌，故主费口舌以求财。</li>
<li>卯酉天同化忌，倘更有羊陀夹，主因财致祸。福德宫见桃花者为色祸。无桃花而昌曲化忌，主替人负债。</li>
</ul>


辰、戌：天同
<ul class = "ds-中州派分析段落">
<li>辰戌宫天同独坐，与巨门相对，基本性质主求财辛苦，颇费周章。唯魁钺夹者，则一生多机缘。</li>
<li>辰戌天同有煞同度，谋为必费心力。</li>
<li>若天同与天姚、阴煞、天虚同度，或铃星同度，皆主求财费尽机关。</li>
<li>辰戌天同化忌，火铃夹者，主受人拖累而破财。更见刑耗、空劫，主官非词讼。</li>
<li>辰戌天同即使化禄，亦只宜服务他人。自行经营，财更难聚。</li>
</ul>


巳、亥：天同
<ul class = "ds-中州派分析段落">
<li>巳亥宫天同独坐，与天梁相对，不主财旺。唯命宫「天机巨门」叠禄者为富局。此乃命宫使然，不关财帛宫星曜。</li>
<li>巳亥宫天同，对拱天梁化科，则名大于利。唯若文曲化忌同度，则为伪君子，名虽高而求财手段不正。</li>
<li>巳亥宫天同化忌，天刑同度，因财而招讼事，且易引起损失。</li>
<li>巳亥天同化权，因命宫必同时见巨门化忌，亦主劳心费力，求财多起是非。</li>
<li>巳亥天同于财帛为弱宫，稍见煞即主劳心费力。不见煞，见诸吉，然后晚年始能富裕。</li>
</ul>

-------------------------------------------------------------
廉贞

<ul class = "ds-中州派分析段落">
<li>廉贞征财帛宫，性质相当于复杂。尤其对四化异常敏感，变化甚大。</li>
<li>廉贞主竞争是非以求财，此为其基本性质。其化忌，见煞，且有刑耗诸曜，则不但求财艰难，且易生词讼。见昌曲化忌，且易为人侵吞财物，或替人背债。</li>
<li>但廉贞化禄，或禄存同度，却易成富格。其财每由交际应酬而来。「小钱不出，大钱不入」，即为此种格局的克应。若廉贞化禄而有煞同度，又见吉曜，其人用钱恰能用在刀口。</li>
<li>廉贞化禄见魁钺、昌曲、辅弼，则财源丰茂。有火星擎羊同度，主暴发。唯财不易聚。有铃星陀罗，则艰辛以求财，或多虚少实。遇空劫则多破败，且常常牵涉是非争讼。</li>
<li>廉贞在财帛宫，吉凶诸曜交集者，其人多浮华。昌曲同度，亦主华而不实。一生追逐财富，毕竟但得表面风光。若同时化禄，则主豪奢。</li>
<li>廉贞化忌，主因财失义，亦主受人拖累。</li>
<li>廉贞化忌而空劫同度者，上述情形更甚，不但易涉讼事，而且每每因此倾破家财。</li>
<li>廉贞与煞曜同度，有空劫、大耗者，主遇盗。</li>
<li>廉贞与昌曲化忌同度，见大耗、阴煞，亦主被盗，或主受骗。</li>
</ul>


子、午：廉贞、天相
<ul class = "ds-中州派分析段落">
<li>子午「廉贞天相」，加强廉贞的交际应酬色彩，为长袖善舞的商人。若成财荫夹印之局，则每每因此受到回报。故此种格局，容易经商而成富裕。</li>
<li>「廉贞天相」而刑忌夹印，或廉贞化忌，则主因无谓应酬而致破财。有煞曜空劫者，则主是非词讼。</li>
<li>最喜廉贞化禄，则于交际中得财，可以成富局。</li>
<li>「廉贞天相」与空劫同度，则难以积财。更见刑煞，有是非词讼，或虽表面风光，其实内在空虚，寅吃卯粮。</li>
<li>「廉贞天相」亦可横发横破，但性质较为和缓。</li>
</ul>


丑、未：廉贞、七杀
<ul class = "ds-中州派分析段落">
<li>丑未宫「廉贞七杀」同度，在未宫为「雄宿乾元格」，若得禄则可成富格，唯仍须历艰辛。丑宫虽不成「雄宿乾元格」，但见禄亦可成富，唯财源不如在未宫之畅顺耳。</li>
<li>「廉贞七杀」在财帛宫，为浮华性质至少之星系。故除廉贞化禄外，见破军化禄或贪狼化禄，均可成富，而且主充实而不空虚。</li>
<li>「廉贞七杀」有昌曲同度，则主同情心强，化忌则不宜代人负债。</li>
<li>「廉杀」在财帛宫，擅积财而不擅理财，辅佐诸曜同会者，则理财能力较佳。</li>
</ul>


寅、申：廉贞
<ul class = "ds-中州派分析段落">
<li>寅申宫廉贞独坐，与贪狼相对，理财能力甚强。与火星同度者，亦主横发横破，且对钱财相当重视。</li>
<li>廉贞坐寅申，最宜从事财经事业。若廉贞化禄，可成富局，贪狼化禄，尤存利己之心。火贪化禄，主突发，只需见好收篷，亦不致暴败。</li>
<li>寅申宫廉贞，最喜禄马交驰，不但善于理财，且主贸迁得利。</li>
<li>廉贞于寅申宫化忌，事业受挫折，往往功败垂成。若同时见煞刑耗曜，且主词讼。火铃夹者尤确。</li>
</ul>


卯、酉：廉贞、破军
<ul class = "ds-中州派分析段落">
<li>卯酉宫「廉贞破军」同度，财源不稳定。若破军化禄，往往须兼行兼职。</li>
<li>「廉贞破军」在财帛宫，破军化禄则不宜见火星，主因贪反生破败，或理想太高，屡生挫折。</li>
<li>唯「廉贞破军」与禄存同度者，则喜见火星，或火星与命宫「武曲贪狼」同度，此为巨富之格。但发后仍有破败之机，以流年命宫或财帛宫见煞忌刑凑之年为克应。</li>
<li>「廉破」与空劫同度，主横生破败。但若同时见禄及吉曜者，则吉凶系于一线，一念之差，可以成败立判。大致而言，不宜赶尽杀绝，最宜留有余地，见好收篷。</li>
<li>「廉破」守财帛宫不宜投机，否则必生破败。即使见吉，破败后亦难恢复元气。</li>
</ul>


辰、戌：廉贞、天府
<ul class = "ds-中州派分析段落">
<li>辰戌「廉贞天府」，擅积财，亦擅理财，但若府库空露，则纵腾挪变化，亦难以积存财帛。</li>
<li>若廉贞化禄，或命宫有禄存，或事业宫有禄存（叠禄者更佳），则为富格。</li>
<li>若天府化科，则在财帛方面一诺千金。若此时会武曲化忌，容易手头困乏。故不宜投机，亦不宜轻于言诺，以免受人拖累。</li>
</ul>


巳、亥：廉贞、贪狼
<ul class = "ds-中州派分析段落">
<li>巳亥宫「廉贞贪狼」同度，二者任一化禄，与火铃同度，均主得偏财，但暴得暴失，见刑耗空劫，且主因此破败。</li>
<li>巳亥「廉贪」与桃花同度，则为艺术之星。但倾向于表演。</li>
<li>巳亥「廉贪」为色彩，故主从事美观之事业以生财。又为娱乐及享受，亦宜从事此等行业。</li>
<li>巳亥「廉贪」与昌曲同度，则为浮华，亦主繁文缛节，故有装饰意味，此亦为生财之性质，如美术设计之类。唯进财亦有虚浮现象，见虚耗，主虚空不实。</li>
<li>「廉贪」而廉贞化忌，决生破败，不宜求偏财；贪狼化忌者，则主求财目标不易达到，但因事件进行，反而另外有生财之道。</li>
</ul>

----------------------------------------------------------------------------
天府

<ul class = "ds-中州派分析段落">
<li>天府守财帛宫，不主富厚，仅主其人有储财及理财能力。必须见禄，然后始可成富局，禄存或化禄均可。</li>
<li>天府守财帛宫，亦喜见辅佐诸曜，则增加其储财能力，亦增加其进财能力。</li>
<li>若不见禄，则宜天府化科。此主有信誉，特别于理财及投资方面，必有人予以信赖，由是开展财源与职业。</li>
<li>天府已主保守，用钱非常有计划，倘禄存同度，更主凡事小心谨慎。此则仅利于发展现成事业，不利开创新局。</li>
<li>天府禄存同度，更宜见辅佐诸曜齐会，然后始主有助力及发展机会。</li>
<li>天府不得禄，又不化科，且无辅佐诸曜，则为空库。有财星同度或相会（如武曲）尚可；如无，则仅主利于受薪服务。自行经营，不主发达，易错失良机。</li>
<li>天府与禄存同度而不见辅佐诸曜，则进财艰难，稍见煞即有破耗。</li>
<li>空库者若更与煞忌空劫同会，则为「府库空露」，主困乏，仅宜受薪，自行经营，则必引起争夺。</li>
<li>空库或露库者，与阴煞、天姚同度，则主机关巧计以谋财。若文昌化忌或文曲化忌同躔，则为巧骗或受骗，此须详命宫及福德宫而定其为骗人抑或受骗。</li>
<li>天府得禄见吉，若有空劫同度，每进财亦必受损失，唯进财之数大于损财耳。</li>
</ul>


子、午：天府、武曲 


丑、未：天府
<ul class = "ds-中州派分析段落">
<li>丑未宫天府与「廉贞七杀」相对，且成「日月夹财」之局，古人云，「日月夹财，不权则富，加羊陀冲守则宜僧道。」此局必须见禄、权、科及吉曜，然后始入格。</li>
<li>丑未宫天府亦宜昌曲、左右同度，则亦主掌财权。</li>
<li>丑未宫天府若有昌曲同躔，又同会桃花诸曜，则最忌天姚同度或对拱，则主掌妇人之财，性质优良者，可以为银会会首之类。若性质不佳，则主生事端。</li>
<li>丑未天府会刑忌夹印的天相，亦主在金钱方面受压力，会财荫夹印的天相，则必得荫庇之力，或有管理现成事业的机会。</li>
<li>天府对冲廉贞化忌，则多挫折。有煞同度者更主破败。</li>
</ul>


寅、申：天府、紫微


卯、酉：天府 
<ul class = "ds-中州派分析段落">
<li>卯酉宫天府独坐，与「武曲七杀」相对，以武曲为财星，故掌财权之力，大于丑未宫天府。是故最宜见辅佐诸曜同会，主有现成事业，或经营家族事业。</li>
<li>若见空劫刑耗，且武曲化忌，则主事业破败，或从事新项目投资失败，以致影响原有事业根基动摇。</li>
<li>最喜见武曲化禄，或天相为财荫夹，则经营事业有所发展，利创新猷。</li>
<li>若武曲化科，则理财能力甚佳，所渭「以钱生钱」，易得人信任，最宜积集资金。</li>
</ul>


辰、戌：天府、廉贞


巳、亥：天府
<ul class = "ds-中州派分析段落">
<li>巳亥天府为「紫微七杀」相对，最喜紫微化权，天府同时化科，则一生职掌财权。在现代，宜经营基金、财务机构等。</li>
<li>巳亥天府最宜见禄马交驰，同时见辅佐吉曜，则可掌权于国际性财务机构。</li>
<li>唯若成空库，则在巳亥宫之天府，一生财源波动甚大；虽小心谨慎从事，事机依然有失。</li>
<li>巳亥宫天府最不宜「露库」，否则稍有发展，即为人觊觎。倘见空劫，更主突生破败。</li>
<li>巳亥天府最为惜财，非见现金则无安全感，故若见煞，则影响发展机会。</li>
<li>巳亥天府亦最不喜会刑忌夹的天相，主受财政压力。</li>
</ul>

------------------------------------------------------
太阴

<ul class = "ds-中州派分析段落">
<li>太阴为财星，但太阴主静，主藏，故在财帛上表征为储蓄、积聚，以及思考、享受。此跟天府不同，因为天府仅有守财的意义，不表征为享受及思考。这有如一个银行，银仓可能积存许多钞票，但银行职员本身并不会将存款作为自己的精神享受。太阴则等如基金经理，对因自己计划得出来的成果，自然有满足的感觉。</li>
<li>太阴又与武曲不同。仍以财经机构为例，太阴等于决策投资计划的经理，而武曲则属执行计划的人员。于决策时，固然要力求冷静，在执行计划时，尤其需要当机立断。所以武曲主决断，而太阴则主计划。</li>
<li>太阴之主计划，亦即擅长运用钱财。而且在精神上享受运用钱财的成果。其所得，并不全部属于自己，但仍然可因此而成富裕。</li>
<li>太阴化禄，或与禄存同度，皆主富裕。化权则增加其计划理财，及享受财富的性质。化科则仅为理财的知名度。</li>
<li>太阴入庙，夜生人决主富贵，其贵（即社会地位）亦由富而来；日生人次之。太阴落陷，则进财维艰，夜生人较吉，日生人更甚。</li>
<li>太阴亦喜同会辅佐诸曜，以其亦为夜生人之主星也。</li>
<li>太阳主贵，先有社会地位然后有钱财；太阴主富，先有钱财或在财经界上有成就，然后始有社会地位。</li>
<li>太阴化忌，一般情形下主受引诱而致投资失败。所谓受诱，指受人不正确的引导，亦指受潮流引诱。若更遇文曲化忌冲会，则此种引诱且带受欺骗性质。</li>
<li>若有煞曜，主成败不一，且引致纠纷；见空劫，则为盗贼之灾。太阴化禄冲会文曲化忌，则主被窃或遗失。</li>
</ul>


子、午：太阴、天同


丑、未：太阴、太阳


寅、申：太阴、天机


卯、酉：太阴
<ul class = "ds-中州派分析段落">
<li>卯酉宫太阴独坐，与天同相对，在酉宫入庙，故优于在卯宫落陷，唯二者皆主白手兴家。大致上可参考子午宫「天同太阴」同度时的情形推断。</li>
<li>卯酉太阴，在求财时常过份追求理想，因而亦时有失落感。</li>
<li>卯酉太阴会照太阳及天梁，故当同会昌曲、天才、龙池凤阁，或且见化科之时，亦主其财源带有学术意味，一般表征为财经研究。</li>
<li>卯酉太阴会太阳化科，尤主学术研究。或代理名牌商品，或在有名机构服务。</li>
</ul>


辰、戌：太阴
<ul class = "ds-中州派分析段落">
<li>辰戌太阴与太阳相对。一般情形下，戌宫优于辰宫。唯辰宫「日月反背」，若禄权科会，见吉曜，亦主成大富之格。</li>
<li>辰戌太阴最喜太阴化禄，主富而后贵；太阳化禄则先贵而后富，故一般宜以先任职企业然后自创；会天同化禄亦主白手兴家；会天机化禄，则仅主过手钱财数量大。</li>
<li>若天同化忌，则劳心劳力以求财；天机化忌，则奔波劳碌；太阳化忌，则得财亦同时有是非。太阳化禄、化科，均主有产业承继。</li>
</ul>


巳、亥：太阴
<ul class = "ds-中州派分析段落">
<li>巳亥宫太阴与天机相对，亥宫优于巳宫。在亥宫者每因机遇而成巨富；在巳宫者，辛霜酸露，四处奔流。</li>
<li>亥宫太阴守财帛宫，却主先有地位名誉然后有财富，故绝非经商致富之命。即使太阴得禄亦然。若见科文诸曜，尤宜学术研究。</li>
<li>在巳宫者，虽有相同性质，但名誉地位减等，而进财亦较少，或较艰辛。</li>
<li>由于天机的影响，故主变动。性质不吉时，主财帛不聚，或主所任职务长期带流动色彩。若性质优良，则主所涉及者有大量金钱流通。例如从事社会工作，所牵涉者为大量救济金；从事银行业务，所牵涉者为大量存贷；从事学术研究，由专利导致大额投资，或成为公众的日常必须品。</li>
<li>巳亥宫太阴不宜与空劫同度，否则常主枉抛心力，或时生挫折，功败垂成。</li>
<li>太阴在十二宫，唯居巳亥守财帛宫主贵不主富。</li>
</ul>

贪狼

<ul class = "ds-中州派分析段落">
<li>贪狼守财帛宫，手段阔绰大方，但仍精于盘算计较，所以对于财帛的态度，其内心与外表常不一致。</li>
<li>贪狼守财帛，主擅长运用金钱，但却不擅长守财，且常常有以攻为守的倾向，因此往往倾向于投机。</li>
<li>贪狼与羊陀、空劫、大耗会同，则以不投机为宜，否则倾家。即使贪狼化禄，亦主波折挫败。不从事投机，则仅主事业上有极大竞争，或计划常因客观环境影响而生变化，虽受一时损失，容易东山复起。</li>
<li>贪狼化权，反而增加投机性及多生变动；贪狼化禄，则无论财政状况如何，均表现阔绰，追求豪华风光的享受。唯贪狼化忌，反容易自我克制。</li>
<li>火星或铃星与贪狼同度，更见禄存或化禄，决主突发或暴发。若同时有空劫者，一发即生破败。无空劫，会吉曜，然后始能保泰，且必须及时收手，不可以为永远幸运。</li>
<li>贪狼叠禄，常得意外之财。但若同时与煞刑虚耗诸曜同会，则亦带有灾难性。如得伤残保险赔偿，亲属人寿保险赔偿，或有后遗症之赌博得利，均主得财而同时惹来灾祸。</li>
<li>贪狼躔昌曲桃花，因色损财，更见煞刑诸曜，因色倾家，且惹官非词讼。</li>
<li>贪狼与阴煞同度，主受骗；与天月同度，主因病破财；与劫煞同度，主田财起争夺。此均以同时见羊陀空劫者为然。</li>
<li>贪狼化忌，与火铃同度，更见天虚，又会禄存，则其人宜以偏门起家。</li>
<li>贪狼守财帛，会廉贞带禄存，又见破军，有火铃会照，亦宜财不由正路来。</li>
</ul>


子、午：贪狼
<ul class = "ds-中州派分析段落">
<li>子午贪狼守财帛，与紫微相对，得紫微化权拱照，则财源稳定，而可得多方面的财源。紫微化科拱照，则终生先名后利，亦主以交际应酬，多结人缘，由是营运得利。</li>
<li>子宫贪狼成「泛水桃花」之局者，吉则宜经营以异性为对象之行业；凶则主因色破败。吉凶交集而同时有火铃同度，财由异性而来，亦因此生破败。</li>
<li>子午贪狼若不见紫微吉化，纵使贪狼化禄、化权，亦主动荡变迁，或需时时出差，或多到外埠洽谈商务。见事业宫破军与禄马同躔者，尤主外务得意（不一定迁移外地）。</li>
<li>子午贪狼守财帛者，以掌握财权为满足，不一定要自己拥有大量资财。故亦宜任职大企业营业部门。但不宜作计划工作。</li>
</ul>


丑、未：贪狼、武曲


寅、申：贪狼
<ul class = "ds-中州派分析段落">
<li>寅申宫贪狼独坐，与廉贞相对，一般来说，廉贞化忌，则宜半偏行业，或「凶事以求财」，以带血光为佳。在现代，经营冻肉亦属此类，亦可为外科、牙科。</li>
<li>倘廉贞化禄对拱，与火铃同度，亦主横发，有煞曜会合，更见阴煞天姚，则主半偏行业得财。必须见吉，然后主艺术或带装饰享受性质的行业起家。若吉煞交集，则进财同时亦有损耗，且主横发横破。</li>
<li>寅宫贪狼化忌，主时时改变行业。不化忌，但成「风流采杖」之局者，则主投机致祸，或经营半偏及偏门行业而致身败名裂。</li>
</ul>


卯、酉：贪狼、紫微


辰、戌：贪狼
<ul class = "ds-中州派分析段落">
<li>辰戌宫贪狼独坐，与武曲相对。火铃宜与贪狼同度，若与武曲同度者，则虽横发亦必同时起竞争。</li>
<li>喜武曲财星化禄，则运用财帛的能力甚强；化权、化科次之。</li>
<li>辰戌宫「火贪」、「铃贪」以得贪狼化禄为贵；主突然财运亨通，且略带投机色彩，但必须见好收篷，得意不宜再往。若贪狼不化禄者，突发程度较逊。</li>
<li>辰戌贪狼落罗网，若命宫或迁移宫有禄马交驰，则主离乡背井以成富局。唯一生事业财源必多变动。</li>
</ul>

巨门

<ul class = "ds-中州派分析段落">
<li>巨门守财帛宫，基本性质为劳心费神以求财，或主以口舌求财。且不宜心高气傲，否则必生破败。</li>
<li>巨门化禄，亦仅主利于口舌求财，如律师、教师、推销、传播、表演等行业。巨门化忌者尤然，否则必多是非词讼。</li>
<li>巨门坐财帛宫，有禄、权、科同会，更见辅佐吉曜，必可成富。唯若锋芒毕露，盛气凌人者，必反遭人排挤，以致失败。</li>
<li>巨门之财，不过一代。故凡巨门坐财帛宫，宜兼视子女宫，子女宫不吉者，主子女散财，更见刑耗煞曜，及阴煞、大耗、天虚、破碎、蜚廉诸曜，则子女因牵涉官非而破败，或骄奢致败。</li>
<li>巨门化忌固易惹是非，太阳化忌亦主生极大口舌，且见天刑，则主因财而致兴讼。唯从事刑法纪律行业者，则反吉祥。</li>
<li>巨门与火、铃星、空劫、大耗同度，则主因火灾、兵灾而致破耗。</li>
<li>巨门、擎羊、天刑同度，主词讼，或主财帛竞争。从事有竞争意味的行业，如法律拍卖行、经纪行等，则主吉祥。</li>
<li>巨门与太阳同度或对照，有吉化者，或命宫见禄马交驰者，主一生受异族信赖而成富。且主其人有语言天才，此种技能，亦可帮助其进财。</li>
</ul>


子、午：巨门
<ul class = "ds-中州派分析段落">
<li>子午宫巨门与天机相对，见禄，为「石中隐玉格」，主中年后脱颖而出，突成巨富。唯早年必受艰辛。发达之后，若锋芒太露者，又易成为他人攻击目标。若骄奢淫佚，亦往往因此致败，且易一蹶不振。</li>
<li>子午巨门居财帛宫，亦宜在异族经营之机构服务，则主先得信誉，然后事业发展而富裕。</li>
<li>若见科文诸曜，则以传播事业为宜。亦先有名然后有利。</li>
<li>倘子午巨门与桃花诸曜同度，则宜从事以口才为主之表演事业。化禄化忌皆然。</li>
<li>子午巨门与煞曜同度者，事业每多纠纷。与桃花诸曜同度，又见阴煞、天姚，则因财而招色祸。从事演艺者尤须慎防。</li>
<li>子午巨门不见吉化，但有禄存同度，又见火铃，且躔刑耗者，主争财兴讼。有天巫，则主遗产争夺，或主现成事业起竞争。</li>
</ul>

巳、亥：巨门

<li>巳亥巨门见太阳化禄，亦主受异族推崇，在异族中有信誉。</li>
<li>若巨门化禄，亦宜门市、零售等业。以异族为主要顾客对象者更宜。亦宜以外地人士或游客为顾客对象。</li>
<li>巨门化忌及会天机化忌之时，则以任职为佳，不宜自行经营。</li>
<li>巨门化为权星，在巳亥宫，必须从事「口舌生财」的行业。</li>

天相

<ul class = "ds-中州派分析段落">
<li>天相坐财帛宫，有些情形与天府相似。天府为财库。但天相却亦可为财印。以现代意义而言，天相等于提款的印鉴，若印鉴不符，或失去印鉴，则虽府库充盈亦无所施用。所以当天相坐财帛宫时，必须辨其是否属于财印。</li>
<li>成财印的天相，如得财荫夹印，与禄存、化禄同度或对拱者皆是。此主富厚发达。若不见禄，但见吉曜会聚，而天府虽不得禄但亦不空露者，天相虽非财印，但亦不致成为破印，亦可得富足。</li>
<li>破印者，为刑忌夹、火铃夹、空劫夹、有煞同度又会煞，或空劫同度会煞者，皆是。凡破印，主财帛波动，煞忌重重者，或且致倾家。唯无论何种程度之不吉，大多数情况下均主受人牵累而致。</li>
<li>天相与禄存同度者，其对钱财之谨慎态度甚于天府。唯与天府不同之处，则在于天相守财帛时，常能慷慨解囊以济人之危，天府则对此更小心谨慎。</li>
<li>天相喜与化禄或禄存同度，必主富厚。尤喜化禄，财帛更主丰盈。且亦可掌财赋之权，能在现成局面下发展事业。</li>
<li>天相为财荫夹，更易得现成机会，发展现成事业。故亦喜同会魁钺，或得魁钺夹。</li>
<li>天相若为刑忌夹，则主受掣肘，或受拖累。更加有煞同宫，或空劫同度，则主倾覆破败，再见天刑、大耗、阴煞、会化忌，可主牢狱之灾。</li>
<li>天相最不宜与陀罗同度，否则主得财同时招忌，更遇火铃冲破（尤忌铃星），则主破尽祖业，四十以后始渐兴家。</li>
</ul>

<li>丑未天相之突发，仍视会合星曜而定其程度。见吉化吉曜则突发之后可平稳，若有煞忌，则仍主波动。</li>
<li>丑未天相，常以「紫破」或天相大运或流年，为突发克应之期。</li>

寅、申：天相、武曲


卯、酉：天相
<ul class = "ds-中州派分析段落">
<li>卯酉宫天相独坐，与「廉贞破军」相对，基本性质不吉。故稍见煞曜，其财源即不稳定，且时生改变之心。</li>
<li>若与空劫同度，再见煞忌（尤不宜廉贞化忌），则易因拖累而致生纠纷。若再有天刑、大耗、阴煞，或更与昌曲化忌对冲，于此大限流年，必生严重破败，可至倾家。唯若继续行至吉限、吉年，则主可以东山复起。此为一项重要征验。</li>
<li>卯酉天相见禄，又见吉化吉曜，财帛虽丰盈，但仍主改变，故一生必至少有一次重大的转行转业，转变财源。</li>
<li>卯酉天相若见吉曜太多，则容易过分奢望，生改变之心亦由此而来。盖以为改变环境即可达到理想，然结果必仍然失望。后天趋避，以不作急剧转变为宜。</li>
</ul>


辰、戌：天相、紫微

巳、亥：天相
<ul class = "ds-中州派分析段落">
<li>巳亥宫天相，对「武曲破军」，财源亦不主稳定，唯较对拱「廉贞破军」为佳。天相独坐的三种情况，其实皆主不稳定，唯对拱「紫微破军」之不稳定，系主突发，以及突发后仍有波动；「廉贞破军」之不稳定，常常出于自发；「武曲破军」之不稳定，则为客观环境所致。人类常易克服客观环境的改变。但却不易克服内心的情绪，因此天相对「武破」，其客观境遇一般便比对拱「廉破」为佳。</li>
<li>巳亥天相最畏武曲化忌冲拱，则一生波折重重，当境况较佳时，又生变化。惟若仍有吉曜会合，而煞不重者，则可从事「凶事以求财」的行业。如饮食业、冻肉业等，则波动较少。或从事专业亦可，则以受雇用为宜。</li>
<li>巳亥天相见武曲化禄，必躔陀罗，或与陀罗冲拱，故性质亦不全美。</li>
<li>巳亥天相见武曲化权、化科，又有吉曜同拱，则财源较为稳定，且主终有积蓄。</li>
<li>巳亥天相躔空劫刑煞，则有破败。客观环境改变频仍。</li>
<li>巳亥天相见禄马交驰，主常得外地提供之生财机会。</li>
</ul>

天梁

<ul class = "ds-中州派分析段落">
<li>天梁守财帛宫一般不宜经商。（反而天梁居迁移宫者可成富商巨贾）。从事商业活动，亦以任职为宜，否则必生重大困难，有时且可能引起危机。</li>
<li>天梁不宜化禄，若化禄，反主有纠纷、困扰。有煞忌同会者，往往因此引致官司词讼。若同时会吉曜，则困扰较小，亦较易化解。</li>
<li>天梁有管理、监察、查核的性质。居财帛宫，亦以此类性质为宜。若天月同度，则变为查察病情，故可以医药业求财；若天刑同度，则刑法纪律的监管，故可以司法、军警等业为求财的手段。</li>
<li>天梁为荫星，故主有父祖之荫庇。有时并不主遗产的承继，而是藉长辈的声誉而得助力，由是生财。有时又主因长辈的人际关系而得事业，由是生财。或主有现成事业生财。</li>
<li>天梁守财帛宫，亦先名后利。故天梁化科，优于天梁化禄。</li>
<li>天梁守财帛宫，宜以服务为目的，不宜以求财为目的。志在服务者，财禄自然充裕，若志在求财，则易渎职，反滋困扰是非。</li>
<li>天梁在财帛宫化禄、叠禄，皆只主一时之富，但背后却受人讥评。交入不佳岁运，立时惹起是非困扰。故宜广结人缘，以不损害他人利益为求财之大前提，且须顾及公正，然后才能后天趋避。然究不如志在服务者之佳。</li>
<li>天梁会禄权科，而非天梁化禄者，名高利厚。</li>
<li>天梁天马同度于财帛宫，财帛难聚。且主时时调换工作环境。</li>
<li>天梁守财帛，纵见凶曜，亦终不致贫乏。</li>
</ul>


子、午：天梁
<ul class = "ds-中州派分析段落">
<li>子午宫天梁，子优于午，盖子宫天梁得午宫太阳正照，招是非妒忌的性质大为减少，尤以化禄者为然。</li>
<li>子宫天梁，一生多进财钱会，但散财亦不少。必须天梁本身不化禄，而会照化禄或禄存，然后始成富局。</li>
<li>午宫天梁，进财机会较少，散财机会亦较少。唯一旦富足，或享受较奢即易受人妒忌。</li>
<li>子午宫天梁，亦可成「阳梁昌禄」之局，若更见科文诸曜，尤喜文昌化科，或文曲化科，则名高利厚。在现代，亦可以专利发明致富。</li>
<li>若昌曲桃花同会，则为艺术、传播。宜以此为业生财。</li>
<li>经商者，则多为承继现在事业，应以商誉为重。若建立商誉后即妄自尊大，即立生破败，或引起重大风波。</li>
</ul>


丑、未：天梁
<ul class = "ds-中州派分析段落">
<li>丑未宫天梁独坐，与天机相对，财帛动荡。唯左辅右弼夹者，始主能积存。</li>
<li>丑未天梁，有昌曲夹，一生名大于利。故求利不如求名。</li>
<li>丑未天梁，以丑宫优于未宫。此与子午宫同意。</li>
<li>丑未天梁会太阴化禄，主有现成事业或遗产承继。天巫躔天梁，或躔太阴者尤确。</li>
<li>丑未天梁不成「阳梁昌禄」格。但日月并照，亦成格局。在未宫者，日月反背，必须见吉化始可补救。但仍主在是非纠纷困扰中进财。且得财后必有破耗随来。</li>
<li>丑未天梁，见太阴化科，则天机必同时化忌，主因财受谤。未宫者尤甚。故得财亦不宜张扬。</li>
</ul>

巳、亥：天梁
<ul class = "ds-中州派分析段落">
<li>巳亥天梁与天同相对，主白手兴家。若禄马交驰，决主一生财源在于异域。</li>
<li>由于天同的影响，主早年以至中年财帛难聚。纵有祖业亦不克享。唯若左辅右弼，一居命宫，一居财帛宫者，则主定有祖业承继。或必有主理现成事业的机会。</li>
<li>巳亥天梁落陷，财源不广，亦较难积聚，此亦由于本身对财帛并不重视，宁愿重视精神享受有关。然而凡得财必主劳心费神，以致引起矛盾，为本星系居财帛宫的最大缺点。</li>
<li>巳亥天梁若先名后利者，对上述缺点即可弥补。故最宜见昌曲交会，亦宜化科。</li>
<li>若会天同化忌，只宜专业人士，或零售业亦可。</li>
</ul>

七杀

<ul class = "ds-中州派分析段落">
<li>七杀守财帛宫，主掌财权。这种掌财权的性质与天府不同。大致而言，天府主守，七杀主攻。天府主在现成局面下发展，七杀则主创出新局面。由于天府与七杀永远相对，所以虽各具不同本质，然而亦互相影响，故主守者亦有攻势，主攻者亦有守势。彼此相成相因而不离其本质，故在其他星曜会合下，天府亦可能变为以攻为守，七杀亦可以变为以守为攻。在推断时对此应仔细分别。</li>
<li>关于七杀守财帛宫，中州派传有一首歌诀：「寅申子午能横发，卯酉得财亦耗磨，辰戌二宫为富局，丑未积财巳亥多。」由此歌诀可见，凡七杀守财帛宫，本质皆为富局，仅卯酉二宫陷地，横发后亦易消磨殆尽，然而亦曾一度成富也。坊本不喜七杀坐财帛宫，至少不许为富局，实误。</li>
<li>七杀成富，首要条件为得禄（此点与天府相同）。禄存或同会化禄均可。盖必须得禄，然后所掌财权始成局面。否则虽掌财亦仅主范围极小。</li>
<li>七杀喜见禄、权、科会。更加辅佐诸吉，即掌财赋大权。</li>
<li>然而七杀守财帛宫，亦与守命宫相同，一生必有一次波折。故必须经历风险而成富。唯若投机失败，则甚难渡过风险。</li>
<li>若七杀有禄，且见吉曜而无煞，则经风险亦可夷然，能在友人帮助下东山复起。但若无吉有煞，或吉少煞重，则经风波后恢复元气维艰。投机者且主一蹶不振。</li>
<li>七杀见禄马交驰，又见禄权科会及辅佐吉曜者，在现代，往往为跨国机构掌财权人物。局面小者亦得掌远方财。</li>
<li>七杀与空劫、大耗同度相会，谋财费力。且得财同时亦生破耗。更见天刑，则不宜偏门行业，否则主易犯官司刑法。</li>
<li>七杀躔擎羊主争夺；躔陀罗主招忌；躔火铃则主横发横破。</li>
<li>七杀会煞曜重重，且见刑耗，或文曲化忌者，主一生易招盗贼。</li>
<li>凡七杀在财帛宫，则田宅宫必是太阴，若太阴落陷，火星同度，而七杀又见煞耗者，则须防火灾或兵灾。</li>
</ul>


子、午：七杀
<ul class = "ds-中州派分析段落">
<li>子午宫七杀独坐，对宫「武曲天府」，见吉化或禄存者，常得意外财。唯此意外财并非由非法途径得来。</li>
<li>子午七杀躔火铃，则主冒险犯难之财，且不易积聚。</li>
<li>子午七杀无吉化吉曜，一生至少亦得一次意外财。须注意保存元气，因为机会难再。武曲化忌者，则破败后只能慢慢培养元气，稍急躁即环境更劣。</li>
</ul>

</ul>

破军

<ul class = "ds-中州派分析段落">
<li>破军守财帛宫，不擅理财，其本质为开创财源。其开创财源的方式亦与常人不同。唯财源开创之后，跟随者亦必众。有煞，且易引起极大竞争；无煞，则虽有跟随者亦影响不大。天机守财帛亦有类似性质，不同的是，天机所得者为现成机会，但无人过问，一旦过问却引起竞争。此与破军之自行创造机会不同。</li>
<li>破军守财帛宫，有多方面的财源。</li>
<li>破军火星同宫，亦主横发，其横发性质与「火贪」、「铃贪」不同。贪狼所得之财，有得诸意外，不期而得的性质，但破军所得者，却因开创财源而致。</li>
<li>破军喜与禄存或化禄同度。倘破军本身化禄，则必兼行兼职以生财。有禄，固主收入丰厚，亦主财源较为稳定。</li>
<li>破军守财帛宫者，多主转变行业然后始能致富。即使见煞忌，转行之后亦可改善环境。若见禄权科会，则其转变必出于突然。故破军化禄，或会禄权科者，最喜见魁钺，则一生多机会。</li>
<li>破军与擎羊同宫，亦主横发。但此横发带有争夺的意味，即由横夺而来。故惹起是非纠纷甚大，若有空劫、刑耗，则主横得横失。</li>
<li>破军守财帛宫，主先破后兴。唯若无吉化吉会，且见煞曜者，则破败之后，其复兴亦每感外强中干。</li>
<li>破军与武曲化忌同躔，常感财力不足。若勉强周张，则不但东挪西补，而且愈图改变愈加速破败。</li>
<li>破军在亥子丑三宫，皆不宜文曲化忌同度，主生重大破败。如更见煞曜刑耗冲会，则主倒闭或倾家。</li>
</ul>


子、午：破军
<ul class = "ds-中州派分析段落">
<li>子午宫破军，对宫为「廉贞天相」，最喜廉贞化禄，则破军必同时化权，主权力与财帛俱增，名利双收。</li>
<li>唯若破军无禄，仅见辅佐诸吉，而煞曜亦不重者，则一旦掌权，破耗立至，必须重新整理业务，然后始能恢复旧观。</li>
<li>若有煞忌空劫，则破财损耗极大。而且往往功败垂成。</li>
<li>子午破军有吉化吉曜，则先有名然后有利。虽名大于利，然而亦必富裕。</li>
</ul>

</ul>

<li>辰戌破军与空劫同度，主周转困难，故极不宜经商。有时分期付款购物亦须小心，可能因此左支右絀。</li>
</ul>
